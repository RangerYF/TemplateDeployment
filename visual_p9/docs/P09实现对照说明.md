# P09 天体运动与引力模拟器实现对照说明

## 1. 技术路径

- 推荐路径：`2D Canvas + 参数 Store + 物理纯函数 + requestAnimationFrame`。
- 实际路径：一致。中央演示区使用 `src/components/scene/OrbitCanvas.tsx`，状态由 `src/store/simulationStore.ts` 管理，物理计算集中在 `src/engine/orbitalMechanics.ts`。
- 风格路径：复用 `visual_template` / `visual_chem02` 的 `TopBar + 左侧模型列表 + 中央演示区 + 右侧折叠面板` 结构，复用 `design_guid` 的 token、按钮、滑块、输入框、开关和面板视觉。
- 数据路径：所有场景数据集中在 `data/celestialModels.ts`，业务层通过 `src/data/celestialData.ts` 引用；没有在组件内硬编码模型参数。

## 2. 需求实现对照

| 需求 | 实现状态 | 实现位置 | 技术说明 |
|---|---:|---|---|
| 选择天体模型：单星、卫星变轨、双星、天体追及 | 已实现 | `ModelListPanel`、`TopBar` | 支持 `CEL-001/002/011/012/021/031` 六个场景切换 |
| 圆轨道运动动画 | 已实现 | `buildCircularFrame`、`OrbitCanvas` | 用 `v=sqrt(GM/r)`、`omega=sqrt(GM/r^3)` 计算并绘制速度/加速度箭头 |
| 高轨低速大周期展示 | 已实现 | `computeCircularMetrics`、`ParameterPanel`、`MetricsPanel` | 拖动半径实时联动速度、角速度、周期、加速度 |
| 椭圆轨道与开普勒三定律 | 已实现 | `buildEllipseFrame`、`computeEllipseMetrics` | 牛顿迭代求解开普勒方程，展示近日点/远日点、速度比和 `T^2/a^3` |
| 面积定律阴影标注 | 已实现 | `buildEllipseFrame`、`drawSectors` | 相同时间间隔生成面积扇形，可通过显示开关控制 |
| 霍曼转移轨道 | 已实现 | `computeHohmannMetrics`、`buildHohmannFrame`、`ControlPanel` | 点火按钮在低轨、转移轨、高轨之间切换，显示 `vA > v1 > v2 > vB` |
| 三宇宙速度情景 | 已实现 | `computeEscapeMetrics`、`buildEscapeFrame` | 调节初速度，判定落回、圆轨道、椭圆、逃逸、逃出太阳系 |
| 双星系统 | 已实现 | `computeBinaryMetrics`、`buildBinaryFrame` | 计算 `r1=m2L/(m1+m2)`、`r2=m1L/(m1+m2)`，标注质心 |
| 天体追及与相遇 | 已实现 | `computeChaseMetrics`、`buildChaseFrame` | 计算角速度差、第一次相遇时间和追一圈后相遇时间 |
| 播放/暂停/调速/回到起点 | 已实现 | `ControlPanel`、`simulationStore` | `requestAnimationFrame` 驱动，倍率 0.2x 至 4x |
| 速度/加速度箭头显示 | 已实现 | `OrbitCanvas` | 通过 `showVectors` 控制 |
| 参数预设与真实参考数据 | 已实现 | `data/celestialModels.ts`、`SourcePanel` | 内置地球、月球、太阳、近地卫星、地球同步轨道参考数据 |
| 数值面板 | 已实现 | `MetricsPanel` | 按当前场景显示核心物理量与教学结论 |
| 公式展示 | 已实现 | `FormulaPanel` | 从 `data/` 读取公式与说明 |
| 数据来源表 | 已实现 | `data/celestialModels.ts`、`SourcePanel`、P09 docx | docx 末尾追加“数据来源表（P09 实现补充）” |
| Snapshot 桥接 | 已实现 | `templateBridge.ts` | 提供默认快照、当前快照、加载和校验 |

## 3. 验收标准对照

| 验收标准 | 结果 | 说明 |
|---|---:|---|
| 椭圆轨道近日点和远日点速度之比与角动量守恒一致 | 通过 | `v近/v远=(1+e)/(1-e)`，默认 `e=0.3` 时为 `1.857` |
| 双星质量比为 1:2 时轨道半径比为 2:1 | 通过 | `r1/r2=m2/m1`，公式实现直接满足质心条件 |
| 卫星在低轨加速后进入高轨，高轨速度小于低轨速度，方向正确 | 通过 | 霍曼转移数值面板显示 `vA > v1 > v2 > vB`，轨道方向按切向箭头绘制 |
