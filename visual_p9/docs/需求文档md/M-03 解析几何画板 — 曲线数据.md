**1. 数据结构说明**

每个曲线配置包含以下字段：

  ----------------------- ----------------------- ------------------------------------
  字段                    类型                    说明

  id                      string                  编号 CRV-XXX

  name_cn                 string                  中文名

  category                string                  分类（圆/椭圆/双曲线/抛物线/直线）

  level                   string                  学段

  standard_eq             string                  标准方程

  params                  array                   可调参数及范围

  key_elements            object                  关键要素（焦点、准线、渐近线等）

  properties              object                  几何性质

  teaching_points         array                   教学要点
  ----------------------- ----------------------- ------------------------------------

**2. 直线**

**CRV-001 直线（斜截式）**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            y = kx + b

  学段                                高中必修
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  k              斜率           1              \[-20, 20\]    0.1

  b              截距           0              \[-20, 20\]    0.1
  -------------- -------------- -------------- -------------- --------------

**CRV-002 直线（一般式）**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            Ax + By + C = 0

  学段                                高中必修
  ----------------------------------- -----------------------------------

**直线间关系：**

  ----------------------------------- -----------------------------------
  关系                                条件

  平行                                A₁B₂ = A₂B₁ 且 A₁C₂ ≠ A₂C₁

  垂直                                A₁A₂ + B₁B₂ = 0

  重合                                A₁/A₂ = B₁/B₂ = C₁/C₂

  点到直线距离                        d = \|Ax₀ + By₀ + C\| / √(A² + B²)
  ----------------------------------- -----------------------------------

**CRV-003 直线（参数式）**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            x = x₀ + t cosα, y = y₀ + t sinα

  学段                                高中选修

  用途                                与曲线联立求弦长
  ----------------------------------- -----------------------------------

**教学要点：**

参数 t 的几何意义：有向距离

弦长公式 \|AB\| = \|t₁ - t₂\|

韦达定理应用：t₁ + t₂ 和 t₁ · t₂

**3. 圆**

**CRV-011 圆（标准方程）**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            (x - a)² + (y - b)² = r²

  学段                                高中必修
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  a              圆心横坐标     0              \[-10, 10\]    0.1

  b              圆心纵坐标     0              \[-10, 10\]    0.1

  r              半径           3              \[0.1, 10\]    0.1
  -------------- -------------- -------------- -------------- --------------

**关键要素：**

  ----------------------------------- -----------------------------------
  要素                                公式/位置

  圆心                                (a, b)

  半径                                r

  面积                                πr²

  周长                                2πr
  ----------------------------------- -----------------------------------

**CRV-012 圆（一般方程）**

  ----------------------------------- -----------------------------------
  参数                                值

  一般方程                            x² + y² + Dx + Ey + F = 0

  圆心                                (-D/2, -E/2)

  半径                                r = √(D² + E² - 4F) / 2

  存在条件                            D² + E² - 4F \> 0
  ----------------------------------- -----------------------------------

**直线与圆的位置关系：**

  ----------------------------------- -----------------------------------
  关系                                条件（d 为圆心到直线距离）

  相离                                d \> r

  相切                                d = r

  相交                                d \< r

  弦长                                L = 2√(r² - d²)
  ----------------------------------- -----------------------------------

**教学要点：**

圆的切线方程：过圆上点 (x₀, y₀) 的切线为 x₀x + y₀y = r²（圆心在原点时）

两圆位置关系由圆心距 d 和 r₁ + r₂、\|r₁ - r₂\| 的大小关系决定

圆是离心率 e = 0 的圆锥曲线

**4. 椭圆**

**CRV-021 椭圆（焦点在 x 轴）**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            x²/a² + y²/b² = 1（a \> b \> 0）

  学段                                高中选修
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  a              半长轴         5              \[1, 10\]      0.1

  b              半短轴         3              \[0.1, a)      0.1
  -------------- -------------- -------------- -------------- --------------

**关键要素：**

  ----------------------------------- -----------------------------------
  要素                                公式/位置

  焦点                                F₁(-c, 0), F₂(c, 0)

  焦距                                2c, 其中 c = √(a² - b²)

  离心率                              e = c/a（0 \< e \< 1）

  准线                                x = ±a²/c = ±a/e

  顶点                                (±a, 0), (0, ±b)

  长轴长                              2a

  短轴长                              2b

  通径（半弦）                        l = b²/a

  面积                                S = πab
  ----------------------------------- -----------------------------------

**焦点弦性质：**

  ----------------------------------- -----------------------------------
  性质                                公式

  焦半径公式                          PF₁ = a + ex, PF₂ = a - ex

  焦点弦长                            \|AB\| = 2a - e(x₁ + x₂)（过 F₂）

  通径长（过焦点且垂直于长轴的弦）    2b²/a

  焦点三角形面积                      S = b² tan(θ/2)（θ 为 ∠F₁PF₂）
  ----------------------------------- -----------------------------------

**CRV-022 椭圆（焦点在 y 轴）**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            x²/b² + y²/a² = 1（a \> b \> 0）

  焦点                                F₁(0, -c), F₂(0, c)

  准线                                y = ±a²/c
  ----------------------------------- -----------------------------------

**教学要点：**

椭圆定义：到两个定点距离之和等于常数 2a 的点的轨迹

离心率 e 越接近 0 越接近圆，越接近 1 越\"扁\"

焦半径公式是解题核心

a² = b² + c² 是基本关系

**5. 双曲线**

**CRV-031 双曲线（焦点在 x 轴）**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            x²/a² - y²/b² = 1（a \> 0, b \> 0）

  学段                                高中选修
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  a              实半轴         3              \[0.5, 10\]    0.1

  b              虚半轴         4              \[0.5, 10\]    0.1
  -------------- -------------- -------------- -------------- --------------

**关键要素：**

  ----------------------------------- -----------------------------------
  要素                                公式/位置

  焦点                                F₁(-c, 0), F₂(c, 0)

  焦距                                2c, 其中 c = √(a² + b²)

  离心率                              e = c/a（e \> 1）

  准线                                x = ±a²/c = ±a/e

  实轴顶点                            (±a, 0)

  渐近线                              y = ±(b/a)x

  通径                                2b²/a
  ----------------------------------- -----------------------------------

**焦点弦性质：**

  ----------------------------------- -----------------------------------
  性质                                公式

  焦半径公式                          PF₁ = \|a + ex\|, PF₂ = \|a - ex\|

  焦点三角形面积                      S = b² \|tan(θ/2)\|（θ 为 ∠F₁PF₂）
  ----------------------------------- -----------------------------------

**CRV-032 双曲线（焦点在 y 轴）**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            y²/a² - x²/b² = 1（a \> 0, b \> 0）

  焦点                                F₁(0, -c), F₂(0, c)

  渐近线                              y = ±(a/b)x
  ----------------------------------- -----------------------------------

**CRV-033 等轴双曲线**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            x² - y² = a²（或 y² - x² = a²）

  离心率                              e = √2

  渐近线                              y = ±x

  渐近线夹角                          90°
  ----------------------------------- -----------------------------------

**教学要点：**

双曲线定义：到两个定点距离之差的绝对值等于常数 2a 的点的轨迹

c² = a² + b²（注意与椭圆的区别）

渐近线是理解双曲线形状的关键

等轴双曲线的渐近线互相垂直

**6. 抛物线**

**CRV-041 抛物线（开口向右）**

  ----------------------------------- -----------------------------------
  参数                                值

  标准方程                            y² = 2px（p \> 0）

  学段                                高中选修
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- ---------------- -------------- -------------- --------------
  参数           含义             默认值         范围           步长

  p              焦点到准线距离   2              \[0.5, 10\]    0.1
  -------------- ---------------- -------------- -------------- --------------

**关键要素：**

  ----------------------------------- -----------------------------------
  要素                                公式/位置

  焦点                                F(p/2, 0)

  准线                                x = -p/2

  顶点                                O(0, 0)

  离心率                              e = 1

  通径                                2p

  焦半径                              PF = x₀ + p/2
  ----------------------------------- -----------------------------------

**CRV-042 抛物线（四种方向）**

  ----------------- ----------------- ----------------- -----------------
  方向              方程              焦点              准线

  开口向右          y² = 2px          (p/2, 0)          x = -p/2

  开口向左          y² = -2px         (-p/2, 0)         x = p/2

  开口向上          x² = 2py          (0, p/2)          y = -p/2

  开口向下          x² = -2py         (0, -p/2)         y = p/2
  ----------------- ----------------- ----------------- -----------------

**焦点弦性质：**

  ----------------------------------- -----------------------------------
  性质                                公式

  焦点弦长                            \|AB\| = x₁ + x₂ + p（开口向右）

  通径长                              2p（过焦点且垂直于轴的弦）

  焦点弦中点到准线距离                (x₁ + x₂)/2 + p/2

  以焦点弦为直径的圆与准线相切        永远成立
  ----------------------------------- -----------------------------------

**教学要点：**

抛物线定义：到一个定点（焦点）和一条定直线（准线）距离相等的点的轨迹

抛物线只有一个焦点（区别于椭圆和双曲线）

焦半径公式 PF = x₀ + p/2 极其常用

抛物线的光学性质：平行于轴的光线反射后都过焦点

**7. 圆锥曲线统一性质**

**7.1 焦点、准线统一定义**

  ----------------------- ----------------------- ------------------------
  曲线                    离心率 e                定义

  圆                      e = 0                   到一个定点距离等于常数

  椭圆                    0 \< e \< 1             到焦点距离 / 到准线距离
                                                  = e

  抛物线                  e = 1                   到焦点距离 = 到准线距离

  双曲线                  e \> 1                  到焦点距离 / 到准线距离
                                                  = e
  ----------------------- ----------------------- ------------------------

**7.2 离心率演变可视化**

滑块调节 e 从 0 到 2，展示曲线形状演变：

  ----------------------- ----------------------- -----------------------
  e 值                    曲线                    形状特征

  0                       圆                      完美的圆

  0.3                     椭圆                    接近圆的椭圆

  0.6                     椭圆                    较明显的椭圆

  0.9                     椭圆                    很\"扁\"的椭圆

  1.0                     抛物线                  抛物线（开口图形）

  1.2                     双曲线                  渐近线夹角较小

  √2                      等轴双曲线              渐近线互相垂直

  2.0                     双曲线                  渐近线夹角较大
  ----------------------- ----------------------- -----------------------

**教学要点：**

圆锥曲线名称来源：用平面截圆锥面得到的截线

离心率是刻画圆锥曲线形状的关键参数

e 越接近 0 越像圆，e 越大双曲线的\"开口\"越大

**8. 典型题型配置**

**8.1 弦长问题**

  ----------------------- ----------------------- ------------------------
  题型                    参数配置                求解方法

  直线截椭圆              椭圆 + 直线方程         联立→韦达定理→弦长公式

  过焦点弦                曲线 + 过焦点直线       焦点弦公式

  过定点弦                曲线 + 过定点直线       联立方程组
  ----------------------- ----------------------- ------------------------

**8.2 面积问题**

  ----------------------- ----------------------- -----------------------
  题型                    参数配置                求解方法

  焦点三角形              椭圆/双曲线上点 +       S = b² tan(θ/2)
                          两焦点                  

  弦与坐标轴围成面积      曲线 + 直线             联立→积分或面积公式

  最大三角形面积          椭圆内接三角形          参数方程 + 三角函数
  ----------------------- ----------------------- -----------------------

**8.3 轨迹问题**

  -------------------------------- ----------------------- -----------------------
  题型                             动点条件                轨迹类型

  到两定点距离之和为常数           \|PF₁\| + \|PF₂\| = 2a  椭圆

  到两定点距离之差的绝对值为常数   \|\|PF₁\| - \|PF₂\|\| = 双曲线
                                   2a                      

  到定点和定直线距离相等           \|PF\| = d(P, l)        抛物线

  到两定点距离之比为常数           \|PA\|/\|PB\| = k (k≠1) 阿波罗尼圆
  -------------------------------- ----------------------- -----------------------
