**2. 一次函数**

**FUN-001 一次函数**

  ----------------------------------- -----------------------------------
  参数                                值

  表达式                              y = kx + b

  学段                                初中

  定义域                              (-∞, +∞)

  值域                                (-∞, +∞)
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  k              斜率           1              \[-10, 10\]    0.1

  b              截距           0              \[-10, 10\]    0.1
  -------------- -------------- -------------- -------------- --------------

**特征点：**

  ----------------------------------- -----------------------------------
  点                                  公式

  x 轴交点                            (-b/k, 0)（k ≠ 0）

  y 轴交点                            (0, b)
  ----------------------------------- -----------------------------------

**性质：**

  ----------------------------------- -----------------------------------
  性质                                描述

  单调性                              k \> 0 递增，k \< 0 递减

  奇偶性                              b = 0 时为奇函数

  图像                                直线
  ----------------------------------- -----------------------------------

**教学要点：**

k 的几何意义：直线的倾斜程度（正切值）

b 的几何意义：直线与 y 轴的交点纵坐标

k 相同、b 不同的直线互相平行

两条直线垂直 ⟺ k₁ · k₂ = -1

**3. 二次函数**

**FUN-011 二次函数（一般式）**

  ----------------------------------- -----------------------------------
  参数                                值

  表达式                              y = ax² + bx + c

  学段                                初中/高中

  定义域                              (-∞, +∞)

  值域                                a \> 0 时 \[4ac-b²/4a, +∞)；a \< 0
                                      时 (-∞, 4ac-b²/4a\]
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  a              二次项系数     1              \[-5,          0.1
                                               5\]（≠0）      

  b              一次项系数     0              \[-10, 10\]    0.1

  c              常数项         0              \[-10, 10\]    0.1
  -------------- -------------- -------------- -------------- --------------

**特征点：**

  ----------------------------------- -----------------------------------
  点                                  公式

  顶点                                (-b/2a, (4ac-b²)/4a)

  对称轴                              x = -b/2a

  与 y 轴交点                         (0, c)

  与 x 轴交点                         x = (-b ± √(b²-4ac)) / 2a（Δ ≥ 0
                                      时）

  判别式                              Δ = b² - 4ac
  ----------------------------------- -----------------------------------

**教学要点：**

a 决定开口方向和大小：a \> 0 开口向上，\|a\| 越大越窄

b 与 a 共同决定对称轴位置：同号对称轴在 y 轴左侧，异号在右侧

c 决定图像与 y 轴的交点

Δ \> 0 两个交点，Δ = 0 一个交点（切线），Δ \< 0 无实数交点

**FUN-012 二次函数（顶点式）**

  ----------------------------------- -----------------------------------
  参数                                值

  表达式                              y = a(x - h)² + k

  学段                                初中/高中

  定义域                              (-∞, +∞)
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  a              开口系数       1              \[-5,          0.1
                                               5\]（≠0）      

  h              顶点横坐标     0              \[-10, 10\]    0.1

  k              顶点纵坐标     0              \[-10, 10\]    0.1
  -------------- -------------- -------------- -------------- --------------

**教学要点：**

顶点式最直观地展示平移变换

(h, k) 就是顶点坐标

由 y = ax² 向右平移 h 个单位，向上平移 k 个单位得到

**4. 反比例函数**

**FUN-021 反比例函数**

  ----------------------------------- -----------------------------------
  参数                                值

  表达式                              y = k/x

  学段                                初中

  定义域                              (-∞, 0) ∪ (0, +∞)

  值域                                (-∞, 0) ∪ (0, +∞)
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  k              比例系数       1              \[-10,         0.1
                                               10\]（≠0）     
  -------------- -------------- -------------- -------------- --------------

**性质：**

  ----------------------------------- -----------------------------------
  性质                                描述

  单调性                              k \> 0 时每个象限内递减；k \< 0
                                      时每个象限内递增

  奇偶性                              奇函数，关于原点对称

  渐近线                              x 轴和 y 轴

  图像                                双曲线
  ----------------------------------- -----------------------------------

**教学要点：**

k \> 0 图像在一、三象限；k \< 0 图像在二、四象限

\|k\| 的几何意义：曲线上任一点到两条坐标轴所围矩形的面积

反比例函数与一次函数的交点问题是初中重点

**5. 指数函数**

**FUN-031 指数函数**

  ----------------------------------- -----------------------------------
  参数                                值

  表达式                              y = aˣ

  学段                                高中必修

  定义域                              (-∞, +∞)

  值域                                (0, +∞)
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  a              底数           2              (0, 10\]（≠1） 0.1
  -------------- -------------- -------------- -------------- --------------

**特征点：**

  ----------------------------------- -----------------------------------
  点                                  公式

  过定点                              (0, 1)，所有指数函数都过此点
  ----------------------------------- -----------------------------------

**性质：**

  ----------------------------------- -----------------------------------
  性质                                描述

  单调性                              a \> 1 递增，0 \< a \< 1 递减

  奇偶性                              非奇非偶

  渐近线                              x 轴（y = 0）为水平渐近线

  有界性                              下有界（y \> 0）无上界
  ----------------------------------- -----------------------------------

**教学要点：**

所有指数函数都过 (0, 1) 点

a \> 1 时，底数越大增长越快

0 \< a \< 1 时，底数越小衰减越快

指数爆炸：即使底数略大于 1，长远来看增长也极快

**FUN-032 一般指数函数**

  ----------------------------------- -----------------------------------
  参数                                值

  表达式                              y = b · aˣ + c

  学段                                高中必修

  定义域                              (-∞, +∞)

  值域                                b \> 0 时 (c, +∞)；b \< 0 时 (-∞,
                                      c)
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  a              底数           2              (0, 10\]（≠1） 0.1

  b              系数           1              \[-5,          0.1
                                               5\]（≠0）      

  c              平移量         0              \[-10, 10\]    0.1
  -------------- -------------- -------------- -------------- --------------

**教学要点：**

c 决定水平渐近线 y = c

b 决定翻转和伸缩

实际应用：人口增长、放射性衰变、复利计算

  -------------- -------------- -------------- -------------- --------------

**7. 幂函数**

**FUN-051 幂函数**

  ----------------------------------- -----------------------------------
  参数                                值

  表达式                              y = xⁿ

  学段                                高中必修

  过定点                              (1, 1)
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  n              幂指数         2              \[-3, 3\]      0.1
  -------------- -------------- -------------- -------------- --------------

**教材要求掌握的五个幂函数：**

  -------------- --------------- --------------- -------------- --------------
  表达式         定义域          值域            奇偶性         单调性

  y = x          (-∞, +∞)        (-∞, +∞)        奇             递增

  y = x²         (-∞, +∞)        \[0, +∞)        偶             x\<0
                                                                递减，x\>0
                                                                递增

  y = x³         (-∞, +∞)        (-∞, +∞)        奇             递增

  y = x\^(1/2)   \[0, +∞)        \[0, +∞)        非奇非偶       递增

  y = x\^(-1)    (-∞,0)∪(0,+∞)   (-∞,0)∪(0,+∞)   奇             每个区间递减
  -------------- --------------- --------------- -------------- --------------

**教学要点：**

所有幂函数都过 (1, 1) 点

n \> 0 时过原点（0 在定义域内时）

n 为正整数时定义域为全体实数

幂函数在第一象限的图像特征：n\>1 下凸、0\<n\<1 上凸、n\<0 递减

**8. 三角函数**

**FUN-061 正弦函数（一般式）**

  ----------------------------------- -----------------------------------
  参数                                值

  表达式                              y = A sin(ωx + φ) + k

  学段                                高中必修

  定义域                              (-∞, +∞)

  值域                                \[k - A, k + A\]（A \> 0 时）
  ----------------------------------- -----------------------------------

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  A              振幅           1              \[0.1, 5\]     0.1

  ω              角频率         1              \[0.1, 10\]    0.1

  φ              初相           0              \[-π, π\]      π/12

  k              偏移量         0              \[-5, 5\]      0.1
  -------------- -------------- -------------- -------------- --------------

**特征点（五点法）：**

  ----------------------- ----------------------- -----------------------
  点序号                  x 值                    y 值

  1                       -φ/ω                    k

  2                       -φ/ω + π/(2ω)           k + A

  3                       -φ/ω + π/ω              k

  4                       -φ/ω + 3π/(2ω)          k - A

  5                       -φ/ω + 2π/ω             k
  ----------------------- ----------------------- -----------------------

**性质：**

  ----------------------------------- -----------------------------------
  性质                                公式

  周期                                T = 2π/ω

  最大值                              k + A

  最小值                              k - A

  对称轴                              x = -φ/ω + kπ/ω + π/(2ω)（k ∈ Z）

  对称中心                            (-φ/ω + kπ/ω, k)（k ∈ Z）
  ----------------------------------- -----------------------------------

**教学要点：**

A 控制振幅（纵向拉伸）

ω 控制周期（横向压缩）

φ 控制左右平移

k 控制上下平移

变换顺序很重要：先平移再伸缩 ≠ 先伸缩再平移

**9. 绝对值函数**

**FUN-071 绝对值函数**

  ----------------------------------- -----------------------------------
  参数                                值

  表达式                              y =

  学段                                高中必修
  ----------------------------------- -----------------------------------

**常见形式：**

  ----------------------------------- -----------------------------------
  形式                                图像变换

  y = \|x\|                           V 形图像

  y = \|ax + b\|                      V 形平移

  y = \|ax² + bx + c\|                将 x 轴下方部分翻折到上方

  y = f(\|x\|)                        保留 y 轴右侧图像，关于 y 轴对称
  ----------------------------------- -----------------------------------

**教学要点：**

y = \|f(x)\|：将 f(x) 图像中 x 轴下方的部分关于 x 轴翻折

y = f(\|x\|)：保留 f(x) 在 y 轴右侧的部分，关于 y 轴作对称

绝对值不等式的几何意义：\|x - a\| \< b 表示到 a 的距离小于 b

**12. 导数相关**

**FUN-101 切线工具参数**

  ----------------------------------- -----------------------------------
  参数                                值

  学段                                高中选修

  功能                                在 f(x) 上任取一点 x₀，显示切线 y =
                                      f\'(x₀)(x - x₀) + f(x₀)
  ----------------------------------- -----------------------------------

**常用导数公式：**

  ----------------------------------- -----------------------------------
  f(x)                                f\'(x)

  xⁿ                                  nxⁿ⁻¹

  aˣ                                  aˣ ln(a)

  eˣ                                  eˣ

  log_a(x)                            1/(x ln a)

  ln(x)                               1/x

  sin(x)                              cos(x)

  cos(x)                              -sin(x)

  tan(x)                              sec²(x) = 1/cos²(x)
  ----------------------------------- -----------------------------------

**FUN-102 极值与拐点**

**极值判断：**

  ----------------------------------- -----------------------------------
  条件                                结论

  f\'(x₀) = 0 且 f\'\'(x₀) \> 0       x₀ 是极小值点

  f\'(x₀) = 0 且 f\'\'(x₀) \< 0       x₀ 是极大值点

  f\'(x₀) = 0 且 f\'\'(x₀) = 0        需进一步判断
  ----------------------------------- -----------------------------------

**拐点判断：**

  ----------------------------------- -----------------------------------
  条件                                结论

  f\'\'(x₀) = 0 且 f\'\'\'(x₀) ≠ 0    x₀ 是拐点
  ----------------------------------- -----------------------------------

**13. 函数变换规则**

**13.1 平移变换**

  ----------------------- ----------------------- -----------------------
  变换                    规则                    效果

  y = f(x) → y = f(x - h) x 替换为 x - h          图像向右平移 h
                                                  个单位（h \> 0）

  y = f(x) → y = f(x) + k 加常数 k                图像向上平移 k
                                                  个单位（k \> 0）
  ----------------------- ----------------------- -----------------------

**13.2 对称变换**

  ----------------------- ----------------------- -----------------------
  变换                    规则                    效果

  y = f(x) → y = -f(x)    y 变号                  关于 x 轴对称

  y = f(x) → y = f(-x)    x 变号                  关于 y 轴对称

  y = f(x) → y = -f(-x)   都变号                  关于原点对称

  y = f(x) → x = f(y)     互换 x, y               关于 y = x 对称
  ----------------------- ----------------------- -----------------------

**13.3 伸缩变换**

  ----------------------- ----------------------- -----------------------
  变换                    规则                    效果

  y = f(x) → y = Af(x)    纵坐标 × A              纵向拉伸/压缩

  y = f(x) → y = f(ωx)    横坐标 ÷ ω              横向压缩/拉伸
  ----------------------- ----------------------- -----------------------

**教学要点：**

变换顺序很重要：\"先平移后伸缩\"和\"先伸缩后平移\"结果不同

正弦函数的变换：建议统一使用\"先平移后伸缩\"的顺序

变换的复合可以用来理解复杂函数图像
