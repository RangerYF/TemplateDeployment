**1. 产品线概述**

**1.1 为什么做这个？**

数学教学中，大量概念需要**空间想象力**和**动态变化感知**------立体几何要在脑中旋转三维模型，函数要看图像随参数变化，解析几何要理解曲线与直线的位置关系，概率要感受随机性\...\...这些内容在黑板上画静态图，学生理解起来事倍功半。

数学教具产品线旨在为每个核心教学难点提供一个**可交互的可视化工具**，覆盖小学到高中阶段的主要教学需求。

**3. M-01 立体几何展示台（已上线）**

[Untitled
page](https://kcnw8jqlswwm.feishu.cn/wiki/QsPrwyx1XisKSNkxztPc7Q6Pnbd)

**一、这个东西是干什么的？**

高中数学课讲立体几何，老师最头疼的就是：正方体的截面、棱锥的侧面展开、圆柱的外接球\...\...这些东西，在黑板上怎么画都是\"感觉差点意思\"，学生脑子里转不过来。

**立体几何展示台**就是帮老师解决这个问题的，目标是覆盖小学、初中、高中80%的立体几何教学需求，第一版暂时不介入自然语言-LLM入口，后续可以做一些Agent。

老师选一个几何体，设置几个参数，就能得到一个**可以用鼠标拖着转的 3D
模型**。顶点名字可以改，线段可以标字母，还能加坐标系、画外接球、选几个点做外接圆，侧面积体积自动算好显示出来。

**二、用户是谁？他们怎么用？**

**目标用户：** 小学/初中/高中数学老师，不要求有任何编程或软件操作基础。

**执行人：** 全栈工程师 \@程俊楠

**典型使用场景：**

课前：提前做好一个\"正四棱锥，底面边长3，高4\"的模型，上课时打开浏览器直接展示

课中：在投影仪上拖动几何体旋转，让学生从各个角度观察

讲题时：边讲边指出\"这条边叫 AB，这个面叫侧面
ABC\"，顶点名字都标在模型上

**老师对工具的期望（直白版）：**

我不会编程，不要让我写代码

操作越简单越好，最好点几下就搞定

必须好看，要能在教室大屏上清晰显示

数据要准确，什么外接球半径、体积，算错了我就被学生问懵了

**三、功能清单（老师视角描述）**

**3.1 第一步：选几何体类型**

打开工具，老师首先看到的是**几何体类型选择区**，提供以下选项，点击即选：

  ----------------------------------- ----------------------------------------
  几何体                              说明（不选择均自设默认值）

  棱锥                                可设置底面边数（三棱锥、四棱锥等）和高

  圆锥                                可选设置底面半径和高

  圆柱                                可选设置底面半径和高

  长方体                              可选设置长、宽、高三个参数

  正方体                              可选只需设置边长一个参数

  球                                  可选只需设置半径一个参数
  ----------------------------------- ----------------------------------------

选好之后，右侧立刻出现一个**默认参数的 3D 模型预览**。

**3.2 第二步：设置参数**

选完几何体后，出现该几何体的**参数输入面板**，老师用数字输入框修改：

棱锥：底面边数、底面边长、高

圆锥：底面半径、高

圆柱：底面半径、高

长方体：长、宽、高

正方体：边长

球：半径

参数修改后，3D 模型**实时更新**，无需点击确认按钮。

**3.3 第三步：标注与编辑**

这一组功能让老师可以把 3D
模型\"教学化\"------加上数学符号、名字标注，让它看起来像教材里的图。

**✅ 功能 1：给顶点改名字**

每个顶点旁边有一个小标签，默认显示
A、B、C、D......老师双击标签，可以输入任意文字，比如改成 P、Q，或者
A₁、B₁。修改后，3D 模型上立即显示新名字。

**✅ 功能 2：给线段加名字**

老师点击两个顶点，即可选中这两点之间的线段。选中后出现一个输入框，可以给这条线段命名，比如写\"斜高
l\"。名字显示在线段中点附近。

**✅ 功能 3：点击取新的点**

老师选中一条线段，点击\"取中点\"按钮，系统自动在线段中点处添加一个新的顶点，可以给这个中点命名（比如
M）。中点和两端点之间会自动连线。老师也可以点击边上任意位置取任意点，且该点可以在边上滑动。

**✅ 功能 4：拖动顶点位置**

老师可以按住某个顶点，拖动来调整它的位置。拖动顶点时，所有相关联的线段和面自动跟着更新。用于做一些非标准形状，比如讲斜切棱柱的时候。

**3.4 第四步：辅助功能**

这一组是高中数学中经常配合几何体讲解的工具。

**✅ 功能 5：建立坐标系**

点击\"建立坐标系\"按钮，出现一个选择原点的提示。老师点击模型上的某个顶点作为原点，系统自动以这个顶点为原点，生成
x、y、z 三轴，轴上有刻度。各顶点自动显示对应坐标值。

**✅ 功能 6：外接球**

点击\"显示外接球\"按钮，系统自动计算当前几何体的外接球，并用半透明的球面显示出来。同时显示外接球半径的计算结果，精确到小数点后两位。仅对有外接球的几何体开放此功能（球本身不显示）。

**✅ 功能 7：对选定点做外接圆**

老师在模型上选定 3
个或以上的顶点，点击\"做外接圆\"按钮。系统在这几个点所在平面内画出外接圆，并显示圆的半径。

**✅ 功能 8：显示表面积和体积**

页面某个角落（建议右下角）始终显示当前几何体的表面积（S =
**\_\_\_）和体积（V =
\_\_\_**）。参数变化后实时更新。支持精确分数或根号形式展示（如正四面体的体积显示为
√2/12 × a³），也提供数值近似。点击数字详情可以看到计算步骤。

**✅ 功能 9：两点间可以创建线段**

创建线段颜色可以选择，虚实可以选择（几何体原有线段默认黑色）。点击可以选择计算线段长度。

**✅ 功能 10：切面展示**

工具区可以选择建立切面。点击切面可在展示区展示切面。可计算切面面积。

**✅ 功能 11：展开图和三视图**

工具区可以选择建立展开图。工具区可选择查看三视图。可以下载三视图和展开图图片。

**3.5 第五步：旋转和查看 3D 模型**

这是整个工具最核心的视觉体验：

\- **鼠标左键拖动**：旋转模型，从各个角度查看

\- **鼠标滚轮**：放大缩小

\- **鼠标右键拖动**：平移模型位置

触屏设备（平板）：支持单指旋转、双指缩放

提供一个\"重置视角\"按钮，一键回到默认角度

提供\"正视图 / 俯视图 / 侧视图\"三个快捷视角按钮

**4. M-02 函数图像实验室（规划中）**

[Untitled
page](https://kcnw8jqlswwm.feishu.cn/wiki/QFDbwdM84iZPpDk5Mw6c1A3Ynmg)

**一、这个东西是干什么的？**

函数是初高中数学的\"主线任务\"，从初中的一次函数、二次函数、反比例函数，到高中的指数、对数、幂函数、三角函数，再到导数------函数贯穿了整个中学数学。

老师讲函数最大的问题是**参数变化对图像的影响**。比如 y = ax² + bx + c，a
从 1 变到 0.5，抛物线变宽了；b 从 0 变到
2，顶点移动了------这些在黑板上只能画两张图让学生对比，学生还是看不出\"连续变化\"的感觉。再比如讲导数的几何意义，切线随着切点移动而旋转，黑板上根本画不出来。

**函数图像实验室**就是解决这个问题的。老师输入一个函数表达式（或者从预设库里选），拖动参数滑块，图像**实时跟着变**。还能同时画多条函数做对比，自动标出零点、极值点，画切线看斜率，甚至把
f\'(x) 的图像也画出来和 f(x) 对照着看。

**PM备注：**
函数图像是数学工具的核心模块，复用率最高。初中老师讲二次函数就需要，高中从必修到选修都在用。和
M-04
三角函数有一定重叠（三角函数图像部分），但三角函数有单位圆联动等专属功能，所以分开做。

**二、用户是谁？他们怎么用？**

**目标用户：** 初中/高中数学老师

**覆盖学段：**

\[初中\] 一次函数 / 二次函数 / 反比例函数

\[高中必修\] 指数函数 / 对数函数 / 幂函数

\[高中选修\] 导数及其应用

**典型使用场景：**

\- **场景 1 --- 初中讲二次函数开口方向：** 老师打开工具，选\"二次函数 y
= ax² + bx + c\"，把 b 和 c 固定为 0，拖动 a 的滑块，学生看到 a \> 0
时开口朝上、a \< 0 时开口朝下、\|a\| 越大开口越窄。整个过程 10
秒钟，比画 5 张图管用。

\- **场景 2 --- 高中讲指数函数与对数函数的关系：**
老师在同一个坐标系里画 y = 2ˣ 和 y = log₂x，点击\"显示 y = x
对称轴\"，学生立刻看到两条曲线关于 y = x 对称。再切换底数 a = 3、a =
10，对称关系始终成立。

\- **场景 3 --- 高中讲导数的几何意义：** 老师选 y = x³ -
3x，开启\"导数可视化\"，屏幕上同时出现 f(x) 和 f\'(x)
两条曲线。拖动曲线上的一个点，该点的切线跟着旋转，f\'(x)
图像上对应点同步高亮。学生可以直观看到\"f\'(x) = 0
的地方，原函数有极值\"。

\- **场景 4 --- 讲函数变换：** 老师选 y = sin(x)，点击\"向右平移
π/6\"，图像动画滑动到新位置，坐标轴上的关键点跟着标注更新。学生理解\"左加右减\"不再靠死记硬背。

**老师对工具的期望：**

输入函数要简单，最好能直接打数学表达式，别让我学编程语法

参数滑块要灵敏，拖一下图像马上变

颜色要鲜明，投影仪上多条曲线能分清楚

特征点（零点、极值）要自动标出来，别让我一个个算坐标再手动标

**三、功能清单（老师视角描述）**

**3.1 第一步：选择或输入函数**

打开工具，老师看到两个入口：

对数 / 幂 / 三角 / 绝对值 / 复合），点击即加载到画布上

\- **自由输入框**：直接输入数学表达式（如 \`y = x\^2 + 2x - 1\`、\`y =

**✅ 功能 1：函数表达式输入与解析**

老师在输入框中输入数学表达式（支持 \`\^\` 表示次方、\`sqrt()\`
表示根号、\`ln()\` / \`log()\` 表示对数、\`sin()\` / \`cos()\` /
\`tan()\` 表示三角函数、\`abs()\` 表示绝对值、\`pi\` 和 \`e\`
表示常量）。输入后按回车或点击\"绘制\"按钮，图像立即出现在坐标系上。如果表达式有语法错误，输入框下方用红色文字提示\"表达式格式有误，请检查\"，**不要弹窗打断操作**。

**✅ 功能 2：预设函数一键加载**

预设函数按学段和类型分类展示，老师点击某个预设（如\"二次函数 y = ax² +
bx + c\"），坐标系立即绘制该函数的默认图像（a=1, b=0,
c=0），同时参数滑块面板自动出现。预设函数不是只能用默认参数------加载后老师可以修改表达式或调滑块。

**✅ 功能 3：参数滑块实时调节**

函数中的每个可调参数（a、b、c、k、ω、φ
等）自动生成一个水平滑块。滑块拖动时，图像**逐帧跟随更新**（不能有明显延迟）。每个滑块旁边显示当前数值，支持手动输入精确值。滑块的范围和步长根据函数类型自动设定合理默认值（如二次函数的
a 范围 \[-5, 5\]，步长 0.1），老师也可以手动修改范围。

**✅ 功能 4：多函数对比**

老师可以在同一个坐标系中添加多条函数曲线。每条曲线自动分配不同颜色（至少支持
6
种高对比度颜色），图例区显示每条函数的表达式和对应颜色。支持单独隐藏/显示某条曲线（点击图例中的颜色条即可切换）。最多同时显示
6 条曲线。

**✅ 功能 5：特征点自动标注**

开启后，系统自动在图像上标注以下特征点（用圆点+坐标标签显示）：

与 x 轴的交点（零点）

与 y 轴的交点

极大值点、极小值点

拐点（二阶导数为零的点）

标注可以一键开启/关闭，避免标注过多时图像杂乱。鼠标悬停在标注点上会显示精确坐标值（支持分数和根号形式，如零点
x = (-1+√5)/2）。

**✅ 功能 6：切线工具**

老师点击曲线上的某一个点，系统在该点画出切线（用虚线或不同颜色区分），并在旁边显示斜率值
k =
f\'(x₀)。切点可以沿曲线拖动，切线实时跟着旋转，斜率值实时更新。这是讲\"导数的几何意义\"最核心的演示功能。

**✅ 功能 7：导数可视化**

点击\"显示导函数\"按钮，系统自动计算 f(x) 的导函数
f\'(x)，并在同一个坐标系中画出 f\'(x)
的图像（用不同颜色/线型区分）。当老师拖动切点时，f(x) 上的切线和 f\'(x)
上的对应点同步联动。老师可以直观演示：f\'(x) \> 0 → f(x) 递增、f\'(x) =
0 → f(x) 极值、f\'(x) \< 0 → f(x) 递减。

**✅ 功能 8：定积分面积可视化**

老师在 x 轴上选择一个区间 \[a,
b\]（通过拖动两个竖线标记，或手动输入数值），系统自动着色显示曲线与 x
轴围成的区域，并在旁边显示面积数值（x
轴上方为正面积，下方为负面积，总面积也显示）。可选择显示黎曼和的矩形近似（n
可调），直观展示积分的\"无穷细分\"思想。

**✅ 功能 9：函数变换动画**

提供以下变换按钮，每个变换都以**动画方式**展示变换过程（不是瞬间跳变）：

上下平移：y = f(x) → y = f(x) + k

左右平移：y = f(x) → y = f(x - h)（动画展示\"左加右减\"）

关于 x 轴对称：y = f(x) → y = -f(x)

关于 y 轴对称：y = f(x) → y = f(-x)

关于原点对称：y = f(x) → y = -f(-x)

纵向伸缩：y = f(x) → y = A·f(x)

横向伸缩：y = f(x) → y = f(ωx)

老师点击变换按钮后，图像用 0.5-1
秒的动画平滑过渡到新位置。变换前后的图像可以同时保留对比（原图变浅色虚线，新图实线）。

**✅ 功能 10：分段函数**

支持分段函数的输入和绘制。老师点击\"添加分段\"按钮，可以为不同的 x
区间定义不同的表达式。界面上每个分段显示为一行：区间输入框 +
表达式输入框。系统自动处理分段点处的连续性（实心点表示取到、空心点表示取不到）。

**✅ 功能 11：参数动画播放**

老师设定某个参数的起止值和步长，点击\"播放\"按钮后，参数自动从起始值连续变化到终止值，图像跟着连续变化，形成动画效果。例如设定
a 从 -3 到 3
自动播放，学生可以完整观察抛物线从\"朝下很窄\"到\"朝下很宽\"再到\"消失\"再到\"朝上很宽\"最后到\"朝上很窄\"的完整过程。支持暂停、调速、循环播放。

**✅ 功能 12：坐标读取与网格**

鼠标在坐标系区域移动时，实时显示鼠标位置对应的坐标值（跟随鼠标的小标签）。点击曲线上的点，弹出该点的精确坐标。坐标系支持显示/隐藏网格线、调整坐标轴刻度范围、双击恢复默认视野。鼠标滚轮缩放，拖动平移。

**5. M-03 解析几何画板（规划中）**

[Untitled
page](https://kcnw8jqlswwm.feishu.cn/wiki/Bv63wzDIeiWOtMksARIczeEZnmg)

**一、这个东西是干什么的？**

解析几何是高中数学的\"大
Boss\"------特别是圆锥曲线（椭圆、双曲线、抛物线），是高考压轴题的常客。学生最头疼的不是公式不会背，而是**脑子里没有画面**：椭圆的焦点在哪？离心率变大图像怎么变？过焦点的弦旋转时弦长怎么变化？这些全靠想象力，而大多数学生想象力不够用。

老师在黑板上画椭圆，画出来的要么像鸡蛋要么像橄榄球，焦点位置全靠目测。更别提双曲线了------两支曲线画得歪歪扭扭，渐近线也对不齐。

**解析几何画板**就是让老师能画出\"标准的\"圆锥曲线，而且所有关键元素（焦点、准线、对称轴、渐近线）都自动标注。最重要的是，老师可以在上面画直线、看交点、算弦长、搞焦点弦旋转------这些高考常考的操作全部可视化。

**PM备注：**
解析几何是高考大题的重灾区（选择性必修第一册），焦点弦、弦长公式、面积计算都是常考题型。这个工具的核心价值是\"让抽象的坐标计算变成看得见的图形操作\"。离心率演示（从圆到椭圆到抛物线到双曲线的连续变化）是亮点功能，能帮助学生理解圆锥曲线的统一定义。

**二、用户是谁？他们怎么用？**

**目标用户：** 高中数学老师

**覆盖学段：**

\[高中必修\] 直线与圆的位置关系

\[高中选修\] 椭圆、双曲线、抛物线（圆锥曲线）

**典型使用场景：**

\- **场景 1 --- 讲椭圆的定义：** 老师选择\"椭圆\"，拖动一个动点
P，屏幕上实时显示 \|PF₁\| + \|PF₂\| 的值------学生看到这个和始终等于
2a，椭圆的定义就理解了。

\- **场景 2 --- 讲焦点弦（常考）：**
老师画一条过焦点的直线，拖动旋转这条直线，看到弦长在变化、焦点三角形的面积也在变化，弦垂直于长轴时（通径）弦长最短------这就是高考题的结论，学生自己\"看出来\"的比老师直接告诉的印象深
10 倍。

\- **场景 3 --- 讲离心率：** 老师从 e = 0（圆）开始拖滑块，e
慢慢增大，图形从圆变扁变成椭圆、e = 1 时变成抛物线、e \> 1
时变成双曲线------圆锥曲线的统一性就出来了。

\- **场景 4 --- 讲直线与圆的位置关系：**
老师画一个圆和一条直线，拖动直线平移或旋转，显示\"d \< r 相交 / d = r
相切 / d \> r 相离\"，d 值实时更新。

**老师对工具的期望：**

曲线要画得标准、精确，不要像黑板上那样歪歪扭扭

焦点、准线、渐近线这些要自动标，别让我算坐标再手动画

画直线要方便，输入方程或者直接在图上拉两个点

弦长、面积这些值要自动算出来，而且要和手算结果一致

**三、功能清单（老师视角描述）**

**3.1 第一步：选择曲线类型**

打开工具，老师看到曲线类型选择区，提供四种圆锥曲线 + 直线与圆：

  ----------------------- ----------------------- -----------------------
  曲线类型                标准方程                可调参数

  圆                      (x-a)² + (y-b)² = r²    圆心坐标 (a, b)、半径 r

  椭圆                    x²/a² + y²/b² = 1       半长轴 a、半短轴 b

  双曲线                  x²/a² - y²/b² = 1       实半轴 a、虚半轴 b

  抛物线                  y² = 2px                焦准距 p
  ----------------------- ----------------------- -----------------------

选好后，坐标系中立即绘制默认参数的标准曲线。

**✅ 功能 1：参数调节与曲线更新**

每个参数对应一个滑块，拖动即可实时调整曲线形状。例如椭圆的 a 从 5 拖到
3，椭圆变\"瘦\"了。参数变化时，所有关联元素（焦点位置、准线位置、离心率数值）同步更新。面板上始终显示当前的标准方程、焦点坐标、离心率
e 值、准线方程。

**✅ 功能 2：自动标注焦点、准线和关键元素**

选择曲线后，系统自动在图上标出：

焦点 F₁、F₂（用红色实心点标注，附坐标标签）

准线（用蓝色虚线标注，附方程标签）

对称轴（用灰色虚线）

顶点（用绿色点标注）

双曲线的渐近线（用橙色虚线标注，附方程标签）

所有标注可以单独开关，避免标注过多时图面拥挤。

**✅ 功能 3：直线工具**

老师可以通过以下方式在坐标系中添加直线：

输入直线方程（如 y = 2x + 1 或 x = 3）

在图上点击两个点确定一条直线

指定过某个特殊点（如焦点）并设置斜率

直线与曲线的交点自动标出（红色大圆点 +
坐标标签）。如果无交点，显示\"该直线与曲线无公共点\"。

**✅ 功能 4：弦长自动计算**

当直线与曲线有两个交点时，系统自动计算弦长并在图上标注（用双箭头线段 +
数值标签）。弦长支持精确值显示（如含根号的形式）。直线可以拖动旋转，弦长值实时更新------这是演示\"弦长公式\"最好的方式。

**✅ 功能 5：焦点弦专题演示**

选择\"焦点弦模式\"后，系统自动画一条过焦点的直线（倾斜角可调）。拖动旋转这条直线时，实时显示：

弦长 \|AB\|

\|AF\| 和 \|BF\|（焦半径）

焦点三角形面积 S△F₁AB

倾斜角 α

当弦垂直于长轴时（通径位置），弦长取最小值------系统用高亮提示这个特殊位置。

**✅ 功能 6：面积计算**

老师选定图上的封闭区域（如焦点三角形、弓形区域），系统自动计算面积并标注数值。支持用着色显示选定区域。

**✅ 功能 7：动点轨迹**

老师设定一个动点的运动规则（如\"到 F₁ 和 F₂
距离之和为常数\"或\"到定点和定直线距离相等\"），点击\"播放\"后动点开始运动，其轨迹自动描绘在坐标系上。这是讲圆锥曲线定义的核心演示功能------学生亲眼看到\"按规则运动的点画出了椭圆/抛物线\"。

**✅ 功能 8：离心率连续变化演示（亮点功能）**

提供一个离心率 e 的滑块，从 0 连续变化到 2+：

e = 0：圆

0 \< e \< 1：椭圆（e 越大越扁）

e = 1：抛物线

e \> 1：双曲线（e 越大开口越大）

这个演示能让学生理解\"圆锥曲线是一个连续的家族\"，是整个模块的亮点功能。拖动滑块时，图形在圆→椭圆→抛物线→双曲线之间平滑过渡。

**✅ 功能 9：直线与圆的位置关系**

专门针对\"直线与圆\"这个教学单元：老师画一个圆和一条直线，面板上实时显示圆心到直线的距离
d 和半径 r 的大小关系：

d \< r → 相交（显示两个交点、弦长）

d = r → 相切（显示切点、切线）

d \> r → 相离（显示最近距离）

直线可以拖动平移/旋转，位置关系随着变化实时更新。

**✅ 功能 10：光学性质可视化（选学拓展）**

选学功能，演示椭圆和抛物线的反射性质：

椭圆：从一个焦点发出的光线经椭圆反射后汇聚到另一个焦点

抛物线：平行于对称轴的光线经抛物线反射后汇聚到焦点

用动画展示光线路径，帮助学生理解为什么卫星天线是抛物面形状。

**7. M-05 概率统计模拟器（规划中）**

[Untitled

**一、这个东西是干什么的？**

概率统计是高中数学里最\"反直觉\"的部分。学生的典型困惑：

\"抛硬币正面的概率是 0.5\"------可是我抛了 10 次，6 次正面啊？

\"大数定律说频率趋近概率\"------那到底要多\"大\"才行？

\"正态分布曲线下面积 = 1\"------这什么意思？

这些概念必须通过**大量重复实验**才能真正理解。可是课堂上时间有限，抛 100
次硬币就要花好几分钟，抛 10000 次更不现实。

**概率统计模拟器**就是用计算机代替手工实验------点一下按钮，1 秒钟模拟
10000 次抛硬币，频率直方图实时更新，学生亲眼看到频率从乱七八糟逐渐稳定到
0.5 附近。这比任何说教都有效。

概率统计在新高考中比重增加，特别是必修的\"统计与概率\"和选修的\"随机变量\"部分。蒙特卡罗估算

**✅ 功能 6：正态分布可视化**

提供 μ（均值）和 σ（标准差）两个参数滑块。拖动时正态曲线实时变化：

μ 变化 → 曲线左右平移

σ 变大 → 曲线变矮变宽

σ 变小 → 曲线变高变窄

一键显示\"68-95-99.7 法则\"：在曲线下用三种颜色着色标出 \[μ-σ,
μ+σ\]（≈68.27%）、\[μ-2σ, μ+2σ\]（≈95.45%）、\[μ-3σ,
μ+3σ\]（≈99.73%）三个区间。

老师也可以手动输入任意区间 \[a,
b\]，系统计算并着色显示该区间的面积（概率）。

**✅ 功能 7：二项分布可视化**

老师设定 n（试验次数）和 p（单次成功概率），系统绘制 B(n, p)
的概率分布柱状图（横轴是成功次数 k = 0, 1, \..., n，纵轴是 P(X=k)）。

显示期望 E(X) = np 和方差 D(X) = np(1-p)。当 n
较大时，可以叠加一条正态曲线 N(np,
np(1-p))，展示\"二项分布趋近正态分布\"。

**✅ 功能 8：线性回归分析**

老师输入散点数据（支持手动点击坐标系添加点、表格输入、粘贴数据）。系统自动：

绘制散点图

计算回归直线 ŷ = bx + a 的系数（最小二乘法）

画出回归直线

显示相关系数 r 和 r² 值

标出每个点到回归直线的残差（竖线段）

老师可以手动增删散点，观察回归直线如何随数据变化而调整。支持导出回归方程和计算数据。

**✅ 功能 9：大数定律动画演示**

这是一个综合演示功能。系统从 N=1 开始逐步增加实验次数，实时绘制\"频率 vs
实验次数\"折线图。折线的波动随 N 增大而减小，最终趋于理论概率值。

支持调整动画速度（慢速逐次、中速跳 10 次、快速跳 100
次）。可以暂停在任意 N 值，查看当前的频率数值。

**8. M-06 向量运算演示台（规划中）**

[Untitled
page](https://kcnw8jqlswwm.feishu.cn/wiki/UZd7w2QYXiB1KAk9tIOc6q5FnSh)

**一、这个东西是干什么的？**

向量是高中数学里\"看起来简单但用起来难\"的内容。学生的困难主要是：

1\.
**加减法的几何意义搞不清**：平行四边形法则和三角形法则，画图的时候总搞混\"哪个箭头接哪个箭头\"。

2\. **数量积的几何意义不直观**：a⃗·b⃗ = \|a⃗\|\|b⃗\|cosθ，书上说\"等于 a⃗ 在
b⃗ 方向上的投影乘以 b⃗
的长度\"------这到底什么意思？黑板上画的投影虚线歪歪扭扭，看不清楚。

3\.
**空间向量更抽象**：到了选修课的空间向量，方向角、法向量什么的，全靠空间想象力，有些学生完全跟不上。

**向量运算演示台**让老师可以在坐标系上直接\"画向量、做运算\"------画两个向量
a⃗ 和
b⃗，点击\"加法\"按钮，平行四边形法则自动画出来，结果向量用红色标注。拖动某个向量的端点，所有运算结果实时更新。

**PM备注：**
向量分为平面向量（必修）和空间向量（选修），两部分的核心运算相同（加减、数乘、数量积），但维度不同。产品设计上用
2D/3D 模式切换来处理，底层运算引擎支持 2D 和 3D。M-01
立体几何展示台的坐标系功能和 M-06 有一定关联，可以考虑共享 3D 渲染组件。

**二、用户是谁？他们怎么用？**

**目标用户：** 高中数学老师

**覆盖学段：**

\[高中必修\] 平面向量（向量概念、加减法、数乘、数量积）

\[高中选修\] 空间向量与立体几何（空间向量运算、法向量）

**典型使用场景：**

\- **场景 1 --- 讲向量加法（平行四边形法则）：**
老师在坐标系上画两个向量 a⃗ = (3, 1) 和 b⃗ = (1,
2)。点击\"加法\"按钮，系统自动补全平行四边形，对角线就是 a⃗ + b⃗ = (4,
3)，用红色粗箭头标出。老师拖动 a⃗ 的端点，平行四边形和结果向量实时更新。

\- **场景 2 --- 讲数量积的几何意义（常考）：** 老师画 a⃗ 和
b⃗，点击\"数量积\"，系统在 b⃗ 方向上画出 a⃗ 的投影（蓝色虚线 +
投影点），显示 a⃗·b⃗ = \|a⃗\|\|b⃗\|cosθ 的计算过程和结果。拖动 a⃗
旋转，投影长度实时变化，学生看到\"夹角越大，投影越短，等于 90° 时投影为
0（垂直时数量积为 0）\"。

\- **场景 3 --- 讲空间向量法向量（选修）：** 切换到 3D
模式，老师输入一个平面的两个方向向量，系统自动计算法向量并显示。法向量垂直于平面，这在
3D 模型上一目了然。

\- **场景 4 --- 讲向量分解：** 老师画一个向量 c⃗ 和两个基底方向 e⃗₁ 和
e⃗₂，系统自动把 c⃗ 分解为 c⃗ = λ₁e⃗₁ + λ₂e⃗₂，用不同颜色画出分量。

**老师对工具的期望：**

向量的箭头要明显------不是线段，是有箭头的有向线段

运算结果的可视化要清晰------加法画平行四边形，投影画虚线

坐标和运算结果都要标出来，不能光画图没数据

3D 模式要能旋转查看，和 M-01 一样的体验

**三、功能清单（老师视角描述）**

**3.1 第一步：创建向量**

打开工具，默认显示 2D 坐标系。老师可以通过以下方式创建向量：

\-
**拖拽创建**：在坐标系上按住左键从起点拖到终点，自动生成一个带箭头的向量

\- **坐标输入**：在面板上输入起点和终点坐标（或分量坐标），点击\"创建\"

\- **预设向量**：从预设列表中选择（如\"单位向量 i⃗ = (1, 0)\"、\"45°
方向的单位向量\"等）

每个向量自动分配颜色和名字（a⃗、b⃗、c⃗......），名字可以修改。向量旁边显示分量坐标和模的长度。

**✅ 功能 1：向量拖拽与实时编辑**

老师可以拖动向量的**端点**来调整方向和长度，也可以拖动向量的**中间部分**来平移整个向量。拖动时，分量坐标和模长实时更新。也可以拖动向量的**起点**来改变起点位置（保持方向和长度不变）。所有关联的运算结果（加法、数量积等）随拖动实时更新。

**✅ 功能 2：向量加法可视化**

老师选择两个向量 a⃗ 和 b⃗，点击\"加法\"按钮，系统展示两种法则：

\- **平行四边形法则**：以 a⃗ 和 b⃗ 为邻边补全平行四边形，对角线为 a⃗ +
b⃗（红色粗箭头）

\- **三角形法则**：将 b⃗ 的起点移到 a⃗ 的终点，从 a⃗ 起点到 b⃗ 终点的向量为
a⃗ + b⃗

可以在两种法则之间切换，帮助学生理解\"同一个运算的两种几何解释\"。结果向量旁边标注分量坐标。

**✅ 功能 3：向量减法可视化**

选择两个向量 a⃗ 和 b⃗，点击\"减法\"，系统展示：

将 a⃗ 和 b⃗ 画在同一起点

从 b⃗ 终点到 a⃗ 终点的向量为 a⃗ - b⃗

同时显示 -b⃗（反方向），用 a⃗ + (-b⃗)
的三角形法则再演示一次------两种方式得到相同结果。

**✅ 功能 4：数乘向量**

面板上提供一个标量 k 的滑块（范围 \[-3, 3\]，步长 0.1）。选择向量 a⃗
后拖动 k，k·a⃗ 实时变化：

k \> 1：向量变长，方向不变

0 \< k \< 1：向量变短，方向不变

k = 0：向量变成零向量（一个点）

k \< 0：向量反向

学生可以直观看到\"数乘向量改变长度，k 的正负决定方向\"。

**✅ 功能 5：数量积（点积）与投影可视化（核心功能）**

选择 a⃗ 和 b⃗，点击\"数量积\"，系统展示：

在 b⃗ 方向上画出 a⃗ 的投影（蓝色虚线从 a⃗ 端点垂直落到 b⃗ 所在直线上）

投影点用圆点标出

投影长度用蓝色线段标注（\|a⃗\|cosθ）

旁边显示完整计算：a⃗·b⃗ = \|a⃗\|·\|b⃗\|·cosθ = 数值，同时显示 a⃗·b⃗ = x₁x₂ +
y₁y₂ = 数值

拖动 a⃗ 旋转角度时，投影实时变化。当 θ = 90° 时投影为 0，数量积 =
0------系统用高亮提示\"a⃗ ⊥ b⃗\"（这是考试重点）。

**✅ 功能 6：向量分解**

老师创建一个向量 c⃗ 和两个不共线的基底向量
e⃗₁、e⃗₂。点击\"分解\"按钮，系统自动计算 c⃗ = λ₁e⃗₁ +
λ₂e⃗₂，用不同颜色画出两个分量向量，分量向量首尾相连到 c⃗。显示系数 λ₁ 和
λ₂ 的值。

拖动 c⃗ 或基底向量，分解结果实时更新。

**✅ 功能 7：3D 向量模式**

点击\"切换3D\"按钮，坐标系从 2D 变为 3D（xyz
三轴）。支持鼠标旋转、缩放、平移（和 M-01 相同的 3D 交互）。

3D 模式下所有运算功能照常使用：加减法、数乘、数量积。额外增加：

\- **叉积（向量积）可视化**：a⃗ × b⃗ 的结果向量垂直于 a⃗ 和 b⃗
所在平面，系统画出结果向量并标出模长

\- **法向量**：给定平面上两个方向向量，自动计算并显示法向量

\- **方向角和方向余弦**：显示向量与 x、y、z 轴的夹角

**✅ 功能 8：共线与垂直判定**

老师选择两个向量，点击\"判定关系\"：

\- **共线（平行）**：如果 a⃗ = k·b⃗，显示\"a⃗ ∥ b⃗\"，标注 k
值。坐标判定条件 x₁y₂ - x₂y₁ = 0 也显示出来。

\- **垂直**：如果 a⃗·b⃗ = 0，显示\"a⃗ ⊥ b⃗\"。坐标判定条件 x₁x₂ + y₁y₂ = 0
也显示出来。

\- **既不共线也不垂直**：显示夹角 θ 值。
