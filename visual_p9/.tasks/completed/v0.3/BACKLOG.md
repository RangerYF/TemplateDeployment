## P0 — 不做产品价值有缺陷

### F01 坐标系重构
- **当前问题**：坐标系自动推断轴方向，Z 轴不一定朝上，不符合数学教学惯例；只有几何体顶点能作原点
- **目标行为**：
  1. Z 轴朝上（默认，符合教学惯例）
  2. 任意点可作原点（顶点、棱上取的点、面上取的点、坐标输入点）
  3. 交互式选轴：先选一个面确定 Z 轴法向（垂直于该面向上），再选一个方向确定 X 轴，Y 轴由右手定则自动确定
  4. 建系后各顶点自动显示坐标值（保留现有功能）
- **涉及模块**：`editor/tools/coordSystemTool.ts`、`engine/math/coordinates.ts`、`CoordSystemRenderer`、`CoordSystemInspector`
- **来源**：老师反馈
- **依赖**：F06（面交互）— 需要面可被选中才能实现"先选面确定 Z 轴"

### F02 体积表面积显示优化
- **当前问题**：所有几何体都统一显示精确值（含分数/根号），信息过载
- **目标行为**：
  1. 卡片上直接显示"体积"/"表面积"文字说明
  2. 含 π 时：显示公式和近似值，格式为 `V = 公式 ≈ 数值`（如 `V = ⅓πr²h ≈ 4.71`），不显示中间代入/化简过程
  3. 不含 π 且为整数时：直接显示精确值，格式为 `V = 值`（如 `V = 8`）
  4. 不含 π 且非整数时（含分数/根号）：只显示近似值，格式为 `V ≈ 数值`（如 `V ≈ 2.67`），不显示分数/根号等精确表达式
  5. 点击卡片可弹窗查看完整计算推导步骤（保留现有功能）
  6. 卡片底部显示"点击查看推导过程"提示文字
- **涉及模块**：`components/info/MeasurementDisplay.tsx`
- **来源**：PM 反馈 + 需求补充

### F03 三视图/展开图默认关闭
- **当前问题**：默认打开三视图和展开图面板，占用屏幕空间
- **目标行为**：
  1. 默认不显示三视图和展开图面板
  2. 面板区有"三视图"和"展开图"按钮，点击展开对应面板
  3. 再次点击或关闭按钮可收起
- **涉及模块**：`editor/store/uiStore.ts`、`components/views/`、`components/panels/`
- **来源**：PM 反馈 + 需求补充

### F04 V0.1 遗留交互优化
- **子项**：
  1. **Escape 退出绘制模式**：按 Escape 键退出任何绘制模式（画线/截面/坐标系选点），回到选择工具
     - V0.2 架构已在 `shortcuts.ts` 中定义 Escape → SelectTool，需验证是否全部生效
  2. **模式提示条**：进入画线/截面模式时，3D 场景顶部显示醒目的模式提示条（如"画线模式 — 请选择第一个点"）
  3. **画线起点高亮**：画线选第一个点后，起点出现脉冲/呼吸动画高亮
  4. **切换几何体自动清理**：切换几何体类型时，自动清理旧的用户自定义线段/截面/辅助实体
     - V0.2 的 `ChangeGeometryTypeCommand` 已实现清理，需验证是否完整
- **涉及模块**：`editor/tools/*`、`components/scene/ToolBar.tsx`、`editor/commands/changeGeometryType.ts`
- **来源**：V0.1 T6.4 遗留项

## P1 — 不做核心场景走不通

### F05 Hover 高亮
- **当前问题**：鼠标靠近线段或面时无视觉反馈，用户不知道当前指向哪个元素
- **目标行为**：
  1. 鼠标靠近线段时，线段出现高亮效果（颜色变亮或加粗）
  2. 鼠标靠近面时，面出现高亮效果（半透明色变化）
  3. 移开鼠标后恢复原样
  4. Hover 状态不影响选中状态
- **涉及模块**：`editor/store/selectionStore.ts`（增加 hoveredId）、`SegmentEntityRenderer`、`FaceEntityRenderer`
- **来源**：PM 反馈 + 需求补充

### F06 面交互
- **当前问题**：面不是可交互元素，无法选中、无法操作
- **目标行为**：
  1. 面可被左键选中（进入选中态，高亮显示）
  2. 面可被右键弹出菜单（包含"面上取点"等操作）
  3. 支持在面上取点：点击面上某个位置，创建一个约束在该面内的点
  4. 面上的点可以在面内拖动移动
  5. 面上的点可命名、可被线段连接、可参与截面/外接圆等操作
- **涉及模块**：`editor/tools/selectTool.ts`（面命中检测）、`FaceEntityRenderer`（选中高亮）、`FaceInspector`、`ContextMenu3D.tsx`
- **扩展路径**：V0.2 架构已预留，Face 已是 Entity，Point constraint 已预留 `type: 'face'`
- **来源**：PM 反馈 + 需求补充

### F07 正四面体
- **描述**：独立几何体类型，所有棱长相等
- **参数**：棱长 a（单一参数）
- **自动计算**：高 h = (√6/3)·a
- **公式**：
  - 体积：V = (√2/12)·a³
  - 表面积：S = √3·a²
  - 外接球半径：R = (√6/4)·a
  - 内切球半径：r = (√6/12)·a
- **顶点数**：4 ｜ **棱数**：6 ｜ **面数**：4
- **教学要点**：外接球半径:内切球半径 = 3:1；重心/外心/内心重合；是最简单的正多面体
- **涉及模块**：新增 `engine/builders/tetrahedronRegularBuilder.ts`、扩展 `types/geometry.ts`
- **来源**：老师反馈 + 需求文档v2 GEO-011

### F08 墙角四面体
- **描述**：三条棱两两垂直的特殊四面体（直角四面体）
- **参数**：三条直角边 a, b, c
- **公式**：
  - 体积：V = abc/6
  - 底面积（斜面）：S = ½√(a²b² + b²c² + a²c²)
  - 外接球半径：R = √(a² + b² + c²) / 2
  - 直角顶点到斜面距离：h = abc / √(a²b² + b²c² + a²c²)
- **顶点数**：4 ｜ **棱数**：6 ｜ **面数**：4
- **教学要点**：以直角顶点为原点、三条棱为坐标轴是最自然的建系方式；外接球球心在斜面上
- **涉及模块**：新增 `engine/builders/tetrahedronCornerBuilder.ts`、扩展 `types/geometry.ts`
- **来源**：老师反馈 + 需求文档v2 GEO-031

### F09 正棱柱
- **描述**：底面为正多边形的直棱柱，当前 V0.1 缺少通用棱柱类型（只有正方体和长方体）
- **参数**：底面边数 n（3~8）、底面边长 a、高 h
- **覆盖范围**：正三棱柱、正四棱柱、正五棱柱、正六棱柱等
- **公式**（以正 n 棱柱为例）：
  - 底面积：S_底 = (n/4)·a²·cot(π/n)
  - 体积：V = S_底 · h
  - 侧面积：S_侧 = n·a·h
  - 表面积：S = S_侧 + 2·S_底
- **教学要点**：正四棱柱当 h=a 时即为正方体；正六棱柱底面可分为6个正三角形
- **涉及模块**：新增 `engine/builders/prismBuilder.ts`、扩展 `types/geometry.ts`
- **来源**：需求文档v2 GEO-003/004/005
- **说明**：与现有"棱锥"类比（棱锥支持底面边数3~8），棱柱也做成可变底面边数

### F10 角度度量
- **描述**：支持三种角度的度量与可视化
- **子功能**：
  1. **二面角**：选择一条棱（或两个面），系统计算并显示二面角大小
  2. **线面角**：选择一条线段和一个面，显示线面角
  3. **线线角**：选择两条线段（可以是异面直线），显示所成角
- **输出格式**：
  - 精确值（如 arctan(√2)、45°、90°）
  - 近似值（度数，保留两位小数）
- **可视化**：在 3D 场景中用弧线标注角度，弧线旁显示角度值
- **涉及模块**：
  - 新增 EntityType `'angleMeasurement'`
  - 新增 `AngleMeasurementRenderer`、`AngleMeasurementInspector`
  - 新增 `AngleTool`（选择元素创建角度标注）
  - 新增 `engine/math/angleCalculator.ts`
- **扩展路径**：V0.2 架构验证通过（§12.1），只需注册新实体类型+渲染器+工具
- **来源**：老师反馈 + 需求文档v2 §8

### F11 输入坐标显示新点
- **前置条件**：坐标系已建立（F01）
- **目标行为**：
  1. 坐标系建立后，Inspector 面板中出现"添加坐标点"区域
  2. 输入坐标 (x, y, z)，点击确认，3D 场景中显示对应位置的点
  3. 创建的点可命名、可选中、可参与所有后续操作（画线段、做截面、度量角度等）
  4. 点的位置随坐标系变化而更新
- **实现方式**：创建 Point Entity，constraint 类型为 `'coordinate'`（V0.2 架构已预留）
- **涉及模块**：`CoordSystemInspector`、`PointEntityRenderer`（增加 coordinate 约束的位置计算）
- **来源**：老师反馈

### F12 几何体选择器分类
- **当前问题**：顶部几何体选择是扁平列表，V0.3 新增几何体后类型增多（10+），找目标困难
- **目标行为**：
  - 分类/分组显示：
    - **棱柱类**：正方体、长方体、正棱柱
    - **棱锥类**：棱锥
    - **旋转体**：圆柱、圆锥、球、圆台
    - **特殊四面体**：正四面体、墙角四面体、(P2:对棱相等/对棱垂直)
  - 可以用下拉分组、Tab 切换、或网格分类等形式
- **涉及模块**：`components/layout/TopBar.tsx`
- **来源**：延伸需求

## P2 — 做了产品更完整

### F13 对棱相等四面体（等腰四面体）
- **描述**：三组对棱分别相等的四面体
- **参数**：三组对棱长 p, q, r（AB=CD=p, AC=BD=q, AD=BC=r）
- **公式**：
  - 体积：V = (√2/12)·√((p²+q²-r²)(p²+r²-q²)(q²+r²-p²))
  - 外接球半径（p=q=r时）：R = √(p²+q²+r²) / (2√2)
- **教学要点**：可内接于长方体；当 p=q=r 时即为正四面体
- **涉及模块**：新增 `engine/builders/tetrahedronIsoscelesBuilder.ts`
- **来源**：老师反馈 + 需求文档v2 GEO-032

### F14 对棱垂直四面体
- **描述**：三组对棱分别垂直的四面体
- **参数**：六条棱长（需满足对棱垂直约束：AB²+CD² = AC²+BD² = AD²+BC²）
- **特征**：三组对棱中点连线两两垂直，可以此建立正交坐标系
- **教学要点**：正四面体是对棱垂直四面体的特例
- **涉及模块**：新增 `engine/builders/tetrahedronOrthogonalBuilder.ts`
- **来源**：老师反馈 + 需求文档v2 GEO-033
- **注意**：参数输入需要约束校验（确保满足对棱垂直条件）

### F15 圆台
- **描述**：圆锥被平行于底面的平面截去顶部后的几何体
- **参数**：上底半径 r₁、下底半径 r₂、高 h
- **公式**：
  - 体积：V = (π/3)h(r₁² + r₂² + r₁r₂)
  - 母线长：l = √((r₂-r₁)² + h²)
  - 侧面积：S_侧 = π(r₁+r₂)l
  - 表面积：S = π(r₁² + r₂² + (r₁+r₂)l)
- **教学要点**：r₁=0 退化为圆锥；r₁=r₂ 退化为圆柱
- **涉及模块**：新增 `engine/builders/frustumConeBuilder.ts`
- **来源**：需求文档v2 GEO-024

### F16 棱台
- **描述**：棱锥被平行于底面的平面截去顶部后的几何体
- **参数**：底面边数 n、上底边长 a₁、下底边长 a₂、高 h
- **公式**：
  - 体积：V = (h/3)(S₁ + S₂ + √(S₁S₂))
  - 侧面积：S_侧 = ½(C₁+C₂)·h'（h' 为斜高）
- **教学要点**：S₁=0 退化为棱锥；S₁=S₂ 退化为棱柱
- **涉及模块**：新增 `engine/builders/frustumPyramidBuilder.ts`
- **来源**：需求文档v2 GEO-034

### F17 侧棱长参数 ✅ 已完成
- **当前问题**：棱锥只能通过"底面边长 + 高"定义，高考题经常给出侧棱长作为已知条件
- **目标行为**：
  1. 棱锥参数面板增加"侧棱长"字段
  2. 支持两种定义模式：(底面边长 + 高) 或 (底面边长 + 侧棱长)
  3. 输入侧棱长后自动计算高度（h = √(l² - R²)，R 为底面外接圆半径）
  4. 两种模式互斥，切换时自动换算
- **实际实现**：同时显示高度和侧棱长两个字段，联动同步（非互斥切换），有"↕ 高度与侧棱长联动"视觉标记
- **涉及模块**：`PyramidBuilder`、`ParameterPanel` / `GeometryInspector`
- **来源**：老师反馈
- **完成时间**：2026-03-11

### F18 实体列表面板（Entity List Panel） ✅ 已完成
- **当前问题**：编辑器缺少实体管理面板，无法总览所有实体；面隐藏后无法恢复（因为不可见就不可选中）
- **目标行为**：
  1. 左侧新增实体列表面板，按类型分组展示所有实体（点、线段、面、截面、角度度量等）
  2. 点击实体名 → 选中对应实体
  3. hover 出现隐藏/锁定图标按钮
  4. 隐藏：toggle visible，隐藏的实体在列表中仍可见（灰色），可随时恢复
  5. 锁定：toggle locked，锁定的实体在场景中不可被鼠标选中/拖拽，但仍然可见
- **实际实现**：按8种类型分组（含距离度量），增加搜索功能，空分类显示"暂无xx"，距离度量显示可读名称（如 `A → 面DCBA`），面板可折叠，locked 过滤集中在 ToolEventDispatcher
- **涉及模块**：新增 `EntityListPanel`、修改 `AppLayout`、`Entity` 接口新增 `locked` 字段
- **来源**：交互讨论（原 F18 为截面切割可视化，经讨论调整为实体列表面板）
- **完成时间**：2026-03-11

### F19 距离度量 ✅ 已完成
- **描述**：度量空间中的距离，与角度度量（F10）互为补充
- **子功能**（5种距离类型）：
  1. **点到点距离**：两个点之间的直线距离
  2. **点到线距离**：点到线段的垂直距离 + 垂足直角标记
  3. **点到面距离**：点到面的垂直距离 + 垂足直角标记
  4. **线到线距离**（异面直线）：公垂线段 + 两端直角标记（平行线自动退化）
  5. **线到面距离**：需平行约束，不平行时提示用户
- **输出格式**：精确值（√n、a√b、分数有理化）+ 近似值
- **可视化**：虚线段 + 直角标记 + 距离标签 + TubeGeometry 命中体积
- **涉及模块**：
  - 新增 EntityType `'distanceMeasurement'`
  - 新增 `DistanceMeasurementRenderer`、`DistanceMeasurementInspector`
  - 新增 `engine/math/distanceCalculator.ts`
  - 新增 `editor/tools/distanceTool.ts`
- **来源**：延伸需求
- **完成时间**：2026-03-11

```
F01(坐标系重构) ←依赖← F06(面交互)     ← 面选中才能"选面定Z轴"
F01(坐标系重构) ←前置← F11(输入坐标点)  ← 需要坐标系才能输入坐标
F10(角度度量)   ←依赖← F05(Hover高亮)   ← 选择线/面时需要视觉反馈
F10(角度度量)   ←依赖← F06(面交互)      ← 线面角需要面可选中
F19(距离度量)   ←依赖← F06(面交互)      ← 点到面距离需要面可选中
F12(选择器分类) ←前置← F07/F08/F09(新几何体) ← 几何体增多后才需要分类
F18(实体列表面板) ←依赖← Entity 框架成熟
F17(侧棱长)     ←依赖← 现有棱锥
```
