## 子任务清单（串行执行）

### T5B.1 棱锥展开图

**产出文件**：`src/engine/unfolding/pyramidUnfold.ts`

**算法**：
- 输入：PyramidParams { sides (n), sideLength (a), height (h) }
- 斜面高 = √(h² + r²)，r = a / (2 × tan(π/n))（底面内切圆半径/边心距）
- 展开布局：底面正 n 边形居中，n 个等腰三角形依次向外展开（扇形排列）
- 每个三角形底边 = a，腰 = 斜面高（√(h² + r²) 时底面边心距方向，实际斜面高用斜高 l = √(h² + r²)）
- 输出：CuboidUnfoldResult 兼容格式（kind: 'polygon', faces[]）

**注意**：棱锥斜面高 ≠ 母线长。对于正 n 棱锥：
- 斜高 l_slant = √(h² + apothem²)，apothem = a / (2tan(π/n))
- 侧面三角形：底边 a，高 l_slant

**手算验证**：正四棱锥 (n=4, a=2, h=2) → apothem=1, l_slant=√5 ≈ 2.236

### T5B.2 圆柱展开图

**产出文件**：`src/engine/unfolding/cylinderUnfold.ts`

**算法**：
- 输入：CylinderParams { radius (r), height (h) }
- 侧面展开为矩形：宽 = 2πr，高 = h
- 两个底面圆：半径 = r，放在矩形上下方
- 输出：新增 CylinderUnfoldResult 类型（kind: 'cylinder'，包含矩形 + 两个圆参数）

**手算验证**：圆柱 (r=1.5, h=2) → 矩形宽 = 3π ≈ 9.42，高 = 2

### T5B.4 球截面

**修改文件**：`src/engine/math/crossSection.ts` 或 **产出文件**：`src/engine/math/sphereCrossSection.ts`

**算法**：
- 输入：3 个点（定义平面）+ 球心 (0, r, 0) + 半径 r（球 Builder positionOffset 使球心在 y=r）
- 球心到平面的距离 d = |dot(center - P0, normal)|
- 若 d ≥ r：平面不与球相交 → 返回 null
- 若 d < r：截面圆半径 R_section = √(r² - d²)
- 截面圆心 = 球心在平面上的投影点
- d = 0 时为大圆（R_section = r）

**渲染**：
- 在 CrossSection.tsx 中增加球截面分支：渲染圆环（非多边形）
- 显示截面半径

**手算验证**：球 (r=1)，水平平面过球心 → d=0, R_section=1（大圆）

### T5B.5 棱锥三视图

**修改文件**：`src/engine/projection/threeView.ts`

**算法**（正 n 棱锥，底面在 y=0，顶点 P 在 y=h）：
- 正视图（沿 -Z 看）：底边水平线 + 两条侧边到顶点 → 等腰三角形或梯形（取决于 n）
- 俯视图（沿 -Y 看）：底面正 n 边形 + 中心点（顶点投影）
- 侧视图（沿 +X 看）：类似正视图
- 虚实线：底面后半部分棱为虚线

**手算验证**：正四棱锥 (a=2, h=2) → 正视图宽=2，高=2

## 技术要点

- **棱锥展开图布局**：n 个三角形围绕底面排列，类似"花瓣展开"。可参考数学教材的标准展开方式：底面居中，每条底边向外翻出对应的三角形侧面。
- **圆柱展开图**：UnfoldingPanel 目前渲染 polygon 类型的面（UnfoldFace），圆柱需新增矩形+圆的渲染分支。可定义 CylinderUnfoldResult 与 CuboidUnfoldResult 平级。
- **球截面**：crossSection.ts 当前函数签名限定 PolyhedronResult。球是 SurfaceResult，需新增分支或独立函数。渲染上从多边形改为圆环。
- **三视图虚实线**：棱锥的后方棱线在正视图中被遮挡（虚线），需要基本的可见性判断。

---

## 风险与注意事项

1. **棱锥展开图顶点标签**：需与 Builder 输出的标签一致（P + A/B/C/...），展开后每个侧面三角形共享顶点 P
2. **圆柱展开图弧长标注**：侧面矩形宽 = 2πr，可标注为"2πr ≈ xxx"
3. **球截面交互**：球只有 1 个特征点 O，3 个截面点从哪里来？需支持在球面上取点，或用已有的自定义点（曲线上的点）
4. **UnfoldingPanel 类型适配**：新增 CylinderUnfoldResult 后，panel 的渲染逻辑需要 switch/if 分支处理不同 kind
