### 6. `src/editor/tools/createVectorTool.ts` ← **新建**

**改动内容：** 两步状态机工具：
- Step 1（selectStart）：点击 point 实体 → 记录起点，高亮选中
- Step 2（selectEnd）：点击另一个 point 实体 → 执行 CreateEntityCommand('vector', …) → 切回 select
- 非 point 实体：通知提示；起终点相同：通知提示
- Escape 键：Step2 回到 Step1，Step1 退出工具
- `getNextVectorLabel()` 自动生成 a, b, c … z, a₁, b₁ …
- `getNextVectorColor()` 从 5 色预设中循环选取

**改动理由：** 与现有工具架构（angleTool、coordSystemTool）保持一致，降低维护成本。

**对应需求：** 在几何体顶点间创建向量。

---

### 7. `src/editor/tools/normalVectorTool.ts` ← **新建**

**改动内容：** 一步工具 + 内联 CreateNormalVectorCommand：
- 点击 face 实体 → 计算面法向量（cross product 归一化）和面重心（顶点平均坐标）
- `CreateNormalVectorCommand.execute()` 创建：起点（重心，free 约束）+ 终点（重心 + 法向 × 1.5，free 约束）+ vector 实体
- `CreateNormalVectorCommand.undo()` 逆序删除三实体；redo 路径 restoreEntity
- 标签：n, n₁, n₂ …；颜色：固定青色 #4ECDC4

**改动理由：** 法向量创建需原子性撤销（3实体一组），故使用自定义 Command 而非 BatchCommand，与 coordSystemTool 的原子创建模式对齐。

**对应需求：** 平面法向量显示。
