# V0.5 演示清单

v0.1~v0.4 已演示过的功能不再重复。本清单仅覆盖 v0.5 新增的功能：用户系统、作品管理、编辑器持久化、模板体系。

**标记说明**
- 【核心】= 版本核心功能（用户系统 & 作品管理）
- 【增强】= 在核心基础上的体验增强
- 【预留】= 为未来功能预留的 UI 占位

---

## 一、登录与鉴权

1. 【核心】**登录页** — 居中卡片布局，输入用户名密码完成登录
   - 演示：打开应用 → 自动跳转登录页 → 输入用户名密码 → 点击登录 → 跳转工作台
2. 【核心】**登录失败提示** — 输入错误密码时显示友好错误提示
   - 演示：输入错误密码 → 登录按钮下方显示红色错误提示
3. 【核心】**路由守卫** — 未登录访问任何页面自动重定向到登录页
   - 演示：未登录状态直接访问 `/workspace` 或 `/editor` → 被拦截跳转登录页
4. 【核心】**已登录重定向** — 已登录用户访问登录页自动跳转工作台
   - 演示：已登录状态访问 `/login` → 自动跳转 `/workspace`
5. 【核心】**登出** — 编辑器顶栏和工作台均可登出
   - 演示：点击登出按钮 → Token 清除 → 跳转回登录页
6. 【核心】**Token 过期自动登出** — 401 响应时自动清除登录态
   - 演示：Token 失效后的 API 请求自动触发登出跳转
7. 【核心】**注册功能** — 登录页底部有"去注册"链接，可完成新用户注册
   - 演示：点击"没有账号？去注册" → 进入注册页 → 填写信息完成注册 → 跳转登录

---

## 二、工作台 — 作品管理

8. 【核心】**作品列表展示** — 卡片形式展示用户所有作品（缩略图 + 名称 + 更新时间）
   - 演示：登录后进入工作台 → 查看作品卡片网格
9. 【核心】**搜索作品** — 按名称模糊搜索，300ms 防抖
   - 演示：在搜索框输入关键词 → 列表实时过滤
10. 【核心】**排序功能** — 支持按最近更新/最近创建/名称排序
    - 演示：切换排序方式 → 列表顺序变化
11. 【核心】**分页浏览** — 底部分页控件（上一页/下一页/页码/总数）
    - 演示：作品较多时翻页浏览
12. 【核心】**打开作品** — 点击卡片跳转编辑器加载对应作品
    - 演示：点击一个作品卡片 → 跳转 `/editor/{id}` → 编辑器恢复该作品场景
13. 【核心】**删除作品** — 二次确认后删除，列表刷新
    - 演示：点击卡片右上角更多按钮 → 删除 → 确认弹窗 → 作品从列表消失
14. 【增强】**空状态提示** — 无作品或搜索无结果时有友好提示
    - 演示：清空搜索结果或新用户无作品时展示空状态

## 三、工作台 — 模板推荐

15. 【核心】**模板区展示** — 工作台上方展示几何体模板卡片，按分组排列（棱柱/棱锥棱台/旋转体/四面体）
    - 演示：工作台上方查看 13 种几何体模板卡片的分组布局
16. 【核心】**使用模板创建作品** — hover 模板卡片显示"使用模板"按钮，点击后创建作品并进入编辑器
    - 演示：hover "圆锥"卡片 → 点击"使用模板" → 创建名为"圆锥"的作品 → 编辑器显示圆锥（非默认正方体）

## 四、工作台 — 多学科模板体系

17. 【核心】**学科 Tab 切换** — 数学/物理/化学三学科 Tab
    - 演示：点击"数学"/"物理"/"化学" Tab → 切换展示对应学科的模块和模板

    - 演示：数学 Tab 下查看"M-01 立体几何展示台""M-02 函数图像实验室"等模块分组

    - 演示：M-01 模板可点击使用；切到 M-02 或物理/化学 Tab → 卡片灰度，hover 提示"即将上线"

    - 演示：查看顶部"描述你想要的教学场景..."输入框 + "AI 助手即将上线"提示

## 五、编辑器持久化

21. 【核心】**加载已有作品** — 从工作台打开作品后，编辑器恢复完整场景（几何体、辅助线、截面、标注等）
    - 演示：在工作台打开一个之前编辑过的作品 → 编辑器完整还原之前的状态
22. 【核心】**手动保存** — 顶栏保存按钮 + Ctrl/Cmd+S 快捷键
    - 演示：编辑场景后 → 点击保存按钮（或 Ctrl+S） → 数据写入后端
23. 【核心】**保存状态指示** — 五种状态：未保存/保存中/已保存/保存失败/空闲
    - 演示：编辑后观察"未保存" → 点保存 → "保存中..." → "✓ 已保存" → 几秒后恢复
24. 【核心】**定时自动保存** — 检测到变更后 30 秒自动保存
    - 演示：编辑场景 → 等待约 30 秒 → 观察自动触发保存（网络面板可见 PUT 请求）
25. 【核心】**离开页面保存** — 关闭/刷新页面或路由返回工作台时触发保存
    - 演示：编辑后直接点返回工作台 → 变更自动保存
26. 【核心】**缩略图同步** — 保存时自动截取 3D 视图作为作品缩略图
    - 演示：编辑 → 保存 → 返回工作台 → 作品卡片缩略图已更新为最新 3D 视图
27. 【核心】**作品名称编辑** — 编辑器顶栏可内联编辑作品名称
    - 演示：点击顶栏作品名称 → 变为输入框 → 修改名称 → 失焦保存 → 返回工作台名称已更新
28. 【增强】**编辑器返回按钮** — 名称左侧有返回工作台箭头
    - 演示：点击返回箭头 → 回到工作台
29. 【核心】**新建作品完整流程** — 使用模板新建 → 编辑 → 保存 → 返回 → 重新打开，数据完整
    - 演示：使用模板创建 → 添加辅助线和截面 → 保存 → 返回工作台 → 再次打开 → 场景完整恢复

## 综合演示建议（串联场景）

**场景A：首次用户完整流程（5分钟）**

1. 打开应用 → 自动跳转登录页
2. 输入用户名密码登录 → 进入工作台
3. 查看学科 Tab（数学/物理/化学），切换浏览
4. 在数学 Tab 下选择"正四面体"模板 → 点击"使用模板"
5. 进入编辑器，正四面体已创建
6. 添加辅助线、建截面、度量角度
7. Ctrl+S 手动保存 → 观察保存状态变化
8. 修改作品名称为"正四面体截面演示"
9. 点击返回箭头 → 回到工作台 → 作品卡片显示缩略图和新名称

**场景B：作品管理操作（3分钟）**

1. 在工作台搜索框输入关键词 → 列表过滤
2. 切换排序方式（最近更新/名称）→ 列表顺序变化
3. 点击已有作品 → 编辑器恢复完整场景
4. 编辑后等待 30 秒 → 自动保存触发
5. 返回工作台 → 删除一个作品（确认弹窗）→ 列表刷新

**场景C：多学科模板体系展示（2分钟）**

1. 数学 Tab → 展示 M-01 的 13 种可用模板 + M-02~M-06 即将上线
2. 物理 Tab → 展示 P-01~P-14 模块，全部即将上线灰度状态
3. 化学 Tab → 展示 C-02~C-09 模块，全部即将上线灰度状态
4. 查看 AI 意图输入框占位
5. 回到数学 Tab → 使用一个模板进入编辑器

## 附带体验优化汇总

以下为开发过程中顺带完成的体验优化：

**1. 几何体 SVG 图标重绘**
- 全部 13 种几何体 SVG 图标增加虚线遮挡棱表达，3D 立体感显著增强

**2. 模板区布局优化**
- 分组竖排标签 + 卡片横向紧凑排列，消除右侧空白

**3. 保存按钮自身状态反馈**
- 按钮自身显示「保存中...」/「✓ 已保存」/「✕ 保存失败」状态

**4. 作品名称编辑暗示**
- 名称旁常驻铅笔图标，暗示可编辑

**5. 编辑器返回按钮**
- 名称左侧新增返回工作台箭头，提升导航便利性
