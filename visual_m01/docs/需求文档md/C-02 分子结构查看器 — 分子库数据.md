\> **关联模块：** C-02 分子结构查看器

**1. 数据结构说明**

  ----------------------- ----------------------- -----------------------
  字段                    类型                    说明

  id                      string                  编号 MOL-XXX

  name_cn                 string                  中文名

  name_en                 string                  英文名

  formula                 string                  分子式

  level                   string                  学段

  vsepr                   string                  VSEPR 构型

  geometry                string                  分子空间构型

  central_atom            string                  中心原子

  bond_pairs              number                  成键电子对数

  lone_pairs              number                  孤电子对数

  bond_angle              object                  各键角 (°)

  polarity                string                  极性/非极性

  hybridization           string                  杂化方式

**6.2 键的渲染规则**

  ----------------------- ------------------------ -----------------------
  键类型                  渲染方式                 说明

  单键                    单根圆柱体               标准渲染

  双键                    两根平行圆柱体           间距为键直径的 1.5 倍

  三键                    三根圆柱体（三角排列）   等间距

  离域键（苯环）          虚线环或半透明环         放在六元环中心

  配位键                  箭头形圆柱               箭头指向受体

  氢键                    虚线连接                 O-H···O 等
  ----------------------- ------------------------ -----------------------
