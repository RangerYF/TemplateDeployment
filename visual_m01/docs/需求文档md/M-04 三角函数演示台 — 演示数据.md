**教学要点：**

单位圆是理解三角函数的核心工具

**5. 公式库**

**TRI-031 基本恒等式**

  ----------------------- ----------------------- -----------------------
  编号                    公式                    名称

  1                       sin²α + cos²α = 1       平方关系

  2                       tanα = sinα / cosα      商数关系

  3                       1 + tan²α = sec²α       平方关系（扩展）

  4                       1 + cot²α = csc²α       平方关系（扩展）
  ----------------------- ----------------------- -----------------------

**TRI-032 诱导公式**

  ----------------- ----------------- ----------------- -----------------
  公式组            sin               cos               tan

  α + 2kπ           sinα              cosα              tanα

  π + α             -sinα             -cosα             tanα

  -α                -sinα             cosα              -tanα

  π - α             sinα              -cosα             -tanα

  π/2 - α           cosα              sinα              cotα

  π/2 + α           cosα              -sinα             -cotα
  ----------------- ----------------- ----------------- -----------------

**记忆口诀：** \"奇变偶不变，符号看象限\"

\"奇偶\"指 π/2 的倍数是奇数还是偶数

\"变\"指 sin ↔ cos 互变

\"符号\"将 α 看作锐角，确定原式的符号

**TRI-033 和差公式**

  ----------------------------------- -----------------------------------
  公式                                表达式

  sin(α + β)                          sinα cosβ + cosα sinβ

  sin(α - β)                          sinα cosβ - cosα sinβ

  cos(α + β)                          cosα cosβ - sinα sinβ

  cos(α - β)                          cosα cosβ + sinα sinβ

  tan(α + β)                          (tanα + tanβ) / (1 - tanα tanβ)

  tan(α - β)                          (tanα - tanβ) / (1 + tanα tanβ)
  ----------------------------------- -----------------------------------

**TRI-034 二倍角公式**

  ----------------------------------- -----------------------------------
  公式                                表达式

  sin 2α                              2 sinα cosα

  cos 2α                              cos²α - sin²α = 2cos²α - 1 = 1 -
                                      2sin²α

  tan 2α                              2tanα / (1 - tan²α)
  ----------------------------------- -----------------------------------

**降幂公式（由二倍角推导）：**

  ----------------------------------- -----------------------------------
  公式                                表达式

  sin²α                               (1 - cos 2α) / 2

  cos²α                               (1 + cos 2α) / 2
  ----------------------------------- -----------------------------------

**TRI-035 辅助角公式**

  ----------------------------------- -----------------------------------
  参数                                公式

  标准形式                            a sinx + b cosx = √(a² + b²)
                                      sin(x + φ)

  其中                                tanφ = b/a

  cosφ = a/√(a² + b²)                 sinφ = b/√(a² + b²)
  ----------------------------------- -----------------------------------

**可视化演示：**

左侧：分别画 y = a sinx 和 y = b cosx

右侧：画合成后的 y = √(a² + b²) sin(x + φ)

动画展示两个分量如何合成为一个正弦波

**TRI-036 和差化积 / 积化和差**

**和差化积：**

  ----------------------------------- -----------------------------------
  公式                                表达式

  sinα + sinβ                         2 sin((α+β)/2) cos((α-β)/2)

  sinα - sinβ                         2 cos((α+β)/2) sin((α-β)/2)

  cosα + cosβ                         2 cos((α+β)/2) cos((α-β)/2)

  cosα - cosβ                         -2 sin((α+β)/2) sin((α-β)/2)
  ----------------------------------- -----------------------------------

**积化和差：**

  ----------------------------------- -----------------------------------
  公式                                表达式

  sinα cosβ                           (1/2)\[sin(α+β) + sin(α-β)\]

  cosα sinβ                           (1/2)\[sin(α+β) - sin(α-β)\]

  cosα cosβ                           (1/2)\[cos(α+β) + cos(α-β)\]

  sinα sinβ                           -(1/2)\[cos(α+β) - cos(α-β)\]
  ----------------------------------- -----------------------------------

**6. 解三角形**

**TRI-041 正弦定理**

  ----------------------------------- -----------------------------------
  参数                                公式

  定理                                a/sinA = b/sinB = c/sinC = 2R

  其中                                R 为外接圆半径
  ----------------------------------- -----------------------------------

**应用场景：**

  ----------------------- ----------------------- -----------------------
  已知条件                可求                    注意事项

  两角一边 (AAS/ASA)      第三角和其他边          唯一解

  两边一对角 (SSA)        另一角和第三边          可能两解（模糊情形）
  ----------------------- ----------------------- -----------------------

**TRI-042 余弦定理**

  ----------------------------------- -----------------------------------
  参数                                公式

  定理                                c² = a² + b² - 2ab cosC

  推论                                cosC = (a² + b² - c²) / (2ab)
  ----------------------------------- -----------------------------------

**应用场景：**

  ----------------------- ----------------------- -----------------------
  已知条件                可求                    注意事项

  三边 (SSS)              三角                    唯一解

  两边及夹角 (SAS)        第三边和其他角          唯一解
  ----------------------- ----------------------- -----------------------

**TRI-043 面积公式**

  ----------------------- ------------------------ -----------------------
  公式                    表达式                   适用条件

  基本公式                S = (1/2)ah              已知底和高

  两边夹角                S = (1/2)ab sinC         已知两边和夹角

  海伦公式                S =                      已知三边
                          √(s(s-a)(s-b)(s-c))，s = 
                          (a+b+c)/2                

  内切圆                  S = rs，s = (a+b+c)/2    已知内切圆半径

  外接圆                  S = abc / (4R)           已知外接圆半径
  ----------------------- ------------------------ -----------------------

面积公式 S = (1/2)ab sinC 是最常用的
