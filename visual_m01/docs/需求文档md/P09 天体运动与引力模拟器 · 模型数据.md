**2. 单星系统与开普勒定律**

**CEL-001 圆轨道运行**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  中心天体质量 M 6.0×10²⁴       10²²\~10³⁰     ---            kg

  轨道半径 r     6.8×10⁶        10⁶\~10⁹       ---            m
  -------------- -------------- -------------- -------------- --------------

圆轨道公式：

  ----------------------- ----------------------- -----------------------
  物理量                  公式                    与 r 的关系

  速度 v                  sqrt(GM/r)              r 越大，v 越小

  角速度 ω                sqrt(GM/r³)             r 越大，ω 越小

  周期 T                  2π sqrt(r³/(GM))        r 越大，T 越大

  加速度 a                GM/r²                   r 越大，a 越小
  ----------------------- ----------------------- -----------------------

\"高轨低速大周期\"规律

**CEL-002 椭圆轨道运行**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  半长轴 a       1.5×10⁸        10⁷\~10⁹       ---            km

  离心率 e       0.3            0.01\~0.95     0.01           ---
  -------------- -------------- -------------- -------------- --------------

开普勒三定律：

1\. **第一定律**：行星轨道为椭圆，中心天体在焦点上

2\. **第二定律**：面积速率 dA/dt = 常数 → 近日点速度大，远日点速度小

3\. **第三定律**：T²/a³ = 4π²/(GM) = 常数

近日点/远日点数据：

  ----------------------- ----------------------- -----------------------
  位置                    距离                    速度关系

  近日点                  r_近 = a(1-e)           v_近 最大

  远日点                  r_远 = a(1+e)           v_远 最小

  v_近/v_远 = r_远/r_近 =                         
  (1+e)/(1-e)                                     
  ----------------------- ----------------------- -----------------------

面积定律演示：

每隔固定时间 Δt 画一个扇形

所有扇形面积相等（阴影区域验证）

**3. 卫星变轨**

**CEL-011 霍曼转移轨道**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  低轨半径 r1    6.8×10⁶        6.4×10⁶\~10⁷   10⁴            m

  高轨半径 r2    4.2×10⁷        10⁷\~10⁸       10⁵            m

  地球质量 M     6.0×10²⁴       ---            ---            kg
  -------------- -------------- -------------- -------------- --------------

变轨步骤：

近地圆轨道（半径 r1）→ 加速 → 椭圆转移轨道

椭圆轨道远地点 → 再次加速 → 高轨圆轨道（半径 r2）

各轨道速度：

  ----------------------- ----------------------- -----------------------
  轨道                    位置                    速度公式

  低圆轨道                全程                    v1 = sqrt(GM/r1)

  椭圆转移轨道            近地点                  v_A \> v1（加速）

  椭圆转移轨道            远地点                  v_B \< v2

  高圆轨道                全程                    v2 = sqrt(GM/r2)
  ----------------------- ----------------------- -----------------------

速度大小关系：v_A \> v1 \> v2 \> v_B

**4. 双星系统**

**CEL-021 双星绕转**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  星1质量 m1     2.0×10³⁰       10²⁸\~10³²     ---            kg

  星2质量 m2     1.0×10³⁰       10²⁸\~10³²     ---            kg

  两星距离 L     10⁸            10⁶\~10¹⁰      ---            km
  -------------- -------------- -------------- -------------- --------------

质心条件：m1 r1 = m2 r2，r1 + r2 = L

各自轨道半径：r1 = m2 L/(m1+m2)，r2 = m1 L/(m1+m2)

角速度相同：ω1 = ω2 = ω

引力提供向心力：Gm1m2/L² = m1 ω² r1 = m2 ω² r2

周期：T = 2π sqrt(L³/(G(m1+m2)))

质量比 vs 轨道半径比：

  ----------------------- ----------------------- -----------------------
  m1:m2                   r1:r2                   说明

  1:1                     1:1                     等质量，绕中点

  1:2                     2:1                     轻星轨道大

  1:3                     3:1                     轻星轨道更大

  1:10                    10:1                    接近单星系统
  ----------------------- ----------------------- -----------------------

教学要点：

质量大的星靠近质心，轨道半径小

两星角速度/周期相同

调节质量比实时看轨道变化

**5. 天体追及**

**CEL-031 不同轨道追及**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  内轨半径 r1    6.8×10⁶        10⁶\~10⁸       ---            m

  外轨半径 r2    1.0×10⁷        10⁶\~10⁸       ---            m

  初始角度差 Δθ  60             0\~350         1              °
  -------------- -------------- -------------- -------------- --------------

内轨角速度大于外轨：ω1 \> ω2

追及条件：(ω1 - ω2) t = 2kπ + Δθ

最短追及时间：t_min = (2π + Δθ)/(ω1 - ω2)（追 1 圈多 Δθ）

或 t_min = Δθ/(ω1 - ω2)（第一次相遇）

特殊情况：星下点追击问题

卫星在轨道上运行，地面站在下方

需考虑地球自转

**7. 公式汇总**

  ----------------------------------- -----------------------------------
  公式                                说明

  F = GMm/r²                          万有引力

  GMm/r² = mv²/r                      圆轨道引力=向心力

  v = sqrt(GM/r)                      圆轨道速度

  T = 2π sqrt(r³/(GM))                圆轨道周期

  T²/a³ = 4π²/(GM)                    开普勒第三定律

  v1 r_近 = v2 r_远                   角动量守恒（椭圆）
  ----------------------------------- -----------------------------------
