# 作品数据梳理思路

## 一、背景与目标

AI 推荐功能需要一批"种子作品"作为推荐池。用户在工作台输入自然语言（知识点、题目、可视化需求），LLM 理解意图后从作品池中匹配最合适的作品返回。

作品 = 模板的实例化（class vs object），每个作品是一个可直接加载的完整 3D 场景。

**第一版范围**：仅覆盖"数学 - M01 立体几何"模块。

---

## 二、梳理路径

```
知识点体系 → 常见题型 → 所需几何场景 → 作品清单
```

自顶向下：先穷举知识点，再看每个知识点对应什么题型，每种题型需要什么几何体和辅助元素，最终得出具体作品。

## 三、高中立体几何知识体系

### 3.1 知识模块总览

| 编号 | 模块 | 核心知识点 | 高考权重 |
|------|------|-----------|---------|
| K1 | 空间几何体认知 | 柱/锥/台/球的结构特征、截面形状 | 基础 |
| K2 | 三视图与直观图 | 三视图还原几何体、直观图面积比 | 选填常考 |
| K3 | 表面积与体积 | 柱/锥/台/球的公式、组合体、截断体 | 必考 |
| K4 | 空间点线面位置关系 | 平行/垂直的判定与性质（8大定理） | 大题核心 |
| K5 | 空间角 | 线线角、线面角、二面角 | 大题高频 |
| K6 | 空间距离 | 点面距、线面距、异面直线距离 | 大题高频 |
| K7 | 空间向量法 | 建系、法向量、向量求角/距离 | 大题主流解法 |
| K8 | 外接球/内切球 | 几何体与球的关系、球的截面 | 选填难题 |
| K9 | 展开图 | 侧面展开、最短路径 | 选填中频 |

### 3.2 常见题型 → 所需几何场景

#### K1/K3：表面积与体积

| 题型 | 典型场景 |
|------|---------|
| 基本公式应用 | 正方体、长方体、圆柱、圆锥、球 |
| 棱锥体积 | 正三/四棱锥 |
| 棱台体积 | 正三/四棱台（棱锥截断） |
| 圆台体积 | 圆台 |
| 组合体 | 圆柱+圆锥、半球+圆柱（当前不支持组合体，暂跳过） |

#### K4：空间位置关系证明（大题第一问）

| 题型 | 典型场景 |
|------|---------|
| 线面平行判定 | 正方体（取中点连线 ∥ 对面） |
| 面面平行判定 | 正三棱柱（中位面） |
| 线面垂直判定 | 正三/四棱锥（顶点到底面中心） |
| 面面垂直判定 | 长方体（相邻面）、正三棱锥+底面 |

#### K5：空间角（大题第二问）

| 题型 | 典型场景 |
|------|---------|
| 二面角 | 正方体（对角面）、正三棱锥（侧面与底面）、四棱锥 |
| 线面角 | 正四棱锥（侧棱与底面）、长方体（体对角线与底面） |
| 异面直线角 | 正方体（体对角线与棱）、正三棱柱（异面棱） |

#### K6：空间距离

| 题型 | 典型场景 |
|------|---------|
| 点到面距离 | 正四面体（顶点到对面）、四棱锥 |
| 异面直线距离 | 正方体（异面棱）、正三棱柱 |
| 线到面距离 | 长方体（棱到对面） |

#### K7：向量法

| 题型 | 典型场景 |
|------|---------|
| 建立坐标系 | 正方体（顶点建系）、四棱锥（底面中心建系） |
| 法向量求二面角 | 正三棱柱、正四棱锥 |
| 向量求线面角 | 长方体、正三棱锥 |

#### K8：外接球

| 题型 | 典型场景 |
|------|---------|
| 正方体外接球 | 正方体 + 外接球 |
| 正四面体外接球 | 正四面体 + 外接球 |
| 长方体外接球 | 长方体 + 外接球 |
| 棱锥外接球 | 正三/四棱锥 + 外接球 |

#### K9：截面与展开图

| 题型 | 典型场景 |
|------|---------|
| 截面形状 | 正方体截面（三角形/四边形/五边形/六边形） |
| 展开图最短路径 | 圆柱展开、圆锥展开、正三棱柱展开 |

## 七、数据生成方案

### 7.1 每个作品包含两部分

**scene_data（JSON）**：编辑器可直接加载的场景快照
- entities、nextId、activeGeometryId
- 通过生成脚本调用 builder 函数批量产出

**meta（JSON）**：用于 LLM 匹配的结构化描述

```typescript
interface WorkMeta {
  id: string                    // 唯一标识，如 "cube-basic"
  title: string                 // 作品标题，如 "正方体 — 基础认知"
  description: string           // 自然语言描述
  subject: string               // 学科："数学"
  module: string                // 模块："M01-立体几何"
  geometryType: string          // 几何体类型："cube"
  sceneType: string             // 场景类型编码："S01"
  tags: string[]                // 知识点标签：["正方体", "结构特征", "棱长"]
  difficulty: 'basic' | 'intermediate' | 'advanced'
  sceneFile: string             // scene_data 文件路径
}
```

### 7.2 生成方式

| 层级 | 内容 | 生成方式 |
|------|------|---------|
| 基础层 | geometry + points + segments + faces | 脚本调用 builder 函数自动生成 |
| 轻量增强 | 外接球 (circumSphere) | 脚本添加 circumSphere 实体 |
| 轻量增强 | 自定义线段（对角线/中点连线） | 脚本引用已有顶点 ID 构造 |
| 复杂增强 | 截面/度量/坐标系 | 需引擎计算，视可行性逐步加入 |

### 7.3 文件组织

```
src/data/works/
├── index.ts                    # 作品注册表（导出所有 meta + scene_data）
├── meta/                       # 作品 meta 信息
│   ├── cube.ts                 # 正方体系列 meta
│   ├── pyramid.ts              # 棱锥系列 meta
│   └── ...
└── scenes/                     # scene_data JSON 文件
    ├── cube-basic.json
    ├── cube-diagonal.json
    ├── cube-circumsphere.json
    └── ...
```
