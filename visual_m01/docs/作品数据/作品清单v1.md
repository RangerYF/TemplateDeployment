| 作品 ID |

| cube-S01-1, cube-S12-1 |
| pyramid4-S01-1, pyramid4-S12-1 |
| pyramid3-S01-1, pyramid3-S12-1 |
| prism3-S01-1, prism3-S11-1, prism3-S12-1 |
| cuboid-S01-1, cuboid-S12-1 |
| cylinder-S01-1, cylinder-S11-1 |
| cone-S01-1, cone-S11-1 |
| sphere-S01-1 |
| regTet-S01-1 |

### B 级 — 脚本自动生成，建议抽查校验（14 个）

场景类型 S02（对角线）、S03（外接球）、S13（中点连线）：需要引用顶点 ID 创建额外线段或 circumSphere 实体。

| 作品 ID |
|---------|
| cube-S02-1, cube-S03-1, cube-S13-1 |
| cuboid-S02-1, cuboid-S03-1 |
| prism3-S02-1, prism3-S13-1 |
| pyramid4-S03-1, pyramid3-S03-1 |
| cylinder-S03-1, cone-S03-1 |
| regTet-S03-1 |

**抽查方式**：在编辑器中导入 JSON，目视确认线段连接正确、外接球位置正确。建议从每种几何体各抽 1 个。

### C 级 — 需引擎计算，脚本生成后必须校验（18 个）

场景类型 S04~S10：涉及截面、角度度量、距离度量、坐标系。

| 作品 ID |
|---------|
| cube-S04-1, cube-S05-1, cube-S06-1, cube-S07-1, cube-S09-1, cube-S10-1 |
| pyramid4-S05-1, pyramid4-S06-1, pyramid4-S08-1, pyramid4-S10-1 |
| pyramid3-S05-1, pyramid3-S08-1, pyramid3-S10-1 |
| prism3-S04-1, prism3-S05-1, prism3-S07-1, prism3-S09-1, prism3-S10-1 |
| cuboid-S06-1, cuboid-S07-1, cuboid-S09-1, cuboid-S10-1 |
| regTet-S08-1 |
| sphere-S04-1 |

**校验方式**：编辑器导入后确认度量数值正确、截面形状正确、坐标系方向正确。**这部分需要你在编辑器里逐个打开确认。**
