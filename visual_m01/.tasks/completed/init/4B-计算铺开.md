## 各几何体公式速查

### 棱锥（正 n 棱锥）— `pyramid`
参数：`sides (n)`, `sideLength (a)`, `height (h)`

- 底面内切圆半径（边心距）：`r = a / (2 × tan(π/n))`
- 底面外接圆半径：`R_底 = a / (2 × sin(π/n))`
- 底面积：`S_底 = (n × a² ) / (4 × tan(π/n))`
- 斜高：`l = √(h² + r²)`
- 侧面积：`S_侧 = (1/2) × n × a × l`
- 体积：`V = (1/3) × S_底 × h`
- 全面积：`S = S_底 + S_侧`

**精确值处理**：
- n=3：`tan(π/3) = √3`，`S_底 = (√3/4)a²`，可给出含 √3 的精确值
- n=4：`tan(π/4) = 1`，`S_底 = a²`，纯数值精确
- n=6：`tan(π/6) = √3/3`，`S_底 = (3√3/2)a²`
- 其他 n：tan(π/n) 为无理数，使用数值近似

**外接球**：
- 球心在轴上，距底面高度 `y = (h² - R_底²) / (2h)`
- `R_球 = (h² + R_底²) / (2h)`

### 正方体 — `cube`
参数：`sideLength (a)`

- 直接复用 `calculateCuboid({ length: a, width: a, height: a })`
- V = a³，S = 6a²
- 外接球已在 4A 中支持（circumscribedSphere.ts 已有 cube 分支）

### 圆柱 — `cylinder`
参数：`radius (r)`, `height (h)`

- V = πr²h
- S_侧 = 2πrh
- S_底 = πr²（×2）
- S = 2πr² + 2πrh = 2πr(r + h)

**外接球**：
- 球心 = `(0, h/2, 0)`
- `R_球 = √(r² + h²/4)`

### 球 — `sphere`
参数：`radius (r)`

- V = (4/3)πr³
- S = 4πr²

**外接球**：球本身即为自身的外接球，不单独渲染。

### 圆锥 — 已实现，补充外接球
参数：`radius (r)`, `height (h)`

**外接球**：
- 球心在轴上，`y = (h² - r²) / (2h)`
- `R_球 = (h² + r²) / (2h)`

## 子任务清单（串行执行）

### T4B.1 棱锥计算器

**产出文件**：`src/engine/math/calculators/pyramid.ts`

**要点**：
- 从 `params` 取 `sides`, `sideLength`, `height`
- n=3 和 n=4 提供含 √3 的精确值；n=6 含 √3；其他 n 用数值
- 分步过程需展示底面积 → 体积 → 侧面积 → 全面积的推导

**手算验证**：
- 正四棱锥 (n=4, a=2, h=2)：S_底=4, V=8/3≈2.67, r=1, l=√5, S_侧=4√5, S=4+4√5≈12.94
- 正三棱锥 (n=3, a=2, h=2)：S_底=√3, V=2√3/3≈1.15

### T4B.2 正方体计算器

**产出文件**：`src/engine/math/calculators/cube.ts`

**要点**：
- 内部调用 `calculateCuboid({ length: a, width: a, height: a })`
- 重写 steps 使用 a（边长）而非 l/w/h 表述

**手算验证**：
- a=2：V=8, S=24

### T4B.3 圆柱计算器

**产出文件**：`src/engine/math/calculators/cylinder.ts`

**要点**：
- 参考 cone.ts 的 π 处理模式（simplifyPiFraction 等）
- 分步：V 公式 → 代入 → 结果；S 分底+侧 → 合计

**手算验证**：
- r=1.5, h=2：V=4.5π≈14.14, S_侧=6π, S_底=2.25π×2=4.5π, S=10.5π≈32.99

### T4B.4 球计算器

**产出文件**：`src/engine/math/calculators/sphere.ts`

**要点**：
- V = (4/3)πr³ → piMulFrac(4r³, 3)
- S = 4πr² → piMul(4r²)

**手算验证**：
- r=1：V=4π/3≈4.19, S=4π≈12.57

### T4B.5 注册全部计算器

**修改文件**：`src/engine/math/index.ts`

**动作**：import 4 个新计算器，注册到 `calculators` map（cube 复用 cuboid 则注册 cube 条目）

### T4B.6 扩展外接球

**修改文件**：`src/engine/math/circumscribedSphere.ts`

**新增支持**：
- `pyramid`：正 n 棱锥外接球（球心在轴上，R = (h² + R_底²) / (2h)）
- `cylinder`：R = √(r² + h²/4)，球心 = (0, h/2, 0)
- `cone`：R = (h² + r²) / (2h)，球心在轴上
- `sphere`：返回 null（球本身即外接球，无需额外渲染）

**手算验证**：
- 正四棱锥 (a=2,h=2)：R_底=√2, R_球=(4+2)/(2×2)=1.5
- 圆柱 (r=1.5,h=2)：R_球=√(2.25+1)=√3.25≈1.80
- 圆锥 (r=1.5,h=2)：R_球=(4+2.25)/(2×2)=6.25/4=1.5625

### Bug 修复（续作并入）

**BUG: 棱锥开启坐标系点击顶点崩溃**
- 根因：`coordinates.ts` 中 pyramid 路由到 `buildCuboidCoordinateSystem`，使用 `CUBOID_ADJACENCY`（8顶点长方体邻接表），正三棱锥只有4个顶点，`vertices[4]` 为 undefined
- 修复：在 `CUBOID_ADJACENCY` 使用前增加边界检查 `adj.every((i) => i < vertices.length)`

**BUG: 棱锥坐标系轴不正交**
- 根因：棱锥的 edges 推导分支直接把相邻边方向当坐标轴，但棱锥边不互相垂直
- 修复：新增 `buildPyramidCoordinateSystem`，用 Gram-Schmidt 正交化生成互相垂直的三轴
