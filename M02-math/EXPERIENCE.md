# EduMath Lab 体验文档

> 在线地址：部署在 Vercel，访问仓库绑定的域名即可体验
> 技术栈：React + TypeScript + Vite + math.js + Zustand + Canvas 2D + Tailwind CSS

---

## 项目简介

**EduMath Lab** 是一套面向高中数学教学的交互式可视化实验平台，包含三个独立模块：

| 模块 | 名称 | 入口路由 |
|------|------|----------|
| M02 | 函数图形实验室 | `/m02` |
| M03 | 解析几何画板 | `/m03` |
| M04 | 三角函数演示台 | `/m04` |

首页（`/`）展示三个模块卡片，点击即可进入对应实验。

---

## 模块一：M02 函数图形实验室

### 核心功能

- **自由函数输入**：支持任意数学表达式，如 `sin(x)`、`x^2 + 2x - 1`、`1/x`、`ln(x)` 等
- **参数变换**：通过 a / b / h / k 四个滑块实时控制 `y = a·f(b(x-h)) + k` 变换
- **自动系数检测**：输入 `a*x^2 + b*x + c` 后自动识别 a、b、c 为可调参数，生成对应滑块
- **特征点标注**：自动计算零点、极值点、拐点，画布上用圆点 + 标签标注
- **导数曲线**：一键显示 f'(x) 导数曲线（符号微分，非数值近似）
- **实时切线**：鼠标悬停在曲线上，显示该点处的切线及斜率数值
- **分段函数**：支持定义多段表达式，各段区间独立设置
- **动画系统**：参数变换支持 RAF 插值动画，演示参数变化对函数图形的影响
- **函数库**：内置常用函数模板（一次、二次、三次、指数、对数、三角等），一键添加
- **Undo / Redo**：所有操作支持撤销 / 重做（Command 模式，最多 50 步）

### 画布交互

- 拖拽平移画布，滚轮缩放
- 双击画布重置视口到默认范围
- 右侧面板可手动设置视口范围、切换坐标刻度显示
- 曲线悬停高亮，点击选中

## 模块二：M03 解析几何画板

### 核心功能

- **圆锥曲线**：支持椭圆、双曲线、抛物线、圆，参数实时可调
- **直线工具**：两点确定直线，与曲线求交
- **焦点与准线**：自动标注焦点、准线、通径等关键几何量
- **离心率动画**：滑块或动画控制离心率 e 从 0.01 → 2.0 连续变化，实时观察曲线形态变化
- **光学反射**：焦点光线反射路径可视化
- **预设曲线**：一键加载经典圆锥曲线配置
- **隐式曲线**：支持输入任意隐式方程 F(x,y) = 0，marching squares 算法渲染
- **动点约束**：在曲线上放置可拖动点，支持参数化运动 + 轨迹记录 + 动画

### 画布交互

- 悬停曲线显示发光高亮 + 方程标签（白底卡片）
- 点击选中实体，右侧面板显示详细参数
- 点击空白处取消选中
- 拖拽平移、滚轮缩放、双击重置

## 模块三：M04 三角函数演示台

### 核心功能

#### 单位圆视图（左侧）

- **交互式单位圆**：拖动点 P 沿圆周运动，实时显示角度和三角函数值
- **特殊角吸附**：24 个特殊角（π/12 步进），3° 容差自动吸附
- **精确值显示**：吸附时显示精确分数值（如 sin 60° = √3/2）
- **投影线**：可开关的 sin（绿色竖线）和 cos（蓝色横线）投影
- **角度弧**：从 x 正半轴到当前角的弧线 + 箭头
- **象限标注**：可选显示 Ⅰ Ⅱ Ⅲ Ⅳ 象限编号
- **智能标签避让**：所有文字标签自动避免重叠（8 方向候选 + 碰撞检测）

#### 函数图像视图（右侧，与单位圆同步）

- **三角函数图像**：sin / cos / tan 一键切换，x 轴使用 π 分数刻度
- **双视图同步**：单位圆拖动时，函数图上的追踪点同步移动，中间分割线有同步标记点
- **轨迹历史**：追踪点经过的路径以淡入轨迹点形式保留（最多 300 点）
- **参考曲线**：可开关 y = sin(x) 原始曲线（半透明虚线）对比变换效果
- **变换参数**：A / ω / φ / k 四个参数，对应 `y = A·sin(ωx + φ) + k`，支持 KaTeX 公式实时渲染

#### 五点法作图

- 分步演示五点法：逐步显示 5 个关键点（零点 / 最大值 / 最小值）
- 连接虚线 + 角色着色（零点白色、极大绿色、极小橙色）
- 标签自动避让，不与追踪点重叠

#### 辅助角公式

- 输入 a·sin x + b·cos x，自动合成为 R·sin(x + φ)
- 可分别开关三条曲线（C1 = a·sin x，C2 = b·cos x，CR = 合成结果）

#### 三角形解算（独立模式）

- 已知条件输入（SSS / SAS / ASA / AAS / SSA 五种模式）
- SSA 模式自动检测两解 / 一解 / 无解
- 实时渲染三角形 + 标注边长和角度
- 模式切换：函数图像 ↔ 三角解算

#### 特殊值表

- 24 个特殊角的 sin / cos / tan 完整表格
- KaTeX 渲染精确值（如 √2/2、√3/3）
- 点击任意角度，单位圆 200ms 动画跳转到该角

## 全局设计规范

### 视觉系统

| 层级 | 主色 | 用途 |
|------|------|------|
| 画布 / 布局 | `#32D583`（M02 绿） | 主色高亮、选中态、品牌色 |
| UI 组件 | `#00C06B`（EduMind 绿） | 按钮、滑块、输入框 |

- **曲线默认色**：未选中为深灰 `#6B7280`，选中 / 悬停时切换为主色 `#32D583`
- **三角函数色**：sin 绿色、cos 蓝色、tan 琥珀色、角度弧紫色
- **白色卡片风格**：所有面板和弹窗统一使用白底 + 圆角 + 微阴影

### 画布渲染

- **双层 Canvas**：静态层（坐标轴、网格、曲线）+ 动态层（悬停高亮、切线、追踪点）
- **Y 轴翻转**：数学坐标系 Y 朝上，Canvas 像素 Y 朝下，Viewport 统一转换
- **不连续检测**：`1/x`、`tan(x)` 等函数在不连续处自动断开，不会画出竖直"穿越线"
- **性能优化**：正常 800 采样步，动画时降为 400 步，保证 ≥ 55fps

### 标签避让系统（本次新增）

- 统一的 `LabelPlacer` 模块，每帧创建一个实例
- 8 方向候选位置（右、右上、上、左上、左、左下、下、右下）
- 优先方向由上下文决定（如点 P 的标签优先放在"离圆心"方向）
- 碰撞检测：已放置标签 + 4px 间距
- 边界检测：距画布边缘 8px 内不放置
- 全部候选重叠时跳过（宁可不显示也不叠）
- 点、圆心等非文字元素通过 `reserve()` 参与碰撞计算

## 技术亮点

1. **math.js 符号微分**：导数曲线通过 AST 级别的符号微分计算，非数值差分
2. **LRU 表达式缓存**：50 条缓存，避免重复解析
3. **Command 模式**：所有用户操作走 execute/undo 命令，滑块拖拽中间态为直接更新（保证流畅），松手时才写一条命令
4. **Viewport 不可变**：所有视口变换返回新实例，视口变更不进入 Undo 历史
5. **跨视图同步**：单位圆 → 函数图通过 store 单向推送 traceX，分割线通过 syncLineStore 中转
6. **Marching Squares**：隐式曲线使用 marching squares 算法实时渲染
7. **精确值引擎**：24 个特殊角的三角函数值预计算为精确分数 + LaTeX 表示

## 已知限制与后续优化方向

- 响应式参数：目前标签字号、点半径等为静态常量，窄屏下可进一步动态缩放
- 窄屏自适应：canvas 宽 < 300px 时可自动关闭部分辅助元素
- 标签淡入淡出：目前为硬切显示/隐藏，可改为 alpha 过渡动画
- 辅助合成曲线标签：auxiliary 模式下 3 条曲线的图例可加入同样的避让处理
- SpecialValuesTable：窄面板时可折叠为更紧凑的布局
