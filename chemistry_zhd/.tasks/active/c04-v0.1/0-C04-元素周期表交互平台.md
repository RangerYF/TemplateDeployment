# C-04 元素周期表交互平台

## 项目背景
当前项目 chemistry_anim 是从 physics_anim 工程模板复制而来，目标是建设独立的化学可视化教学产品线。

## 当前任务
开发 C-04 元素周期表交互平台第一版 MVP。

## 本阶段目标
实现一个可以本地运行的元素周期表交互页面，包含：

1. 标准 18 列周期表布局
2. 元素分类配色
3. 点击元素查看详情
4. 属性染色模式

## 本阶段不做
1. 电子排布动画
2. 元素故事卡
3. 折线图对比
4. 金属活动性顺序联动

## 页面风格
- 白色背景
- 绿色主色
- 卡片式教学界面

## 验收标准
1. 页面可以正常运行
2. 周期表布局正确
3. 点击元素可以查看详情
4. 可以切换至少 3 个属性染色
