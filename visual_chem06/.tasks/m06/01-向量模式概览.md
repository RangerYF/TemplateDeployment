# M-06 向量模式 — 功能概览

## 功能目标

在 M-01 立体几何展示台基础上，增加"向量模式"子功能，无需独立仓库。向量模式通过右侧面板的开关激活，激活后：

1. 3D 场景内出现世界坐标系（XYZ 三轴）
2. 顶部工具栏追加"画向量"和"法向量"两个工具按钮
3. 用户可以在几何体顶点之间创建有向向量（带箭头标注）
4. 用户可以点击面，自动生成该面的法向量
5. 所有向量实体均支持选中、属性查看、撤销/重做、级联删除

关闭向量模式时，坐标轴隐藏、工具按钮隐藏，但已创建的向量实体保留不删除。

## 范围限制（本次不实现）

- 向量运算动画（加减法、数乘等 VEC-001~050 演示数据）
- 2D 向量展示台（另建仓库）
- 向量坐标系投影显示（需坐标系联动）

---

## 技术选型

采用"方案A扩展"：完全复用 M-01 的 Entity-Command-Tool-Inspector 架构：

- **Entity**：新增 `vector` 类型，复用 PointProperties 作为端点
- **Command**：复用 CreateEntityCommand；法向量使用自定义 CreateNormalVectorCommand（一次性创建 3 个实体）
- **Tool**：createVectorTool（两步状态机）、normalVectorTool（一步）
- **Renderer**：VectorEntityRenderer（Line + ConeGeometry 箭头 + Html 标签）
- **Inspector**：VectorInspector（分量、颜色选择器）
- **UI**：向量模式开关在 AuxiliaryTools 面板；工具按钮在 ToolBar 中条件渲染
