## 数据与技术判断

### 数据契约

- 当前仓库不存在 `data/C06_电化学模型数据.md`
- 本次实现将以 PRD 与电化学常识为基础，在代码中建立一份结构化模型数据源
- 每个模型至少包含：
  - 基础信息（名称、类型、学段、环境、简介）
  - 电极信息（材料、极性/阴阳极、半反应）
  - 总反应
  - 粒子流向定义（电子 / 阳离子 / 阴离子 / 盐桥 / 隔膜）
  - 动画关键帧
  - 物质变化（颜色、气泡、沉积、溶解、pH）
  - 教学提示 / 易错点 / 数据来源标注
  - 若适用：充放电模式、浓度趋势、练习题

### 推荐技术路线判断

- 与建议技术路径一致的部分：
  - 使用 React + TypeScript + Vite 搭建
  - 采用数据驱动模型库
  - 采用 2D 示意图而非 3D WebGL
  - 使用状态管理驱动模型切换、时间轴、练习模式
- 与建议技术路径的工程化落点：
  - 主舞台采用 **SVG 装置层 + DOM 粒子覆盖层 + DOM 信息层**
  - 动画控制采用 `requestAnimationFrame`
  - 时间轴支持播放 / 暂停 / 逐步 / 速度切换
