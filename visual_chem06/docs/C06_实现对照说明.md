# C-06 电化学演示台实现对照说明

## 1. 结论摘要

- 实现仓库：`visual_chem06_new`
- 实现形态：React + TypeScript + Vite + Tailwind 的三栏式 2D 电化学演示器
- UI 复用来源：`visual_template` 的项目骨架、`design_guid` 的 tokens / ui 组件、`visual_chem02` 的三栏布局与面板组织方式
- 数据实现方式：以 `src/data/electrochemModels.ts` 建立 13 个预设模型的数据源，驱动装置、关键帧、粒子流向与教学提示
- 用户最新指令：移除“练习模式”，因此本实现不再包含 PRD 9.3.6 的交互练习入口

## 2. 技术路径对照

| 建议技术路径 | 本次实现 | 是否一致 |
| --- | --- | --- |
| 2D 示意图，不使用 3D WebGL | 使用 SVG 装置层 + DOM 粒子层 + DOM 信息层 | 是 |
| 数据驱动模型库 | `src/data/electrochemModels.ts` 统一维护 13 个模型、场景、流向、关键帧、提示 | 是 |
| requestAnimationFrame 动画驱动 | `src/components/scene/ElectrochemStage.tsx` 中使用 `requestAnimationFrame` 推进时间轴 | 是 |
| 状态管理驱动模型切换与播放控制 | `src/store/electrochemStore.ts` 统一管理模型、场景、播放、速度、进度 | 是 |
| 不使用编辑器式 Entity / Command / UndoRedo | 本项目未引入编辑器系统，完全按演示器模式实现 | 是 |
| 方程式、关键帧、教学提示同步展示 | 右侧知识面板 + 舞台关键帧说明同步读取当前场景数据 | 是 |
