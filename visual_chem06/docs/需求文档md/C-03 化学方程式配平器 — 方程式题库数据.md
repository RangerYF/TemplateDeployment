**1. 数据结构说明**

每条方程式记录包含以下字段：

  ----------------------- ----------------------- ----------------------------------------------------
  字段                    类型                    说明

  id                      string                  唯一编号，格式 EQ-XXX

  category                string                  分类：化合、分解、置换、复分解、氧化还原、离子反应

  level                   string                  学段：初中 / 高中必修 / 高中选修

  difficulty              number                  难度 1-5（1 最简单）

  unbalanced              string                  未配平方程式

  balanced                string                  正确配平方程式

  method                  string                  推荐配平方法

  method_hint             string                  配平方法提示文字

  conditions              string                  反应条件（加热、催化剂、点燃等）

  notes                   string                  教学备注
  ----------------------- ----------------------- ----------------------------------------------------

**2. 配平方法分类**

  ------------------------- ----------------------- ------------------------------------------
  方法                      适用场景                简述

  观察法                    简单反应                直接观察各元素原子数逐一调整系数

  最小公倍数法              某元素原子数差异大      找到两边该元素原子数的最小公倍数确定系数

  奇数配偶法                某元素一侧为奇数        先将奇数变偶数，再调整其他系数

  化合价升降法              氧化还原反应            根据化合价变化量确定氧化剂与还原剂系数

  电子守恒法                复杂氧化还原            得失电子总数相等原理配平

  离子-电子法（半反应法）   离子方程式              分别写出氧化和还原半反应再合并

  待定系数法                复杂反应                设各物质系数为未知数，列方程组求解
  ------------------------- ----------------------- ------------------------------------------

**5.1 金属活动性顺序相关规则**

**金属活动性顺序：** K Ca Na Mg Al Zn Fe Sn Pb (H) Cu Hg Ag Pt Au

规则引擎需内置以下判断逻辑：

  ---------------------------- ------------------------------------------------------ -----------------------
  规则                         说明                                                   示例

  排在 H                       Cu、Ag、Au 等不与稀 HCl / 稀 H₂SO₄ 反应                Cu + HCl → 不反应
  后面的金属不与稀酸反应                                                              

  弱金属不能置换强金属         活动性弱的金属不能将活动性强的金属从盐溶液中置换出来   Cu + ZnSO₄ → 不反应

**6. 配平步骤演示数据**

**示例 1：Fe₂O₃ + CO → Fe + CO₂（化合价升降法）**

  ----------------------- ----------------------- -----------------------
  步骤                    操作                    当前状态

  1                       写出未配平方程式        Fe₂O₃ + CO → Fe + CO₂

  2                       标出变化元素化合价      Fe: +3→0（降低 3）；C:
                                                  +2→+4（升高 2）

  3                       求最小公倍数            升降各 6：Fe 降
                                                  3×2=6，C 升 2×3=6

  4                       确定变化物质系数        Fe₂O₃ 系数 1（含 2 个
                                                  Fe）；CO 系数 3

  5                       配平其余原子            Fe₂O₃ + 3CO → 2Fe +
                                                  3CO₂

  6                       检查各元素              Fe: 2=2 ✓；O: 3+3=6, 6
                                                  ✓；C: 3=3 ✓
  ----------------------- ----------------------- -----------------------

**示例 2：P + O₂ → P₂O₅（最小公倍数法）**

  ----------------------- -------------------------- -----------------------
  步骤                    操作                       当前状态

  1                       写出未配平方程式           P + O₂ → P₂O₅

  2                       找到原子数差异最大的元素   O：左边 2 个，右边 5 个

  3                       求 2 和 5 的最小公倍数     LCM(2,5) = 10

  4                       确定 O₂ 和 P₂O₅ 的系数     5O₂（10 个
                                                     O），2P₂O₅（10 个 O）

  5                       配平 P                     右边 4 个 P → 左边系数
                                                     4

  6                       验证                       4P + 5O₂ → 2P₂O₅ ✓
  ----------------------- -------------------------- -----------------------
