**1. 碰撞模型**

**MOM-001 完全弹性碰撞**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  m1             2.0            0.1\~20        0.1            kg

  m2             3.0            0.1\~20        0.1            kg

  v1 (初速度)    5.0            -20\~20        0.1            m/s

  v2 (初速度)    0              -20\~20        0.1            m/s
  -------------- -------------- -------------- -------------- --------------

恢复系数 e = 1

碰后速度：

v1\' = ((m1-m2)v1 + 2m2 v2)/(m1+m2)

v2\' = ((m2-m1)v2 + 2m1 v1)/(m1+m2)

特殊情况：m1 = m2 → 交换速度

**MOM-002 完全非弹性碰撞**

恢复系数 e = 0（碰后粘在一起）

碰后共同速度：v\' = (m1v1 + m2v2)/(m1+m2)

动能损失最大

**MOM-003 部分弹性碰撞**

  ----------------------------------- -----------------------------------
  参数                                额外参数

  恢复系数 e                          0\~1，步进 0.01
  ----------------------------------- -----------------------------------

碰后速度：

v1\' = v1 - (1+e)m2(v1-v2)/(m1+m2)

v2\' = v2 + (1+e)m1(v1-v2)/(m1+m2)

预设对比（m1=m2=1kg, v1=5m/s, v2=0）：

  ----------------- ----------------- ----------------- -----------------
  e                 v1\' (m/s)        v2\' (m/s)        动能损失 (J)

  0                 2.5               2.5               3.125

  0.5               1.25              3.75              1.563

  1.0               0                 5.0               0
  ----------------- ----------------- ----------------- -----------------

**2. 类碰撞模型**

**MOM-011 爬坡模型**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  滑块质量 m     1.0            0.1\~10        0.1            kg

  斜面质量 M     5.0            1\~20          0.1            kg

  初速度 v0      8.0            1\~20          0.1            m/s

  斜面角度 θ (°) 30             10\~80         1              ---

  地面摩擦       无             有/无          ---            ---
  -------------- -------------- -------------- -------------- --------------

滑块冲上斜面，到达最高点时与斜面共速

无摩擦地面：动量守恒 mv0 = (m+M)v共

最大高度：由能量守恒求解

**MOM-012 弹簧压缩释放**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  m1             1.0            0.1\~10        0.1            kg

  m2             2.0            0.1\~10        0.1            kg

  弹簧劲度系数 k 200            50\~1000       10             N/m

  初始压缩量 x0  0.1            0.01\~0.5      0.01           m
  -------------- -------------- -------------- -------------- --------------

初始动量为零：m1v1\' + m2v2\' = 0 → v1\'/v2\' = -m2/m1

弹性势能全部转化为动能：(1/2)kx0² = (1/2)m1v1\'² + (1/2)m2v2\'²

教学要点：质量越大速度越小，方向相反

**MOM-013 滑块模型**

滑块在平台上碰撞，含摩擦力

需要考虑摩擦力做功导致的动能损失

**3. 反冲模型**

**MOM-021 火箭推进**

  ----------------- -------------- -------------- -------------- --------------
  参数              默认值         范围           步进           单位

  火箭+燃料总质量 M 10             1\~100         1              kg

  喷出气体质量 Δm   0.5            0.1\~5         0.1            kg

  喷出速度 v_gas    1000           100\~5000      100            m/s
  ----------------- -------------- -------------- -------------- --------------

动量守恒：0 = (M-Δm)v_rocket - Δm × v_gas

火箭速度：v_rocket = Δm × v_gas / (M-Δm)

**MOM-022 爆炸分离**

初始静止物体爆炸为两块

动量守恒：m1v1 + m2v2 = 0（方向相反）

**4. 人船模型**

**MOM-031 人在船上行走**

  ---------------------- -------------- -------------- -------------- --------------
  参数                   默认值         范围           步进           单位

  人的质量 m_人          60             30\~100        1              kg

  船的质量 m_船          200            50\~500        10             kg

  人走的距离（相对船）   5.0            1\~20          0.1            m

  地面摩擦               无             有/无          ---            ---
  ---------------------- -------------- -------------- -------------- --------------

无摩擦时动量守恒：

m_人 Δx_人 + m_船 Δx_船 = 0

人向前走 → 船向后退

Δx_船/Δx_人 = -m_人/m_船

人相对地面位移：Δx_人_地 = Δx_相对 × m_船/(m_人+m_船)

船相对地面位移：Δx_船_地 = -Δx_相对 × m_人/(m_人+m_船)

**MOM-032 凹槽滑块模型**

滑块在凹槽内行走

有摩擦/无摩擦两种情况

有摩擦时：系统动量不守恒，需考虑摩擦冲量

**5. 动量定理应用**

**MOM-041 冲量演示**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  力 F           10             1\~100         1              N

  作用时间 t     2.0            0.1\~10        0.1            s

  物体质量 m     2.0            0.1\~20        0.1            kg
  -------------- -------------- -------------- -------------- --------------

冲量：I = Ft

动量变化：Δp = I = Ft

缓冲对比：

  ----------------- ----------------- ----------------- -----------------
  方案              力 F (N)          时间 t (s)        冲量 I (N·s)

  硬着陆            1000              0.02              20

  缓冲              100               0.2               20

  弹性缓冲          50                0.4               20
  ----------------- ----------------- ----------------- -----------------

教学要点：相同冲量，力越小时间越长（缓冲原理）
