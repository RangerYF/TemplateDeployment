**C-05 化学键与晶体结构查看器 --- 晶体结构数据**

\> **关联模块：** C-05 化学键与晶体结构查看器

\> **数据用途：** 3D 晶体模型构建参数（晶胞参数、空间群、原子坐标）

\> **文档版本：** V1.0 ｜ 2026-03-08

\> **数据来源：** CRC Handbook, Inorganic Crystal Structure Database
(ICSD)

**1. 数据结构说明**

每个晶体模型包含以下字段：

  ----------------------- ----------------------- --------------------------
  字段                    类型                    说明

  id                      string                  编号 CRY-XXX

  name_cn                 string                  中文名

  formula                 string                  化学式

  level                   string                  学段

  crystal_system          string                  晶系

  space_group             string                  空间群

  lattice_params          object                  晶胞参数 a, b, c (pm), α,
                                                  β, γ (°)

  atoms                   array                   原子坐标（分数坐标）

  bond_type               string                  化学键类型

  coord_number            number                  配位数

  z                       number                  每个晶胞含几个化学式单位

  teaching_points         array                   教学要点
  ----------------------- ----------------------- --------------------------

不存在\"NaCl 分子\"，只有 Na⁺ 和 Cl⁻ 的比例关系

**5. 分子晶体**

**CRY-011 冰 H₂O（Ice Ih）**

  ----------------------------------- -----------------------------------
  参数                                值

  晶系                                六方

  空间群                              P6₃/mmc

  晶胞参数                            a = b = 452 pm, c = 736 pm

  配位数                              4（每个水分子通过氢键与 4
                                      个水分子相连）

  O-H···O 距离                        276 pm

  密度                                0.917 g/cm³（低于液态水）

  学段                                高中必修
  ----------------------------------- -----------------------------------

**教学要点：**

每个水分子形成 4 个氢键（2 个提供 H，2 个接受 H）

氢键使水分子以四面体排列，形成较空疏的结构

解释了冰密度低于水的反常现象

氢键比共价键弱得多（约 20 kJ/mol vs 460 kJ/mol）

**CRY-013 石墨 C（Graphite）**

  ----------------------------------- -----------------------------------
  参数                                值

  晶系                                六方

  空间群                              P6₃/mmc (No. 194)

  晶胞参数                            a = b = 246 pm, c = 671 pm

  配位数                              3（层内）

  层内 C-C 键长                       142 pm

  层间距                              335 pm

  键角                                120°（层内）

  杂化                                sp²

  学段                                高中必修
  ----------------------------------- -----------------------------------

**教学要点：**

层内：每个 C sp² 杂化，与 3 个 C 共价键连接，形成平面六元环网络

层间：范德华力连接，层间距 335 pm

导电性：层内有离域 π 电子可导电

润滑性：层间范德华力弱，层间易滑动

与金刚石对比：同素异形体，结构和性质差异巨大

**8. 晶胞计算公式参考**

工程师实现\"切割晶胞显示实际含原子数\"功能时使用：

  ----------------------------------- -----------------------------------
  原子位置                            占有份额

  顶点                                1/8

  棱上                                1/4

  面心                                1/2

  体心                                1

  晶胞内部                            1
  ----------------------------------- -----------------------------------

**验证公式：** 晶胞中实际原子数 = 顶点×1/8 + 棱×1/4 + 面心×1/2 + 体心×1
