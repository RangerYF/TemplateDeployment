**一、产品概述**

**1.1 核心问题**

高中物理涉及大量抽象概念（力场、电磁场、微观粒子运动等），传统教学存在以下痛点：

1\. **受力分析不直观** ------
黑板上画力的示意图无法动态调整，学生搞不清不同条件下受哪些力

2\. **运动过程难以感知** ------ 抛体、圆周等运动只看公式没有感性认识

3\. **场的概念极度抽象** ------
电场线、磁感线、等势面无法在课堂中实物展示

4\. **电路实验成本高** ------ 实物电路操作慢、动态电路分析靠口述难以直观

5\. **波动现象看不见** ------ 干涉、驻波、多普勒效应用粉笔画不清楚

6\. **微观世界不可见** ------ 分子热运动、放射性衰变、电子跃迁无法实验

**1.2 解决方案**

提供 **12 个可交互的物理可视化模板**，覆盖高中物理全部核心板块：

力学（受力分析 + 运动模拟 + 动量 + 机械能）

电磁学（电路搭建 + 电磁场 + 电磁感应）

光学（折射/全反射 + 干涉衍射）

振动与波动（简谐运动 + 机械波）

热学（气体分子 + 热力学过程）

天体物理（开普勒定律 + 卫星变轨）

近代物理（核衰变 + 光电效应 + 玻尔模型）

每个模板输出为独立 HTML
文件，教师选择模型、设置参数，系统实时计算并可视化展示。

**P-01 受力分析器**

[Untitled
page](https://kcnw8jqlswwm.feishu.cn/wiki/AVuSwwwxyiNNJmkbHpucmQNnnSN)

**一、这个东西是干什么的？**

高中物理课讲受力分析，老师在黑板上画力的示意图不直观，力的方向、大小难以动态调整，学生往往搞不清楚同一物体在不同条件下受到哪些力。

**受力分析器**帮老师解决这个问题：老师选好受力模型和约束条件，系统自动完成受力分析，并以清晰的力的示意图展示出来，支持正交分解和合力计算。

**二、用户是谁？他们怎么用？**

**目标用户：** 高中物理老师，不要求有任何编程基础。

**典型使用场景：**

**课前**：预设好\"斜面有摩擦，物体静止\"等条件，上课直接展示

**课中**：在投影仪上选择条件，让学生观察受力变化

**讲题时**：动态演示力的分解，让学生直观理解正交分解

选好后，右侧立即出现对应的**默认受力示意图预览**。

**3.2 第二步：设置约束条件（有一个意图识别Agent）**

选完模型后，出现该模型的**条件选择面板**，老师勾选当前情景的约束：

\- **运动状态**：初速度 / 初始力

\- **接触面性质**：光滑（无摩擦）/ 粗糙（有摩擦，可输入动摩擦因数）

\- **斜面角度**：（仅斜面模型）可输入倾角 θ

条件修改后，受力示意图**实时更新**，无需点击确认。

**P-02 运动模拟器（此部分和P-01融合）**

[Untitled
page](https://kcnw8jqlswwm.feishu.cn/wiki/HvaDwpvRkiohiik6917clYa0nHc)

**一、这个东西是干什么的？**

匀变速直线运动、抛体运动、圆周运动，学生只看公式没有感性认识。

**力学沙盒**让老师设定物体的初始条件，系统自动演示运动过程，实时显示速度向量、加速度向量、运动轨迹，并同步生成
v-t 图和 s-t 图，让运动规律一目了然。

**二、用户是谁？他们怎么用？**

**目标用户：** 高中物理老师，不要求有任何编程基础。

**典型使用场景：**

**演示平抛运动**：边播放边指出\"水平速度不变，竖直速度逐渐增大\"

**对比多个初速度**：同屏展示不同初速度下的轨迹，直观看出落点变化

**分析圆周运动**：演示竖直圆周不同位置的速度和向心加速度变化

**三、功能清单**

**3.1 第一步：选择运动场景**

  ----------------------------------- ---------------------------------------------------
  场景                                说明

  水平面运动                          匀速/匀加速直线运动

  斜面运动                            沿斜面的加速/减速运动

  平抛运动                            水平初速，重力加速

  斜面上平抛                          从斜面边缘平抛，落点随初速变化

  水平圆周运动                        圆盘有摩擦力（需要设置临界）/绳连模型，设置角速度

  竖直圆周运动                        给初速度，演示全程速度变化
  ----------------------------------- ---------------------------------------------------

**3.2 第二步：设置初始参数**

\- **物体质量**：数值输入

\- **初速度**：大小 + 方向角（拖拽箭头或数值输入）

\- **受力设置**：重力（默认开启）、附加恒力（可设方向和大小）

\- **环境参数**：斜面角度、圆周半径、摩擦系数等

参数修改后，预览图实时更新。

**3.3 第三步：运动演示与分析**

✅ **功能 1：运动轨迹实时绘制**
播放后物体沿轨迹运动，轨迹线留存在画布上，可选择是否保留历史轨迹

✅ **功能 2：速度/加速度向量显示**
运动过程中物体上实时显示速度向量（蓝色）和加速度向量（红色），向量长度按比例变化；平抛运动中可分别显示水平分速度和竖直分速度

✅ **功能 3：时间轴控制** 播放 / 暂停 / 单步前进 / 单步后退 / 倒放 /
调速（0.1x \~ 2x）

✅ **功能 4：v-t 图 / s-t 图联动**
运动动画与图像同步，时间轴上的当前时刻在图像上用竖线标出，清晰对应

✅ **功能 5：多物体对比模式** 最多同时显示 3
组不同参数的运动，轨迹用不同颜色区分；专用于展示斜面上平抛落点随初速变化、竖直圆周不同初速度对比

✅ **功能 6：圆周运动专项分析**

水平圆周：显示摩擦力/绳拉力随角速度的实时变化，超出最大静摩擦力时高亮提示（考试重点）

竖直圆周：显示最高点/最低点的速度、向心加速度，标注脱离轨道的临界条件

✅ **功能 7：数值读取** 点击轨迹上任意点，显示该时刻的
t、x、y、vx、vy、v、a 值

**P-03 光学实验台**

[Untitled
page](https://kcnw8jqlswwm.feishu.cn/wiki/IsdhwYZrdiUQC7kqW1PcoFKGn1g)

**一、这个东西是干什么的？**

折射、反射、透镜成像靠画图容易出错，干涉、衍射更是极度抽象，学生难以理解光路和图样的成因。

**光学实验台**提供可交互的光路追迹环境，涵盖折射/全反射、干涉、衍射等高中光学重点，老师拖拽摆放光学元件，系统实时绘制光路，让光学现象可见可感。

**二、用户是谁？他们怎么用？**

**目标用户：** 高中物理老师，不要求有任何编程基础。

**典型使用场景：**

**演示全反射**：调整入射角，直观看到临界角，演示光导纤维原理

**展示干涉条纹**：改变双缝间距，实时看到条纹疏密变化

**讲解衍射**：切换波长，看不同颜色光的衍射图样差异

✅ **功能 3：双棒模型分析**

含电容：终态时电流为零，电容器充电到稳定电压，标注电容两端电压；含电源：分析外加电源对终态速度的影响，显示最终电流和速度

✅ **功能 5：安培力方向动画**
在任意时刻暂停，点击\"分析受力\"，系统逐步展示：速度方向→EMF
方向→电流方向→安培力方向，每步有箭头和文字说明

**P-14 机械能守恒**

[Untitled
page](https://kcnw8jqlswwm.feishu.cn/wiki/MR1fwCG0ZiOxO3kPPtOcoFIsnVf)

**一、这个东西是干什么的？**

机械能守恒是高中力学的核心定理，但从自由落体到多物体复杂模型，能量转化过程不直观，弹性势能更是无法直接\"看见\"。

**机械能守恒**工具从最基本的自由落体出发，覆盖各类经典模型，用能量条形图实时展示各种能量的转化过程。

**二、用户是谁？他们怎么用？**

**目标用户：** 高中物理老师，不要求有任何编程基础。

**典型使用场景：**

**自由落体入门**：高度减小，速度增大，势能转化为动能，能量条实时变化

**轻杆双球**：演示从最低点到最高点，动能和势能如何交换

**带弹簧模型**：压缩释放，弹性势能转化为动能的过程

  轻杆圆周（双球）                    杆端两球在竖直面内圆周运动

**3.2 核心功能**

✅ **功能 1：实时能量条形图**
页面右侧或下方始终显示当前各能量分量的柱状图：重力势能（蓝色）、动能（红色）、弹性势能（绿色）、总机械能（灰色）；运动过程中各条实时变化，总机械能柱高不变（无耗散时）

✅ **功能 2：自由落体基础演示**
物体从设定高度开始下落，能量条实时变化；可暂停在任意位置，显示该时刻的高度、速度、动能和势能数值，验证
Ek + Ep = E总

✅ **功能 3：轻杆圆周运动（双球）**
轻杆两端各有一个质量不同的小球，在竖直面内做圆周运动；动画显示从最低点到最高点的运动，能量条随位置变化；标注最高点和最低点的速度、向心力条件和能量关系

✅ **功能 4：定滑轮双物体**
通过轻绳连接的两物体，一上一下，可设置质量比；动画显示两物体的位移和速度；能量条显示各物体势能和动能的变化，验证总机械能守恒；可添加摩擦力，显示摩擦做功后总机械能减少

✅ **功能 5：弹簧弹性势能**
弹簧振子模型：从压缩状态释放，弹性势能转化为动能，经过平衡位置动能最大，到拉伸状态动能又转化为弹性势能；可对比有摩擦（阻尼振动）和无摩擦情况下能量条的衰减

✅ **功能 6：有/无摩擦对比**
任意模型下可切换\"光滑/有摩擦\"，有摩擦时总机械能逐渐减少，减少量等于摩擦力做的功，数值标注
