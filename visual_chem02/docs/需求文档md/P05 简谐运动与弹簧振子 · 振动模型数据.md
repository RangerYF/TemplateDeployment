**2. 竖直弹簧振子**

**SHM-001 竖直弹簧振子**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  弹簧劲度系数 k 100            10\~500        1              N/m

  物体质量 m     0.5            0.1\~5.0       0.1            kg

  初始位移 x0    0.05           0\~0.2         0.01           m

  阻尼系数 b     0              0\~2.0         0.1            N·s/m
  -------------- -------------- -------------- -------------- --------------

平衡位置：x_eq = mg/k（弹簧伸长量）

运动方程（以平衡位置为原点）：

x(t) = A cos(ωt + φ)

v(t) = -Aω sin(ωt + φ)

a(t) = -Aω² cos(ωt + φ) = -ω²x

角频率：ω = sqrt(k/m)

周期：T = 2π sqrt(m/k)

特征位置分析：

  ----------- ----------- ------------- -------------------- --------------- -----------
  位置        位移        速度          加速度               弹簧弹力        回复力

  最高点      +A          0             -ω²A（最大，向下）   k(x_eq -        最大
                                                             A)（可能为0）   

  平衡位置    0           ±Aω（最大）   0                    kx_eq = mg      0

  最低点      -A          0             +ω²A（最大，向上）   k(x_eq + A)     最大
  ----------- ----------- ------------- -------------------- --------------- -----------

教学要点：

平衡位置 ≠ 自然长度位置（重力造成偏移）

不同质量/弹簧下平衡位置的变化（核心教学点）

最高点和最低点关于平衡位置的位移对称性

最高点和最低点的加速度大小相等、方向相反（加速度对称性）

竖直弹簧振子涉及位移对称和加速度对称两个方面

**3. 水平弹簧振子**

**SHM-002 水平弹簧振子**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  弹簧劲度系数 k 100            10\~500        1              N/m

  物体质量 m     0.5            0.1\~5.0       0.1            kg

  初始位移 x0    0.1            -0.2\~0.2      0.01           m
  -------------- -------------- -------------- -------------- --------------

平衡位置 = 弹簧自然长度位置（无重力偏移）

运动方程同 SHM-001（ω = sqrt(k/m)）

能量关系：

动能：Ek = (1/2)mv² = (1/2)kA² sin²(ωt + φ)

弹性势能：Ep = (1/2)kx² = (1/2)kA² cos²(ωt + φ)

总能量：E = (1/2)kA²（守恒）

**4. 单摆**

**SHM-003 简单单摆**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  绳长 L         1.0            0.1\~5.0       0.1            m

  初始摆角 θ0    5              1\~60          1              °

  重力加速度 g   9.8            1\~20          0.1            m/s²
  -------------- -------------- -------------- -------------- --------------

小角度近似（θ \< 10°）：

周期：T = 2π sqrt(L/g)

运动方程：θ(t) = θ0 cos(ωt)，ω = sqrt(g/L)

等时性：周期与振幅无关

精确解（大摆角）：

周期需要椭圆积分

T_精确 \> T_近似，摆角越大差异越大

小角度 vs 精确运动对比：

  ----------------- ----------------- ----------------- -----------------
  摆角 θ0 (°)       T_近似/T0         T_精确/T0         差异

  5                 1.000             1.000             \<0.1%

  15                1.000             1.004             0.4%

  30                1.000             1.017             1.7%

  45                1.000             1.040             4.0%

  60                1.000             1.073             7.3%
  ----------------- ----------------- ----------------- -----------------

教学要点：

小角度近似的适用范围

周期只与绳长和重力加速度有关

单摆测重力加速度实验

**6. 运动图像联动规范**

**x-t 图（位移-时间图）**

  ----------------------------------- -----------------------------------
  属性                                值

  横轴                                时间 t (s)

  纵轴                                位移 x (m)

  曲线形状                            正弦/余弦

  当前时刻标记                        红色竖线

  平衡位置                            x = 0 水平虚线

  振幅标注                            ±A 水平虚线
  ----------------------------------- -----------------------------------

**v-t 图（速度-时间图）**

与 x-t 图相位差 π/2

位移最大时速度为零

经过平衡位置时速度最大

**a-t 图（加速度-时间图）**

与 x-t 图相位差 π（反相）

a = -ω²x

位移最大时加速度最大

三图同步显示，当前时刻竖线对齐

**7. 波的形成预览数据**

**SHM-005 波的形成**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  振子数量 N        20                5\~50             1

  相邻振子相位差 Δφ π/10              π/20\~π/2         π/20

  振幅 A (cm)       5                 1\~15             1

  频率 f (Hz)       1.0               0.1\~5.0          0.1
  ----------------- ----------------- ----------------- -----------------

N 个振子排成一列，每个振子做简谐运动

相邻振子有固定相位差 → 形成行波

波长 λ = 2π/Δφ × 振子间距
