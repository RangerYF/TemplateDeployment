**2. 水平面运动**

**MOT-001 匀速直线运动**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  初速度 v0 (m/s)   5.0               0.1\~50           0.1
  ----------------- ----------------- ----------------- -----------------

运动方程：x = v0 t

图表：x-t（直线）、v-t（水平线）

**MOT-002 匀加速直线运动（水平）**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  初速度 v0 (m/s)   0                 0\~50             0.1

  加速度 a (m/s²)   2.0               -10\~10           0.1

  质量 m (kg)       2.0               0.1\~50           0.1
  ----------------- ----------------- ----------------- -----------------

运动方程：

v = v0 + at

x = v0 t + (1/2)at²

v² - v0² = 2ax

图表：x-t（抛物线）、v-t（直线）、a-t（水平线）

教学要点：

v-t 图中直线斜率 = 加速度

x-t 图中切线斜率 = 瞬时速度

v-t 图与时间轴围成面积 = 位移

**3. 斜面运动**

**MOT-011 斜面匀加速下滑**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  斜面角度 θ (°)    30                5\~85             1

  动摩擦因数 μ      0.2               0\~1.0            0.01

  初速度 v0 (m/s)   0                 0\~20             0.1
  ----------------- ----------------- ----------------- -----------------

加速度：a = g(sin θ - μ cos θ)

临界条件：tan θ \> μ 时才能下滑

**MOT-012 斜面减速上滑后下滑**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  斜面角度 θ (°)    30                5\~85             1

  动摩擦因数 μ      0.2               0\~1.0            0.01

  初速度 v0 (m/s)   10                1\~50             0.1
  ----------------- ----------------- ----------------- -----------------

上滑加速度：a上 = -g(sin θ + μ cos θ)

下滑加速度：a下 = g(sin θ - μ cos θ)

教学要点：上滑和下滑加速度不同（摩擦力方向改变）

**4. 平抛运动**

**MOT-021 标准平抛**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  初始高度 h (m)    20                1\~100            1

  水平初速度 v0     10                1\~50             0.1
  (m/s)                                                 

  重力加速度 g      9.8               1\~20             0.1
  (m/s²)                                                
  ----------------- ----------------- ----------------- -----------------

运动方程：

水平：x = v0 t

竖直：y = (1/2)gt²

轨迹：y = gx²/(2v0²)（抛物线）

关键物理量：

下落时间：t = sqrt(2h/g)

水平射程：R = v0 sqrt(2h/g)

落地速度：v = sqrt(v0² + (gt)²)

落地速度方向：tan α = gt/v0

需显示向量：

vx = v0（恒定，蓝色水平箭头）

vy = gt（增大，蓝色竖直箭头）

v = 合速度（蓝色斜箭头）

教学要点：

水平方向匀速，竖直方向自由落体

分运动的独立性

轨迹为抛物线

**MOT-022 斜面上平抛**

  -------------------------- ----------------- ----------------- -----------------

  平抛高度（斜面边缘距底）   10                1\~50             1

**5. 圆周运动**

**MOT-031 水平圆周（匀速）**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  半径 r (m)        1.0               0.1\~10           0.1

  角速度 ω (rad/s)  2.0               0.1\~20           0.1

  质量 m (kg)       1.0               0.1\~10           0.1
  ----------------- ----------------- ----------------- -----------------

物理量：

线速度：v = ωr

向心加速度：a = ω²r = v²/r

周期：T = 2π/ω

向心力：F = mω²r

**MOT-032 水平圆周（摩擦力提供向心力）**

圆盘模型：物体随圆盘转动

向心力 = 静摩擦力

临界：mω²r = μmg → 物体飞出

设置角速度，观察是否超过临界角速度

**MOT-033 竖直圆周运动**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  半径 r (m)        1.0               0.1\~5            0.1

  最低点初速度 v0   7.0               0\~30             0.1
  (m/s)                                                 

  质量 m (kg)       1.0               0.1\~10           0.1
  ----------------- ----------------- ----------------- -----------------

关键位置分析：

  ----------------------- ----------------------- -----------------------
  位置                    速度条件                向心力方程

  最低点                  v_max                   T - mg = mv²/r

  最高点                  v_min                   T + mg = mv²/r（绳）或
                                                  mg - N = mv²/r（轨道）

  水平位置                中间值                  T = mv²/r
  ----------------------- ----------------------- -----------------------

临界条件（绳连）：

最高点刚好通过：v_top = sqrt(gr)

最低点最小速度：v_bottom = sqrt(5gr)（能量守恒）

教学要点：

绳模型 vs 杆模型 vs 轨道模型的区别

最高点临界条件

脱轨位置判断

**6. 运动图像参考**

**典型 v-t 图形状**

  ----------------------- ----------------------- -----------------------
  运动类型                v-t 图形状              加速度

  匀速直线                水平线                  a = 0

  匀加速                  正斜率直线              a \> 0 常数

  匀减速                  负斜率直线              a \< 0 常数

  先加速后匀速            折线                    a→0

  自由落体                过原点正斜率直线        a = g
  ----------------------- ----------------------- -----------------------

**v-t 图中的物理量**

  ----------------------------------- -----------------------------------
  信息                                读取方法

  加速度                              直线斜率 k = Δv/Δt

  位移                                线下面积（时间轴以上为正）

  瞬时速度                            纵坐标值

  运动方向改变                        v 过零点

  两物体相遇                          面积相等（位移相等）
  ----------------------------------- -----------------------------------
