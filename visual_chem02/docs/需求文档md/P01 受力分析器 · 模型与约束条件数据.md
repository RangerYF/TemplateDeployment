**2. 水平面模型**

**FM-001 水平面静止**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  质量 m (kg)       2.0               0.1\~50           0.1

  动摩擦因数 μ      0.3               0\~1.0            0.01

  外力 F (N)        0                 0\~500            1

  外力方向角 θ (°)  0                 -90\~90           1
  ----------------- ----------------- ----------------- -----------------

约束条件组合：

  -------------- -------------- -------------- -------------- --------------
  编号           接触面         运动状态       外力           受力分析结果

  FM-001-A       光滑           静止           无             G, N

  FM-001-B       粗糙           静止           无             G, N

  FM-001-C       粗糙           静止           水平推力       G, N, f, F

  FM-001-D       粗糙           匀速           水平推力       G, N, f=F, F

  FM-001-E       粗糙           加速           水平推力       G, N, f, F
                                                              (F\>f)

  FM-001-F       光滑           加速           水平推力       G, N, F

  FM-001-G       粗糙           静止           斜向推力       G, N, f, F
                                                              (N\>mg)

  FM-001-H       粗糙           静止           斜向拉力       G, N, f, F
                                                              (N\<mg)
  -------------- -------------- -------------- -------------- --------------

公式：

G = mg

N = mg - F sin θ（斜向拉力）/ N = mg + F sin θ（斜向推力）

f = μN（滑动摩擦力）/ f \<= μN（静摩擦力）

a = (F cos θ - f) / m

教学要点：

区分静摩擦力和滑动摩擦力

斜向力对支持力的影响

\"恰好滑动\"临界条件

**FM-002 水平面运动**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  质量 m (kg)       2.0               0.1\~50           0.1

  初速度 v0 (m/s)   5.0               0\~50             0.1

  ----------------- ----------------- ----------------- -----------------

典型情景：

初速度向右，摩擦减速至停止

**3. 斜面模型**

**FM-011 斜面静止/运动**

  ----------------- ----------------- -------------------------- -----------------
  参数              默认值            范围                       步进

  质量 m (kg)       2.0               0.1\~50                    0.1

  斜面角度 θ (°)    30                5\~85                      1

  动摩擦因数 μ      0.3               0\~1.0                     0.01

  外力 F (N)        0                 0\~500                     1

  外力方向          沿斜面向上        沿斜面/垂直斜面/任意角度   ---
  ----------------- ----------------- -------------------------- -----------------

约束条件组合：

  -------------- -------------- -------------- -------------- ---------------
  编号           接触面         运动状态       外力           受力分析

  FM-011-A       光滑           沿斜面下滑     无             G, N

  FM-011-B       粗糙           静止           无             G, N, f
                                                              (沿斜面向上)

  FM-011-C       粗糙           匀速下滑       无             G, N, f
                                                              (沿斜面向上,
                                                              f=mg sin θ)

  FM-011-D       粗糙           加速下滑       无             G, N, f (mg sin
                                                              θ \> f)

  FM-011-E       粗糙           匀速上推       沿斜面向上     G, N, f
                                                              (沿斜面向下), F

  FM-011-F       粗糙           减速上滑       无             G, N, f
                                                              (沿斜面向下)
  -------------- -------------- -------------- -------------- ---------------

正交分解（沿斜面/垂直斜面）：

沿斜面：F - mg sin θ - f = ma

垂直斜面：N - mg cos θ = 0

摩擦力：f = μN = μmg cos θ

临界条件：

恰好不下滑：mg sin θ = μmg cos θ → tan θ = μ

恰好匀速下滑：mg sin θ = μmg cos θ

教学要点：

正交分解的坐标系选择（沿斜面 vs 水平竖直）

斜面角度对各力的影响

临界问题分析方法

**6. 圆周运动模型**

**FM-041 水平圆周（圆盘 + 摩擦）**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  质量 m (kg)       0.5               0.1\~10           0.1

  转动半径 r (m)    0.2               0.05\~1.0         0.01

  角速度 ω (rad/s)  5.0               0.1\~50           0.1

  最大静摩擦因数 μ  0.4               0\~1.0            0.01
  ----------------- ----------------- ----------------- -----------------

向心力由摩擦力提供：f = mω\^2 r

临界条件：mω\^2 r = μmg → ω_max = sqrt(μg/r)

**FM-042 水平圆周（绳连）**

绳提供向心力：T = mω\^2 r

临界条件：绳断裂

**FM-043 竖直圆周（绳连）**

最高点：T + mg = mv\^2/r，临界 T=0 → v_min = sqrt(gr)

最低点：T - mg = mv\^2/r

**FM-044 竖直圆周（杆连）**

最高点：杆可推可拉

杆拉力：T + mg = mv\^2/r (v \> sqrt(gr))

杆推力：mg - T = mv\^2/r (v \< sqrt(gr))

临界无力：v = sqrt(gr)

**8. 初中模型**

**FM-061 浮力模型**

  ----------------- ----------------- ----------------- -----------------
  参数              默认值            范围              步进

  物体体积 V (cm³)  100               10\~1000          1

  物体密度 ρ物      800               100\~10000        10
  (kg/m³)                                               

  液体密度 ρ液      1000              500\~13600        10
  (kg/m³)                                               
  ----------------- ----------------- ----------------- -----------------

浮力：F浮 = ρ液 g V排

漂浮条件：F浮 = G → ρ液 V排 = ρ物 V物

**FM-062 滑轮模型**

定滑轮：改变力的方向，F = G

动滑轮：省力一半，F = G/2（不计滑轮重和摩擦）

滑轮组：F = G / n（n为绳子段数）

**FM-063 杠杆模型**

平衡条件：F1 × L1 = F2 × L2

**9. 正交分解参考**

**坐标系选择策略**

  ----------------------- ----------------------- --------------------------
  情况                    推荐坐标系              理由

  斜面问题                沿斜面/垂直斜面         加速度沿斜面，N垂直斜面

  圆周运动                切向/法向               向心加速度沿半径

  悬挂平衡                水平/竖直               合力为零，各方向分别平衡

  一般平衡                水平/竖直               默认选择
  ----------------------- ----------------------- --------------------------

**常见力的分解**

  ----------------------- ----------------------- -----------------------
  力                      沿斜面分量              垂直斜面分量

  重力 G                  G sin θ（沿斜面向下）   G cos θ（垂直斜面向里）

  沿斜面外力 F            F                       0

  水平外力 F              F cos θ                 F sin θ
  ----------------------- ----------------------- -----------------------

**10. 力学公式汇总**

  ----------------------------------- -----------------------------------
  公式                                说明

  G = mg                              重力

  f = μN                              滑动摩擦力

  F = kx                              弹簧弹力（胡克定律）

  F = ma                              牛顿第二定律

  F浮 = ρ液gV排                       浮力（阿基米德原理）

  F向 = mv²/r = mω²r                  向心力

  T = 2π/ω = 2πr/v                    圆周运动周期
  ----------------------------------- -----------------------------------
