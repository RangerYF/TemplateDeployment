> **对照需求**：`docs/需求文档md/C化学教具产品线 — 产品需求文档（PRD）.md` §6（C-02 分子结构查看器）

## V0.1 → V0.1-2 变更摘要

| 变更项 | 说明 |
|--------|------|
| 2D 布局引擎 | 内置 71 分子改为拓扑布局（星形/链式/环形），导入分子保留 PCA 投影 |
| 全原子孤电子对 | 用价电子公式为所有原子计算孤电子对，不再仅限中心原子 |
| 2D 孤电子对智能定位 | 电子式中孤电子对根据键方向间隙放置，避免与键重叠 |
| 电子式不合并 H | 电子式模式下所有原子（含 H）独立显示，确保每条键的电子对可见 |
| 键长标注贯通 | showBondLengths 信号从 uiStore 贯通到 3D(BondRenderer) 和 2D(StructuralFormula/SkeletalFormula) |
| 空白点击关闭弹窗 | 3D 场景点击空白处同时清除选中状态和关闭双击弹窗 |
| 字体规范化 | 全部 fontSize 提升至 ≥12px，符合设计系统最低标准 |
