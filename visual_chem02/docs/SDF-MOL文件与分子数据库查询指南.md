# SDF/MOL 文件格式与分子数据库查询指南

## 一、SDF 和 MOL 文件格式简介

### MOL 文件 (.mol)

MOL 文件是由 MDL Information Systems（现 BIOVIA/Dassault Systèmes）定义的化学结构文件格式，用于描述单个分子的原子、键、坐标等信息。

**格式结构（V2000）：**

```
分子名称                              ← 第1行：分子名称
  程序信息                            ← 第2行：程序/时间戳
  注释行                              ← 第3行：注释
  3  2  0  0  0  0  0  0  0999 V2000  ← 第4行：计数行（原子数、键数...）
    0.0000    0.0000    0.0000 O  ...  ← 原子块：x y z 元素符号
    0.7572    0.5860    0.0000 H  ...
   -0.7572    0.5860    0.0000 H  ...
  1  2  1  0  0  0  0                  ← 键块：原子1 原子2 键级
  1  3  1  0  0  0  0
M  END                                ← 结束标记
```

- **坐标单位**：埃 (Å)，1 Å = 100 pm
- **2D vs 3D**：MOL 文件可以包含 2D 坐标（z=0）或 3D 坐标
- **键级**：1=单键，2=双键，3=三键，4=离域键

### SDF 文件 (.sdf)

SDF（Structure-Data File）是 MOL 文件的扩展，支持在一个文件中存储多个分子，并附带属性数据。

```
[MOL 文件内容]
> <属性名1>
属性值1

> <属性名2>
属性值2

$$$$                                   ← 分子分隔符
[下一个分子的 MOL 内容...]
$$$$
```

### 两者关系

| 特性     | MOL (.mol)     | SDF (.sdf)           |
| -------- | -------------- | -------------------- |
| 分子数量 | 单个           | 一个或多个           |
| 属性数据 | 无             | 支持键值对属性       |
| 用途     | 单分子交换     | 数据库导出、批量处理 |
| 兼容性   | SDF 的子集     | MOL 的超集           |

> 本项目同时支持 .sdf 和 .mol 格式的导入。

## 二、分子 3D 结构数据库

以下数据库可用于查询和下载分子的 SDF/MOL 文件：

### 1. PubChem（推荐）
- **网址**：https://pubchem.ncbi.nlm.nih.gov/
- **特点**：美国国家卫生研究院(NIH)维护，收录 1.1 亿+ 化合物
- **下载方式**：
  1. 搜索分子名称或分子式
  2. 进入化合物详情页
  3. 点击 "3D Conformer" 标签
  4. 点击 "Download" → 选择 "SDF" 格式
- **API 下载**：
  ```
  https://pubchem.ncbi.nlm.nih.gov/rest/pug/compound/name/{分子名}/SDF?record_type=3d
  https://pubchem.ncbi.nlm.nih.gov/rest/pug/compound/cid/{CID}/SDF?record_type=3d
  ```
- **注意**：离子（如 CO₃²⁻、SO₄²⁻）和部分小分子（如 H₂、F₂）可能没有 3D 数据

### 2. ChemSpider
- **网址**：https://www.chemspider.com/
- **特点**：英国皇家化学学会维护，收录 1亿+ 化合物
- **下载方式**：搜索后在分子页面找到 "MOL" 下载链接
- **注意**：需要免费注册账号

### 3. RCSB PDB（蛋白质/大分子）
- **网址**：https://www.rcsb.org/
- **特点**：蛋白质数据库，适合查找生物大分子配体
- **下载方式**：搜索配体 → 下载 SDF 格式

### 4. ChEBI
- **网址**：https://www.ebi.ac.uk/chebi/
- **特点**：欧洲生物信息研究所维护，侧重生物活性小分子
- **下载方式**：搜索后在实体页面下载 MOL 文件

### 5. NIST Chemistry WebBook
- **网址**：https://webbook.nist.gov/chemistry/
- **特点**：美国标准技术研究院，提供光谱、热力学等数据
- **用途**：验证分子属性数据（不直接提供 SDF 下载）

### 6. Zinc Database
- **网址**：https://zinc.docking.org/
- **特点**：收录可商业获取的化合物，侧重药物发现
- **下载方式**：搜索后批量下载 SDF

### 7. MolView（在线编辑器）
- **网址**：https://molview.org/
- **特点**：在线分子编辑和 3D 可视化，可导出 MOL 文件
- **用途**：手动绘制分子结构并导出 SDF/MOL

## 三、本项目的 SDF 使用说明

### 内置分子
项目预置了 71 个教学分子的 3D 数据：
- **53 个**分子使用 PubChem 3D SDF 数据（内嵌在 `src/data/sdfData.ts`）
- **18 个**分子（离子、双原子分子等）使用算法推导坐标（`src/data/moleculeFallbacks.ts`）

### 导入自定义分子
1. 在左侧分子列表顶部，点击或拖拽上传 .sdf/.mol 文件
2. 填写中文名称和英文名称（分子式自动从 SDF 推导）
3. 点击"确认导入"，分子将加入列表
4. 在右侧信息面板中，可单击编辑各属性字段（空间构型、杂化方式等）

### 注意事项
- 导入的 SDF 文件必须包含 3D 坐标（z 坐标不全为 0）
- 支持 V2000 格式（V3000 暂不支持）
- 导入的分子仅在当前会话中有效，刷新页面后会丢失
