## 二、与推荐技术路径对照

### chem-C02-analysis.md 推荐 vs 实际实现

| 技术维度 | 推荐方案 | 实际实现 | 是否一致 | 说明 |
|---------|---------|---------|---------|------|
| 主渲染框架 | Three.js WebGL 2.0 | React Three Fiber (R3F) | ✅ 一致 | R3F 是 Three.js 的 React 封装层，底层仍是 Three.js WebGL |
| 原子渲染 | SphereGeometry（30原子内无需InstancedMesh） | SphereGeometry 独立Mesh | ✅ 一致 | `AtomRenderer.tsx` 每个原子一个 `<mesh>` |
| 化学键渲染 | CylinderGeometry 单/双/三键 | CylinderGeometry + 分段染色 | ✅ 一致 | `BondRenderer.tsx` 支持6种键类型 |
| 信息标注 | CSS2DRenderer | @react-three/drei Html | ✅ 一致 | drei Html 底层即 CSS2DRenderer |
| 架构模式 | 参数驱动查看器（非编辑器） | Zustand store + 无command/history | ✅ 一致 | 无undo/redo，无entity系统 |
| CPK配色 | 标准CPK色表 | `elements.ts` cpkColor 字段 | ✅ 一致 | 22种元素颜色 |
| 光照方案 | 环境光+方向光 | ambientLight(0.6) + 2×directionalLight | ✅ 一致 | |
| 交互控制 | OrbitControls（旋转+缩放+平移） | OrbitControls + enableDamping | ✅ 一致 | |
| 分子坐标计算 | Z-matrix→笛卡尔 / VSEPR模板 | VSEPR模板 + 递推链式构建 + 芳香模板 | ✅ 一致 | 混合方案覆盖全部71分子 |
| 本地数据库 | 硬编码JSON/TS数组 | `molecules.ts` 硬编码数组 | ✅ 一致 | ALL_MOLECULES: MoleculeData[] |
