# C03 AI 助手测试清单

## 基础配平
- 配平 H2 + O2 = H2O。
- 配平 Fe + CuSO4 = FeSO4 + Cu。
- 配平 CaCO3 + HCl = CaCl2 + CO2 + H2O。
- 配平 KMnO4 + HCl = KCl + MnCl2 + Cl2 + H2O。

## 典型预设
- 演示一个燃烧反应配平。
- 演示一个分解反应配平。
- 演示一个酸碱中和反应配平。
- 演示一个氧化还原反应配平。

## 离子方程式
- 配平 Fe3+ + OH- = Fe(OH)3。
- 配平 H+ + OH- = H2O。
- 输入 SO42- 参与的离子方程式，检查电荷守恒提示。

## 步骤展示
- 展开完整配平步骤。
- 只显示前两步配平过程。
- 先配平方程式，再逐步展开步骤。

## 错误与边界
- 配平 H2 + O2 = CO2，应提示左右元素不匹配或无法配平。
- 配平 Xx + O2 = XxO，应提示未知元素。
- 清空历史记录。
