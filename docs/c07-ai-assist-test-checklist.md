# C07 AI 助手测试清单

## 反应与预设
- 演示合成氨高压有利于平衡右移。
- 切换到 SO2 氧化的无催化剂对比。
- 演示 NO 氧化，并显示颜色和速率变化。
- 演示 N2O4 和 NO2 的颜色平衡。
- 演示 PCl5 分解的温度和压强影响。

## 条件扰动
- 把当前体系温度调到 760 K。
- 把反应物浓度倍率调成 1.5。
- 加入催化剂，观察达到平衡更快。
- 移除催化剂。
- 压缩容器，观察压强和平衡移动。
- 加入反应物，观察 Qc 和 Kc 的比较。
- 加入产物，观察平衡回调。
- 升温 30 K。

## 播放控制
- 暂停当前模拟。
- 以 2 倍速播放当前模拟。
- 逐步前进一帧。
- 重置当前模拟。
- 连续播放后继续输入“显示 c-t 图”，检查不会出现画布状态变化误报。

## 曲线与图表
- 显示 v-t 曲线，不叠加其他图。
- 显示 c-t 曲线，并叠加 Qc/Kc。
- 显示吸光度-t 图。
- 显示 lgc-t 图，用来分析级数。
- 切换到 lnK-1/T 图。
- 切换到 K-T 图。

## 等效平衡
- 展示恒温恒容等效平衡比较。
- 展示恒温恒压等效平衡比较。
- 展示非等效平衡比较。

## 边界与警告
- 选择不存在的反应。
- 对 HI 平衡执行压缩容器，检查不会误切反应。
- 对不支持催化剂的反应加入催化剂，检查页面保持合理状态。
- 设置播放速度为 3 倍，应返回速度不支持的 warning。
