# M05 AI 助手专项测试清单

## 入口

- 启动 `visual_m05`，确认 bridge 支持 `getAiContext` 和 `applyOperations`。
- 数据库 capability 更新后，确认模板 key 为 `m05`，入口为 `http://localhost:5178/`。

## 核心用例

1. `演示抛硬币频率趋近 1/2，模拟 200 次`
   - 期望：切换到抛硬币，`n=200`，运行后出现结果，可播放动画。

2. `显示两颗骰子点数和的分布`
   - 期望：载入两骰子点数和预设，结果图展示 2-12 分布。

3. `用固定种子 demo-coin 演示抛硬币 100 次，方便复现实验`
   - 期望：运行结果包含 replay seed；重复执行同 seed 时结果可复现。

4. `讲解二项分布如何近似正态分布`
   - 期望：载入 `binomial-normal-approx`，显示 `n=50, p=0.5` 的二项分布。

5. `讲解不放回抽样为什么不是二项分布`
   - 期望：载入 `sampling-without-replacement`，显示超几何分布并开启 CDF。

6. `用收入数据讲解右偏分布`
   - 期望：载入 `histogram-skewed-income`，使用 `DS-05` 并运行。

7. `画标准正态分布并显示 1σ、2σ、3σ 区域`
   - 期望：切换正态分布，`mu=0`、`sigma=1`、开启 σ 区域并运行。

8. `用某班数学成绩画 8 组频率直方图`
   - 期望：切换直方图，使用 `DS-01`，`binCount=8`，运行后显示直方图和统计指标。

9. `输入数据 1, 2, 2, 3, 5, 8, 13 生成茎叶图`
   - 期望：切换茎叶图，数据源为 manual，运行后显示茎叶图。

10. `演示广告投入和销售额的线性回归，并显示残差`
   - 期望：切换线性回归，使用 `REG-01`，`showResiduals=true`，运行后显示回归线和残差。

11. `演示大数定律，抛硬币最大 1000 次，显示 3 条曲线`
   - 期望：切换大数定律，`scenario=coinFlip`、`maxN=1000`、`numCurves=3`，运行后曲线趋近理论概率。

12. `切换为单次模拟并掷一次骰子`
   - 期望：切换骰子实验，动画模式为 single，完成一次单步试验并更新结果。

13. `解释当前模拟结果`
   - 期望：AI 基于 `aiContext.activeSimulation.stats` 和 `resultInterpretationHints` 解释，不编造未出现的统计项。

## 参数校验

- `把正态分布 sigma 设为 0` 应失败并回滚。
- `把二项分布 p 设为 1.2` 应失败并回滚。
- `超几何分布 N=10, M=12, n=3` 应失败并回滚。
- `线性回归使用不存在的数据集 REG-999` 应失败并回滚。

## 回滚和历史

- 一组 operations 中任意一步失败时，前面已应用的参数不应残留。
- 成功执行一组 operations 后，撤销应回到执行前状态。
- AI 不应在 snapshot 中写入 `result.data` 或随机样本；结果必须由模板运行生成。
