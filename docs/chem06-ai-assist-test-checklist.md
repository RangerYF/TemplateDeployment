# C06 AI Assist Test Checklist

- [ ] 打开 `http://localhost:5175/`，页面正常加载。
- [ ] 数据库 capability 更新后，确认模板 key 为 `chem06`。
- [ ] AI 输出只包含 `operations`、`patch`、`explanation`、`warnings`，不包含 envelope。

## 核心用例

- [ ] 演示锌铜双液原电池和盐桥离子迁移。
- [ ] 切到铅蓄电池充电过程，并显示离子标签。
- [ ] 展示铅蓄电池放电过程。
- [ ] 播放电解饱和食盐水，速度调成 2 倍。
- [ ] 演示电解水，并跳到 pH 分区阶段附近。
- [ ] 跳到电解食盐水的阳极竞争阶段。
- [ ] 跳到锌铜双液原电池的盐桥补偿阶段。
- [ ] 展示铜的电解精炼。
- [ ] 演示熔融 NaCl 电解，不要误切到电解食盐水。
- [ ] 演示冰晶石助熔电解 Al2O3。
- [ ] 演示电镀铜，并显示离子标签。
- [ ] 只筛选电解池模型。
- [ ] 把当前动画跳到下一步。
- [ ] 暂停并重置当前演示。

## 重点核查

- [ ] 充放电模型能正确区分 `discharge` 和 `charge`。
- [ ] AI 不直接手写反应式、电子方向、离子迁移等派生数据。
- [ ] 播放速度仅输出 `0.5`、`1`、`2`。
- [ ] 跳转具体阶段时使用 `setKeyframe`，不要硬编码近似 progress。
- [ ] 模型不存在或名称歧义时返回 warnings。
