一、根号值输入支持（回应反馈#1）

   - 技术实现：`ParamPanel.tsx` 新增 `SQRT_VALUES` 数组 + `SqrtButtons` 组件 + `formatAsSqrt()` 格式化显示

2. 【发散】根号值显示格式 — 输入框中精确根号值以符号形式显示
   - 演示：设置向量x分量为 √2 → 输入框显示"√2"而非"1.414..."
   - 【⚠ 需手动确认】显示精度和格式是否满足教学需要

四、数量积演示增强（回应反馈#4）

5. 【需求】三种计算方法切换 — 坐标法 / 投影法 / 极化恒等式
   - 来源：反馈"数量积没有太起到演示的功能，可以增加数量积运算的方法"
   - 演示：切换到"数量积"运算 → 参数面板底部显示三种计算方法的推导过程：
     - **坐标法**：a·b = x₁x₂ + y₁y₂ = 具体数值
     - **投影法**：投影 = a·b/|b| = 具体数值
     - **极化恒等式**：a·b = (|a+b|² − |a−b|²)/4 = 具体数值
   - 验证：三种方法计算结果应一致

6. 【需求】极化恒等式可视化 — 画布上叠加 a+b 和 a-b 辅助向量
   - 演示：勾选 **显示极化恒等式** 复选框 → 画布上出现绿色虚线(a+b)和橙色虚线(a-b) → 右下角公式框显示完整推导
   - 预设：加载 VEC-041-F"极化恒等式"预设 → 自动展示 (3,1)·(1,3) = (|a+b|²−|a−b|²)/4
   - 【⚠ 需手动确认】虚线向量是否影响画面整洁度

7. 【需求】投影向量重新设计 — 投影可视化完全重做
   - 来源：反馈"投影的文字看不清"
   - 变更：
     - 新增从原点到投影点的向量线段
     - "投影"标签改为蓝色背景白字（原为纯文本）
     - 增加直角标记（L形符号）
     - 投影点增大并加白色描边
   - 演示：打开投影显示开关 → 观察投影向量、投影标签、直角标记的清晰度
   - 预设：加载 VEC-041-G"投影演示"预设

六、画布比例优化（回应反馈·内部#4）

10. 【需求】SVG 视口扩大20% — 减少"老人手机"感

    - 演示：对比 v0.1 与 v0.2 整体视觉比例

七、向量颜色方案（回应反馈·内部#2）

11. 【需求】向量颜色评估 — 保持不同向量不同颜色
    - 来源：反馈"不同向量是用不同颜色好，还是统一用黑色"
    - 评估结论：**保持多色方案**
      - 向量A（蓝 #3B82F6）、向量B（绿 #22C55E）、结果（橙 #F97316）色相差异>90°，对色觉障碍友好
      - 投影用蓝色一致性好；标量积辅助向量用虚线区分
      - 教学中区分"已知向量"与"运算结果"是核心需求，统一黑色会增加认知负担
    - 此项无代码变更，属评估决策
    - 【⚠ 需手动确认】是否同意保持当前颜色方案
