# 演示之后，产品经理反馈
选中高亮
默认不存在三视图，应该是有“三视图”和“展开图”的选项，点击了之后，在侧面会出现这个框架。
撤销重做
点击任意点建立坐标系，再点击任意点可以显示该点的坐标，这部分在和老师沟通
靠近线段时，线段出现闪烁或是高亮特效；右键出现菜单，左键为选中功能
直接加说明，点击体积、表面积模块展示计算过程
体积和表面积的中间模块：
如果是分数的话，不表示中间过程
如果是根号，也不表示中间过程
只有出现 π，表示中间过程



# 老师的反馈
我们建系一般是让xy轴在底下，z轴朝上，建系建议：1.点击任意点建立坐标系出来三条线，让老师自行选择哪个是xyz轴

目前只能点图中所标出的点作为坐标原点，可不可以让所有点都能作为坐标原点。我感觉是不是可以先选定该点所在面，向上垂直自动延申z轴，然后在该面引导用户选择一个方向延申x轴
除此之外建系后可以输入任意坐标显示出新的点

其他可以改进的点：
1.常见几何体可以增加：正四面体墙角四面体，对棱相等四面体，对棱垂直四面体
2.几何体画法可以增加：侧棱长
3.增加角度功能
4.画出截面之后，可以用户自行选择切掉哪个面
5.增加动点功能，这些是我们做题比较常用到的
