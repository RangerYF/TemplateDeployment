**5. 数乘向量**

**VEC-031 数乘运算**

  ----------------------------------- -----------------------------------

  分类                                基本运算

  学段                                高中必修

  维度                                2D
  ----------------------------------- -----------------------------------

\| 运算 \| k**a** \|

**可调参数：**

  -------------- -------------- -------------- -------------- --------------
  参数           含义           默认值         范围           步长

  k              标量           1              \[-5, 5\]      0.1
  -------------- -------------- -------------- -------------- --------------

\| **a** \| 向量 \| (2, 1) \| 可拖动 \| --- \|

**预设演示场景：**

\| 编号 \| **a** \| k 值序列 \| 教学重点 \|

\|\-\-\-\-\--\|\-\-\-\-\-\--\|\-\-\-\-\-\-\-\-\--\|\-\-\-\-\-\-\-\-\--\|

\| VEC-031-A \| (2, 1) \| k: -2 → -1 → 0 → 1 → 2 → 3 \| 数乘的完整变化
\|

k = 0：结果为零向量

\- 向量共线 ⟺ 存在实数 k 使 **b** = k**a**（**a** ≠ **0**）

**坐标计算：**

  ----------------------------------- -----------------------------------
  维度                                公式

  ----------------------------------- -----------------------------------

\| 2D \| **a** · **b** = x₁x₂ + y₁y₂ \|

\| 3D \| **a** · **b** = x₁x₂ + y₁y₂ + z₁z₂ \|

**教学要点：**

\- **a** · **b** = 0 ⟺ **a** ⊥ **b**（非零向量）

\- 投影的几何意义：**a** 在 **b** 上的投影 = **a** · **b** /
\\\|**b**\\\|

\- 求夹角：cosθ = **a** · **b** / (\\\|**a**\\\|\\\|**b**\\\|)

**9. 判断条件**

**10. 向量在立体几何中的应用**

**VEC-071 用向量证明/求解**

  ----------------------------------- ------------------------------------------
  应用                                向量方法

  证明线面平行                        直线方向向量可以用平面内两个向量线性表示

  证明线面垂直                        直线方向向量 · 平面内任意向量 = 0

  求二面角                            两个半平面的法向量的夹角（或补角）

  求线面角                            直线方向向量与法向量夹角的余角

  求异面直线角                        两条直线方向向量的夹角（取锐角）

  求点到面距离                        面上一点到该点的向量在法向量上的投影长度
  ----------------------------------- ------------------------------------------

**法向量求法：**

设平面法向量 **n** = (x, y, z)，取平面内两个不共线向量 **a** 和
**b**，解方程组：

\- **n** · **a** = 0

\- **n** · **b** = 0

**教学要点：**

向量法是解决立体几何问题的\"万能钥匙\"

建系→坐标表示→向量运算→得出结论

法向量在求角和求距离中极为重要

向量法虽然计算量可能较大，但思路统一不易出错
