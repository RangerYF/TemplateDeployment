**0. 开发对齐结论（2026-04-25）**

**0.1 当前仓库内的产品定位**

- P06 在本仓库中按“独立公式驱动演示模块”落地，不纳入力学 `Scene / Body / Joint` 模板链路。
- 可复用现有能力：工作台入口、通用布局、参数面板样式、时间轴、图表组件、基础 UI 规范。
- 不直接复用现有力学能力：Planck.js 刚体仿真、模板 JSON 场景资产、物体/约束注册表。

**0.2 与 P05 的边界**

- P05 解决“单体简谐振动”的平衡位置、位移对称性、回复力与周期问题。
- P06 解决“波的传播与图像对应”的波形图、单质点振动图、波叠加、驻波、多普勒问题。
- 实现层允许复用“频率/相位/时间轴”的教学语义，但 P06 不应作为 P05 模板的继续扩展来开发。

**0.3 开发拆分建议**

- 阶段 1：一维传播基座
  - 覆盖：`WAV-001`、`WAV-002`、`WAV-041`、`WAV-051`、`WAV-052`
  - 目标：先打通“波形图快照 + 选中质点振动图 + 时间轴 + 横/纵波切换”的公共骨架。
- 阶段 2：两列波叠加
  - 覆盖：`WAV-011`、`WAV-012`
  - 目标：在同一一维传播骨架上增加第二列波、相位差/异频控制和合成波显示。
- 阶段 3：驻波
  - 覆盖：`WAV-021`
  - 目标：复用叠加逻辑展示入射波、反射波和驻波，并补充波节/波腹标注。
- 阶段 4：多普勒
  - 覆盖：`WAV-031`
  - 目标：独立实现径向波前渲染与移动波源，不与一维弦波渲染器强耦合。

**0.4 实现约束**

- 页面形态应为“独立演示台”，而不是把波动模块塞进当前力学编辑画布。
- `WAV-002` 与 `WAV-051/052` 在实现层可合并为“介质/振动方向切换”，避免重复造轮子。
- `WAV-041` 应视为一维传播模块下的教学模式，不必单独建设第二套渲染器。
- 首版验收以“图像对应正确、参数联动正确、教学现象稳定可读”为主，不追求高精度物理仿真。

**2. 单列波传播**

**WAV-001 横波传播**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  振幅 A         5              1\~20          1              cm

  频率 f         2.0            0.1\~10        0.1            Hz

  波速 v         20             1\~100         1              cm/s

  质点数量 N     30             10\~60         1              ---
  -------------- -------------- -------------- -------------- --------------

波动方程：y(x,t) = A sin(ωt - kx)

ω = 2πf

k = 2π/λ

λ = v/f

波形图（某时刻快照）：y vs x

振动图（某质点）：y vs t

波形图与振动图对应规则：

波形图中某质点位置 → 该质点在振动图中的相位

点击波形图任意质点 → 下方显示该质点振动图

给定振幅、频率、时间条件，描述质点运动轨迹和路程

**WAV-002 纵波传播**

质点沿波传播方向振动

疏密变化动画

密部：质点聚集，位移指向同一点

疏部：质点分散，位移背离同一点

**3. 两列波叠加**

**WAV-011 同频波叠加（干涉）**

  --------------- -------------- -------------- -------------- --------------
  参数            默认值         范围           步进           单位

  波1振幅 A1      5              1\~20          1              cm

  波2振幅 A2      5              1\~20          1              cm

  频率            2.0            0.1\~10        0.1            Hz
  f（两波相同）                                                

  波速 v          20             1\~100         1              cm/s

  相位差 Δφ       0              0\~2π          π/12           rad
  --------------- -------------- -------------- -------------- --------------

叠加结果：y = y1 + y2

同相叠加（Δφ = 0）：振幅最大 A1+A2（加强）

反相叠加（Δφ = π）：振幅最小 \|A1-A2\|（减弱）

干涉条件：

频率相同

相位差恒定

振动方向相同

稳定干涉图样：加强点和减弱点位置固定不变

**WAV-012 异频波叠加（拍）**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  波1频率 f1     2.0            0.1\~10        0.1            Hz

  波2频率 f2     2.5            0.1\~10        0.1            Hz

  振幅 A         5              1\~20          1              cm
  -------------- -------------- -------------- -------------- --------------

拍频：f_拍 = \|f1 - f2\|

叠加图样不稳定，振幅周期性变化

教学要点：

同频 → 稳定干涉（图像不随时间变化）

异频 → 不稳定叠加（图像来回变化）

这是干涉的关键判据

**4. 驻波**

**WAV-021 驻波形成**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  入射波振幅 A   5              1\~20          1              cm

  频率 f         2.0            0.1\~10        0.1            Hz

  波速 v         20             1\~100         1              cm/s

  弦长 L         100            20\~200        1              cm
  -------------- -------------- -------------- -------------- --------------

两端固定弦的驻波可写为：y = 2A sin(kx) cos(ωt)

波节位置：kx = nπ → x = nλ/2（含两端固定点）

波腹位置：kx = (n + 1/2)π → x = (2n+1)λ/4

波节：始终不动（振幅 = 0）

波腹：振幅最大（= 2A）

共振条件（两端固定弦）：

L = nλ/2（n = 1, 2, 3, \...）

基频：f1 = v/(2L)

谐波：fn = nf1

预设共振模式：

  ----------------- ----------------- ----------------- -----------------
  模式              波节数            波腹数            频率

  基频（n=1）       2（两端）         1                 f1

  二次谐波（n=2）   3                 2                 2f1

  三次谐波（n=3）   4                 3                 3f1
  ----------------- ----------------- ----------------- -----------------

显示要素：

入射波（蓝色虚线）

反射波（绿色虚线）

合成驻波（橙黄色实线）

波节标注（黑色圆点）

波腹标注（黄色圆点）

**5. 多普勒效应**

**WAV-031 多普勒效应**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  波源频率 f0    440            100\~1000      10             Hz

  波速 v         340            100\~500       1              m/s

  波源速度 vs    50             0\~300         1              m/s
  -------------- -------------- -------------- -------------- --------------

观测者前方频率：f_前 = f0 v/(v - vs)（频率升高）

观测者后方频率：f_后 = f0 v/(v + vs)（频率降低）

可视化：

波前圆弧实时绘制

前方圆弧间距小（高频）

后方圆弧间距大（低频）

波源运动轨迹

预设场景：

  ----------------- ----------------- ----------------- -----------------
  场景              vs/v              前方频率比        后方频率比

  低速              0.1               1.11              0.91

  中速              0.3               1.43              0.77

  高速              0.5               2.00              0.67

  接近声速          0.9               10.0              0.53
  ----------------- ----------------- ----------------- -----------------

**8. 波动公式汇总**

  ----------------------------------- -----------------------------------
  公式                                说明

  v = λf                              波速 = 波长 × 频率

  T = 1/f = λ/v                       周期

  y = A sin(ωt - kx)                  沿 +x 方向传播的简谐波

  y = A sin(ωt + kx)                  沿 -x 方向传播的简谐波

  Δy = λL/d                           双缝干涉条纹间距

  f\' = f v/(v ± vs)                  多普勒效应频率
  ----------------------------------- -----------------------------------
