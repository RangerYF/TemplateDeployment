**3. 五大实验模板**

**EXP-001 伏安法测电阻（内接法）**

电路拓扑：电源 → 开关 → \[电流表 → (电压表 并联 待测电阻)\]

  ----------------------------------- -----------------------------------
  测量量                              表达式

  电压表示数 U                        U = U_R + U_A ≈ U_R

  电流表示数 I                        I = I_R（准确）

  测量值 R_测                         R_测 = U/I = R + R_A \> R（偏大）
  ----------------------------------- -----------------------------------

适用条件：R \>\> R_A（大电阻用内接法）

系统误差：电流表分压导致 R_测偏大

**EXP-002 伏安法测电阻（外接法）**

电路拓扑：电源 → 开关 → 电流表 → \[(电压表 并联 待测电阻)\]

  ----------------------------------- -----------------------------------
  测量量                              表达式

  电压表示数 U                        U = U_R（准确）

  电流表示数 I                        I = I_R + I_V \> I_R

  测量值 R_测                         R_测 = U/I = R R_V/(R+R_V) \<
                                      R（偏小）
  ----------------------------------- -----------------------------------

适用条件：R \<\< R_V（小电阻用外接法）

系统误差：电压表分流导致 R_测偏小

选择判据：R_A/R vs R/R_V，哪个小选哪个

**EXP-003 半偏法测电流计内阻**

步骤：

调节 R 使电流计满偏 I_g

并联 R_x，调节 R_x 使电流计半偏 I_g/2

此时 R_x = R_g（近似）

误差来源：并联 R_x 后总电阻减小，总电流增大

实际 R_g \< R_x（测量值偏大）

**EXP-004 测电源电动势和内阻**

电路：电源 → 开关 → 电流表 → 变阻器（分压/限流）

电压表接电源两端

  ----------------------- ----------------------- -----------------------
  数据点                  U (V)                   I (A)

  1                       5.80                    0.10

  2                       5.60                    0.20

  3                       5.40                    0.30

  4                       5.20                    0.40

  5                       5.00                    0.50

  6                       4.80                    0.60
  ----------------------- ----------------------- -----------------------

U-I 图：直线，斜率 = -r，截距 = ε

线性拟合：ε = 6.0V, r = 2.0Ω

**EXP-005 欧姆表原理**

电路：内部电源 + 电流计 + 调零电阻 + 待测电阻

满偏时 R_x = 0（短接）

半偏时 R_x = R_内

刻度特点：

非线性（左大右小）

中值电阻 = 内阻

每次换挡需重新调零

**EXP-006 电桥法测电阻**

平衡条件：R1/R2 = R3/R4

灵敏电流计示数为零时达到平衡

精度高于伏安法

**4. 电路分析公式**

**基本定律**

  ----------------------------------- -----------------------------------
  定律                                公式

  欧姆定律                            U = IR

  闭合电路欧姆定律                    ε = I(R + r)

  串联电阻                            R_总 = R1 + R2 + \...

  并联电阻                            1/R_总 = 1/R1 + 1/R2 + \...

  电功率                              P = UI = I²R = U²/R

  焦耳定律                            Q = I²Rt
  ----------------------------------- -----------------------------------

**动态电路分析方法**

分析步骤：

确定哪个元件参数改变

判断外电路总电阻变化方向

由 I = ε/(R+r) 判断总电流变化

由 U = ε - Ir 判断路端电压变化

逐个分析各支路电流和电压
