# 力学与运动模拟需求进度清单

更新时间：2026-04-17

## 说明
- 本文档按用户原始反馈顺序逐条记录。
- 进度口径以当前仓库真实状态为准，优先依据：
  - `src/templates/catalog-data/templates.json`
  - `src/templates/presets/*`
  - `public/templates/scenes/*`
  - `src/engine/*`
  - `src/store/*`
  - `src/components/*`
- 状态定义：
  - `已做完`：当前仓库中已有对应实现，且主目标已经落地
  - `部分完成`：已有基础实现或第一轮修正，但仍需继续打磨/验收
  - `还没做`：当前仓库中未见对应落地，或仍明显不符合需求

### 3. 斜面上物块受力模型

### 4. 传送带模型
- 需求点：

- 当前情况：

- 备注：
