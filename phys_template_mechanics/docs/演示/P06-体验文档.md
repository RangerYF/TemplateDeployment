## 一、当前版本定位

P06 当前已作为独立演示台接入工作台，用于展示波形图、单质点振动图、波叠加、驻波和多普勒效应。

当前版本适合以下课堂场景：
- 用单波传播讲清“波形图快照”和“单质点振动图”的对应关系
- 用双波叠加讲清稳定干涉与拍现象
- 用驻波讲清波节、波腹和谐波模式
- 用多普勒讲清前密后疏、前后频率变化

## 二、体验入口

当前可通过以下方式进入：
- 打开工作台后点击 `P06 波动与振动演示台`
- 直接进入 `#p06`
- 通过 `#p06?module=模块编号` 直达指定模块

当前已接入模块如下：

| 模块 | 名称 | 当前可体验重点 |
|---|---|---|
| `WAV-001` | 横波传播 | 波形快照、选点振动图、相位与轨迹 |
| `WAV-002` | 纵波传播 | 疏密变化、选点位移图、传播方向 |
| `WAV-011` | 同频波叠加 | 分量波、合成波、稳定干涉、相长相消 |
| `WAV-012` | 异频波叠加 | 分量波、合成波、拍频、包络周期 |
| `WAV-021` | 驻波形成 | 入射波、反射波、驻波、波节波腹、谐波模式 |
| `WAV-031` | 多普勒效应 | 波前圆弧、移动波源、前密后疏、前后频率变化 |
| `WAV-041` | 波形平移 | `t` 与 `t + Δt` 的波形叠加对比 |
| `WAV-051 / WAV-052` | 横纵波对比 | 振动方向与传播方向差异、方向箭头动画 |

## 三、推荐体验顺序

### 1. 单波基础
先体验 `WAV-001` 或 `WAV-002`：
- 调整振幅、频率、波速、质点数量
- 点击主视图中的任意质点
- 观察下方振动图如何跟随切换

这一步适合先建立“位置变化”和“相位变化”的基本认知。

### 2. 波形平移
进入 `WAV-041`：
- 调整时间间隔 `Δt`
- 观察实线与虚线之间的平移距离
- 对照读数区中的 `Δx = vΔt`

这一步适合讲清“波向前传播，但单个质点只在平衡位置附近往复振动”。

### 3. 双波叠加
进入 `WAV-011` 和 `WAV-012`：
- 在同频模块里调整 `Δφ`
- 在异频模块里调整 `f₁`、`f₂`
- 观察主视图和选点振动图如何同时变化

这一步适合讲清稳定干涉与拍现象的差异。

### 4. 驻波
进入 `WAV-021`：
- 切换基频、二次、三次谐波
- 观察波节、波腹位置和数量变化
- 点击不同位置的质点，观察振幅大小差异

这一步适合把“固定端驻波”与“单列传播波”区分开。

### 5. 多普勒
进入 `WAV-031`：
- 调整 `f₀`、`v`、`vs`
- 使用预设速度比快速切换
- 对照主视图波前密度和读数区频率变化

这一步适合把图像变化和公式变化对应起来。

## 四、当前关键交互

| 交互 | 当前效果 |
|---|---|
| 模块切换 | 左侧列表切换不同波动主题 |
| 播放控制 | 支持播放、暂停、归零、倍率切换 |
| 质点选中 | 点击主视图中的质点后，下方读图切换到对应位置 |
| 参数滑杆 | 调整后主视图与读数即时更新 |
| 显示选项 | 部分模块支持切换分量波、合成波、波节波腹标注、方向箭头 |
| 多普勒预设 | 可快速切换低速、中速、高速、近声速四类速度比 |

## 五、当前可讲解内容

当前版本已经适合直接讲解以下概念：
- 波长 `λ`、周期 `T`、角频率 `ω`、波数 `k`
- 单个质点的相位、位移、振动区间、累计路程
- 同频稳定干涉与异频拍现象
- 固定端驻波的波节、波腹和谐波模式
- 多普勒效应中的前后频率变化和前密后疏
- 横波与纵波中“振动方向”和“传播方向”的关系

## 六、当前版本边界

当前版本边界如下：
- 当前交付形态是工作台内的独立演示页，不是单独导出的 HTML 成品
- 当前采用公式驱动演示，不接入力学模板编辑器的 `Scene / Body / Joint` 体系
- 当前重点是课堂可读性和概念对应，不追求高精度物理仿真
- 驻波当前采用“谐波模式 + 自动频率计算”的交互方式，不提供独立频率输入

## 七、当前体验结论

P06 当前已经具备完整的首版体验主线：
- 有独立入口
- 有模块切换
- 有参数控制
- 有主视图
- 有教学读数
- 有基础交互和显示辅助
