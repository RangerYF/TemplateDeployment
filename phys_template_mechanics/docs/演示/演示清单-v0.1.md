## 一、版本定位

`v0.1` 已交付一个可用的力学模板编辑器底座，并完成首批教学场景落地。

当前版本包含两类内容：
- 编辑器能力：进入工作台、加载场景、编辑物体、运行仿真、查看分析结果。
- 教学场景：基于编辑器能力落地 `5` 个模块、`40` 个场景。

本版交付范围：

| 模块 | 模板数 | 教学方向 |
|---|---:|---|
| P-01 受力分析 | 16 | 受力图、约束力、典型力学模型 |
| P-02 运动模拟 | 7 | 位移、速度、加速度、轨迹 |
| P-05 简谐运动 | 3 | 弹簧振子、单摆、周期现象 |
| P-12 动量守恒 | 7 | 碰撞、相互作用、系统动量 |
| P-14 机械能守恒 | 7 | 动能、势能、机械能转化 |

## 四、推荐体验顺序

### 步骤 A：进入与切换（3 分钟）
1. 打开工作台，切换 5 个模块。
2. 点击一个模板进入编辑器。
3. 返回工作台后再进入另一个模块。

### 步骤 B：编辑与分析（5 分钟）
1. 打开 `FM-011`，选中物体并调整角度、摩擦等参数。
2. 在同一场景中添加或调整外力，再播放仿真并查看受力分析。
3. 拖动时间轨道回看关键时刻。

### 步骤 C：观察不同运动现象（4 分钟）
1. 打开 `MOT-021`，设置初始速度后观察轨迹和图表变化。
2. 打开 `SHM-003`，调整参数后观察周期变化。
3. 打开 `MOM-001`，运行碰撞过程并观察前后状态。

### 步骤 D：对比体验（3 分钟）
1. 对照 `ENE-021` 和 `ENE-022`。
2. 分别运行仿真并观察两者差异。

## 七、附录 A：主动设计补充

除基础需求外，`v0.1` 还补充了一批在研发过程中沉淀出的主动优化。这些内容不依赖某一个模板，而是作用在编辑器底座本身。

### A. 参数与教学表达

| 补充项 | 内容 |
|---|---|
| 初始运动状态独立配置 | 在属性面板中单独提供“初始运动”入口，支持设置初速度与初始加速度。 |
| 初速度双表示 | 同时支持 `vx/vy` 与“速度大小 + 方向角”两种输入方式，并保持双向联动。 |
| 速度可视化 | 在画布中显示速度箭头与文字说明，便于直接观察方向与大小。 |
| 外力工具 | 可直接对物体施加外力，并用大小、方向可视化表达推力、拉力、斜向力。 |
| 环境参数面板 | 未选中物体时，右侧切换为环境配置面板，可直接设置重力加速度 `g`。 |
| 回放读数增强 | 时间回放时，图表区显示当前时间竖线与对应数值，便于对照分析。 |
| 尺度感知补充 | 画布增加 `1m` 标尺，帮助建立长度与速度的直观尺度感。 |

### B. 编辑体验增强

| 补充项 | 内容 |
|---|---|
| 撤销/重做双入口 | 同时支持快捷键与画布左下角按钮，降低频繁调整时的操作负担。 |
| 框选与多选 | 支持框选、多选、Shift 追加/移除，便于整体移动和分析多个对象。 |
| 连接件创建引导 | 绳、杆、弹簧、滑轮等连接件支持拖拽或点击进入创建流程，并提供步骤提示与 `Esc` 退出。 |
| 拖放实时预览 | 从物体库拖入画布时立即显示真实预览，减少“松手后才发现位置不对”的情况。 |
| 默认尺寸系统化 | 对不同物体的默认大小做统一设计，保证进入场景时比例更自然。 |
| 物体自然翻转 | 斜面等非对称物体在调整过程中可自然翻转方向，减少额外操作。 |

### C. 场景搭建与可读性优化

| 补充项 | 内容 |
|---|---|
| 地面联动物体上移 | 调整地面高度时，位于地面上的物体可随地面一起上移，避免重新逐个摆放。 |
| 地面防穿透 | 物体不能被直接拖到地面以下，场景边界更稳定。 |
| 吸附回正 | 物体从斜面拖回水平面时可自动回正，减少手动旋转修正。 |
| 约束拖拽收敛 | 受绳、杆、滑轮约束的物体在拖拽时优先回收到合法长度，避免出现明显漂移。 |
| 力箭头避让 | 多个力同时显示时，会对箭头和标签进行避让处理，减少重叠遮挡。 |
| 重力箭头外显 | 重力箭头保持从质心出发的语义，同时尽量不遮挡物体主体。 |

### D. 仿真与分析增强

| 补充项 | 内容 |
|---|---|
| 控制区重排 | 播放、暂停、停止和时间轨道统一收纳到独立控制区，使用路径更清晰。 |
| 时间回溯 | 仿真后可回到任意历史时刻，支持重新观察关键瞬间。 |
| 分析组 | 多个物体可组成分析组，从单体分析扩展到系统分析。 |
| 摩擦热与碰撞标记 | 能量与动量分析中补充摩擦热、碰撞时刻等信息，增强教学解释力。 |
