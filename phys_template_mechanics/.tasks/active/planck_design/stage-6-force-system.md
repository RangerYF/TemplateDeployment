## 背景与动机

阶段5已完成约束系统（绳/杆/弹簧/滑轮绳），物体之间可建立物理连接。下一步需要实现力的体系与受力分析视角——这是物理编辑器面向教学的核心能力。

物理课本中受力分析是最基础的教学内容：
- 教师需要展示物体受到哪些力（重力、支持力、摩擦力、张力、弹簧力、外力）
- 需要对力进行正交分解（沿斜面/垂直斜面方向）
- 需要验证力的平衡关系（静止物体合力为零）

## 目标

1. 实现 ForceTool（点击物体 → 拖拽方向和长度 → 创建用户外力）
2. 实现力的收集系统（从 Planck.js 提取系统自动力 + 记录用户主动力）
3. 实现力的可视化渲染（力箭头 + 标签 + 合力 + 防重叠）
4. 实现力的正交分解（水平/竖直方向 或 沿斜面/垂直斜面方向）
5. 实现受力分析视角切换（视角下所有物体显示力箭头）
6. 扩展属性面板（力列表显示、显隐控制、外力编辑、分解操作）

## 6.0 设计方案（待确认）

### A. 力的视觉编码

#### 颜色方案：按力类型色彩编码

物理课本中力通常用统一颜色+字母标签区分。但数字工具中，**色彩编码能大幅提升辨识效率**，尤其当多个力叠加时。方案采用"类型色 + 课本标签"双编码。

| 力类型 | 颜色 | 色值 | 课本标签 | 说明 |
|--------|------|------|---------|------|
| 重力 | 深绿 | `#16a34a` | G | 地球引力，始终竖直向下 |
| 支持力 | 紫色 | `#8b5cf6` | N | 接触面法向反力 |
| 摩擦力 | 橙色 | `#f97316` | f | 接触面切向阻力 |
| 张力/弹簧力 | 青色 | `#0891b2` | T | 约束反作用力 |
| 外力（用户） | 红色 | `#ef4444` | F | 用户手动施加，最醒目 |
| 合力 | 深灰 | `#374151` | F合 | 虚线箭头，区别于单个力 |
| 分量 | 与原力同色淡化 | 原色 50% 透明 | Fx / Fy | 虚线箭头 |
| **选中态** | 蓝色 | `#3b82f6` | — | 与物体/约束选中态统一 |

#### 箭头样式

```
箭头结构示意：
                    ◄─ 箭头宽度 8px（固定）
  ●━━━━━━━━━━━━━━▶
  ↑               ↑
  起点(边缘)      箭头尖端

线宽：2px（默认），2.5px（hover/选中）
箭头头部：实心三角形，长 10px，底宽 8px
起点标记：3px 实心圆点（与箭头同色）

起点位置：从物体边缘出发（非质心），沿力方向从质心偏移到表面 + 2px gap
  - 圆形物体：偏移 = 半径 + EDGE_GAP(2px)
  - 矩形物体：射线与旋转后矩形边界交点 + EDGE_GAP(2px)
  - 来源：p01 实践——从边缘出发箭头不被物体遮挡，视觉更清晰
```

#### 箭头长度与力大小的映射

```
采用对数缩放（非线性），解决小力和大力共存时小力"看不见"的问题。

forceToLength(magnitude) = MIN + (MAX - MIN) × log(1 + magnitude) / log(1 + BASE)

MIN = 30px    — 最小箭头长度（力很小也可见）
MAX = 180px   — 最大箭头长度（防止过长遮挡）
BASE = 100    — 力的典型范围参考值（100N 时接近最大长度）

效果示例：
  1N   → ~34px    （小力依然可见）
  10N  → ~73px    （中等长度）
  50N  → ~133px   （较长但不夸张）
  100N → ~180px   （接近最大值）

来源：p01 实践——线性映射导致 1N 和 50N 共存时小力几乎不可见
```

#### 力标签格式

```
标签格式：「字母符号 中文名 = 数值 N」
示例：G 重力 = 19.6N
      N 支持力 = 19.6N
      f 摩擦力 = 5.9N
      F 外力 = 10.0N
      T 张力 = 8.0N

位置策略（区分水平力和竖直力）：
  水平力（|dx| > |dy|）：标签在箭头尖端外侧（沿力方向延伸 12px）
  竖直力（|dy| ≥ |dx|）：标签在箭头中点侧面（垂直偏移 12px+4px）
  张力/弹簧力：额外垂直偏移 6px（远离绳/杆/弹簧，防止被连接件遮挡）

每个标签生成 4 个候选位置（按偏好排序），贪心选第一个不重叠的。
详见参考文档 stage-6-p01-reference.md。

字号：11px
颜色：与箭头同色
```

### B. ForceTool 交互流程

#### 创建外力

```
交互流程：
1. 切换到力工具（工具栏按钮或快捷键 F）
2. 鼠标移到物体上 → 物体高亮 + 光标变为十字
3. 在物体上按下鼠标 → 确定施力点（物体中心）
4. 拖拽鼠标 → 实时预览力箭头（智能吸附方向 + 跟随距离）
5. 松手 → 创建外力（AddForceCommand，可撤销）

大小 = 拖拽距离 × scaleFactor（与渲染比例一致）
最小距离阈值：拖拽距离 < 10px 时取消创建（防误触）
```

#### 方向智能吸附

```
核心思路：根据物体当前物理环境，自动生成"吸附方向候选集"，
拖拽时在 ±12° 范围内吸附到最近的候选方向。

候选方向生成规则（按物体所处环境自动判定）：

1. 全局基础方向（始终存在）：
   - 水平：0°（→）、180°（←）
   - 竖直：90°（↑）、270°（↓）

2. 接触面方向（物体接触斜面/曲面时追加）：
   - 沿接触面：θ、θ+180°
   - 垂直接触面：θ+90°、θ-90°
   - θ 从 SnapEngine 或接触法线获取

3. 约束方向（物体连有绳/杆/弹簧时追加）：
   - 沿约束连线：指向另一端、背离另一端
   - 垂直约束连线：连线 ±90° 两个方向

吸附行为：
  - 拖拽角度与候选方向夹角 < 12° → 自动吸附（箭头变色提示）
  - 拖拽角度不在任何候选范围内 → 自由方向（无吸附）
  - Alt 键：禁用吸附（与移动物体时 Alt 禁用吸附一致）
  - Shift 键：锁定到 15° 倍数网格（覆盖智能吸附）

视觉反馈：
  - 吸附激活时：箭头方向锁定 + 辅助虚线沿吸附方向延伸
  - 未吸附时：箭头自由跟随鼠标

示例场景：
  物块在 30° 斜面上 → 候选方向：
    0°/90°/180°/270°（全局）
    + 30°/210°（沿斜面）
    + 120°/300°（垂直斜面）
    = 共 8 个吸附方向

  悬挂在绳上的物块 → 候选方向：
    0°/90°/180°/270°（全局）
    + 绳的方向（如 250°）和反方向（70°）
    + 垂直绳方向（160°）和反方向（340°）
    = 共 8 个吸附方向
```

#### 编辑已有外力

```
方案：拖拽箭头尖端调整 + 面板精确编辑

画布上：
  - hover 外力箭头尖端 → 光标变为 move
  - 拖拽尖端 → 实时调整方向和大小（同样启用智能吸附）
  - Shift 拖拽 → 锁定到 15° 网格
  - Alt 拖拽 → 禁用吸附，自由方向
  - 松手 → ChangeForcePropertyCommand

  - 大小：NumberInput（精确输入）

系统自动力（重力/支持力/摩擦力/张力）不可编辑，仅显示。

  - 物体外观不变（不变淡）— 保持场景可读性
  - 力箭头渲染在物体层之上
  - 可同时选中物体查看详细力列表
  - ForceTool 仅在此视角下可用

仿真模式下的受力视角：
  - 力箭头实时更新（跟随仿真计算结果）
  - 更新频率与渲染帧同步（每帧更新力数据）
  - 静止场景：力保持稳定（滤波后无抖动）
```

```

```

#### 分解渲染

```
分解示意（重力沿斜面分解）：

         ╲  Gy
          ╲  ↑(虚线)
           ╲│
            ●━━━━▶ Gx (虚线)
            │
            │ G (实线)
            ▼

分量箭头：
  - 虚线（setLineDash([6,4])）
  - 颜色：原力颜色的 50% 透明度
  - 线宽：与原力相同

直角标记：
  - 分解坐标原点处画直角符号（小方块 6×6px）
  - 颜色：浅灰 #94a3b8

分量标签：
  - 格式同普通力：「Gx = 9.8N」「Gy = 17.0N」
```

#### 分解动画（分阶段渐进）

```
来源：p01 实践——一次性出现所有元素会造成视觉混乱，分阶段渐入更友好。

总时长：0.8s 渐入，0.3s 渐出
进度变量：progress 0→1

阶段 1（progress 0%-30%）：坐标轴参考线淡入
  - 沿分解方向画两条浅色参考线（alpha = 0.3 × progress/0.3）
  - 线过物体中心，长度延伸到视口边缘
  - 虚线 setLineDash([4,4])，颜色 #94a3b8

阶段 2（progress 30%-60%）：分量箭头 + 引导虚线出现
  - 分量箭头从零长度生长到目标长度（线性插值）
  - 引导虚线：从原力终点到分量轴的垂直投影线
  - 引导虚线用细线（1px），颜色 #94a3b8

阶段 3（progress 50%-80%）：直角标记出现
  - 小方块 6×6px 从透明淡入
  - alpha = (progress - 0.5) / 0.3

阶段 4（progress 60%-100%）：分量标签淡入
  - 标签文字从透明到实色
  - alpha = (progress - 0.6) / 0.4

取消分解：反向播放，0.3s 全部淡出
```

### F. 力箭头防重叠策略

#### 箭头起点

```
策略：从物体边缘出发（质心沿力方向偏移到表面）

来源：p01 实践——从边缘出发箭头不会被物体本身遮挡，视觉更干净。

计算方式：
  center = 物体几何中心（注意 block 的 position 是底边中心，需加半高）
  edgeStart = center + normalize(forceDir) × (到边界距离 + EDGE_GAP)
  EDGE_GAP = 2px（屏幕空间）

  圆形：到边界距离 = radius
  矩形：射线与旋转后矩形边界交点（用参数 t 求最近交点）

理由：
  1. 物理课本简化图确实从重心画，但课本是静态图
  2. 数字工具中物体有填充色和描边，从中心出发箭头会被遮挡
  3. 从边缘出发后箭头完全可见，且起点隐含"从物体表面施力"的语义
```

#### 共线防重合（力-力 + 力-连接件）

```
来源：p01 实践——分层防重叠，三层独立处理。

层1：力-力共线检测
  判定：两个力方向的点积绝对值 > 0.87（≈cos30°，夹角 < 30°）
  处理：将后一个力垂直偏移 COLLINEAR_OFFSET = 10px（屏幕空间）
  多个共线力：递增轨道（第1个偏移10px，第2个20px，…）

层2：力-连接件共线检测
  判定：力方向与关联绳/杆/弹簧方向的点积 > 0.87
  处理：力垂直偏移到远离连接件的一侧
  递增轨道：多个力与同一连接件共线时，递增偏移

层3：合力的特殊偏移
  判定：合力方向与某独立力或连接件共线
  处理：合力偏移到"负方向轨道"（偏移方向与独立力相反）
  偏移距离 14px（比独立力的 10px 稍大，视觉区分）

注意：
  - 偏移仅影响渲染位置，不影响力的实际方向/数值
  - 标签跟随偏移后的箭头位置
  - 30° 阈值来自 p01 经验调参，比 15° 更实用
```

#### 合力冗余检测

```
来源：p01 实践——避免合力与独立力重复显示。

规则：
  - 合力大小和方向与某个独立力几乎相同（差值 < 0.01）→ 不渲染合力
  - 合力为零（< 0.01N）→ 不渲染箭头，面板显示"合力为零，受力平衡"
```

#### 标签防重叠

```
策略：候选位置 + 贪心布局（来源 p01 placement.ts）

1. 每个力标签生成 4 个候选位置（按偏好排序）：
   水平力：尖端外侧上方 → 尖端外侧下方 → 中点上方 → 中点下方
   竖直力：中点右侧 → 中点左侧 → 尖端右侧 → 尖端左侧

2. 贪心布局：按偏好顺序逐一尝试，选第一个不重叠的
   重叠检测：AABB 包围盒 + LABEL_PAD = 6px 间距
   occupied 列表：记录已放置的标签区域

3. 全部重叠时：选最高偏好的候选（保证总有输出）

4. 张力/弹簧力标签：额外垂直偏移 6px（远离连接件一侧）

5. 不使用全局障碍物系统（局部物体内防重叠，O(n) 复杂度）
```
