## 交互方案

### 1. 左侧列表新增“连接件”分组

- 在 `ObjectPanel` 中新增分组：`连接件`
- 分组项：`绳`、`杆`、`弹簧`、`滑轮绳`
- 分组项行为：
  - 支持与物体卡片一致的拖拽手势
  - 拖入 Canvas 不直接创建实体，而是进入连接创建流程

### 2. 连接件拖拽后的状态机

#### 绳 / 杆 / 弹簧（两步）

1. `dragStart(连接件)` -> 鼠标旁提示：`① 请选择第一个物体`
2. 选中第一个物体后 -> 提示更新：`② 请选择第二个物体`
3. 选中第二个物体后 -> 创建连接件并结束引导

#### 滑轮绳（三步）

1. `dragStart(滑轮绳)` -> 提示：`① 请选择第一个物体`
2. 选中第一个物体后 -> 提示：`② 请选择滑轮座`
3. 选中滑轮座后 -> 提示：`③ 请选择第二个物体`
4. 选中第二个物体后 -> 创建滑轮绳并结束引导

### 3. 鼠标旁浮层提示规范

- 位置：光标右下偏移（建议 `+12px, +16px`）
- 更新时机：每一步选择完成后立即刷新文案
- 终止条件：
  - 成功创建后自动消失
  - `Esc` 取消后立即消失
  - 点击空白非法目标时保留提示并给短暂错误提示（如“请选择可连接物体”）

## 已决策项（快捷键 / 光标 / Tooltip）

### A. 快捷键方案

1. 方案 A（未选）：保持 `V/J/F`，新增 `Shift+J` 打开连接件快速选择浮层（绳/杆/弹簧/滑轮绳）
2. 方案 B（未选）：为连接件提供直达键（`R` 绳、`B` 杆、`S` 弹簧、`P` 滑轮绳）
3. **方案 C（已选定）**：不新增快捷键，仅保留左侧拖拽入口

### B. 光标方案

1. **方案 A（已选定）**：连接创建阶段统一 `crosshair`，并在鼠标旁显示当前步骤徽标
2. 方案 B（未选）：不同连接件使用不同自定义光标（维护成本更高）
3. 方案 C（未选）：沿用默认箭头，仅靠浮层提示

### C. Tooltip 方案

1. **方案 A（已选定）**：连接件卡片 hover 显示“用途 + 创建步骤 + 快捷键”
2. 方案 B（未选）：仅显示简短名称，详细步骤放到引导浮层
3. 方案 C（未选）：关闭卡片 Tooltip，全部靠画布内引导
