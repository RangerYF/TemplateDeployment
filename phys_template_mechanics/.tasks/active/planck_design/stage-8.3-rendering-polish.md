## 调研总结

### 教材画法核心特征

1. **线条优先，填充极淡**：教材是黑/深灰线条示意图，填充很少或极淡灰
2. **固定端斜线阴影（Hatching）**：几乎所有固定支撑面都有 45° 短斜线标记——这是教材中最具标志性的视觉符号
3. **铰链/轴心统一画法**：空心圆圈 + 中心实心点（表示可转动但固定的轴）

### 当前实现 vs 教材标准

| 物体 | 当前状态 | 教材标准 | 差距 |
|------|----------|----------|------|
| 半球面 | 弧线 + 半透明填充，无厚度 | 单线弧线，弧外侧可有斜线阴影，两端有端部封口 | 中 |
| 滑轮座 | 圆环 + 中心点 + 倒三角 | 圆盘 + 凹槽感 + 中心轴点 + 固定面斜线阴影 | 中 |
| 固定锚点 | 实心圆 + 外圈环 + 三角指示器 | 固定面粗线 + 45°斜线阴影 + 铰链圆圈 | 大 |
| 传送带 | 矩形+滚轮+轴心+箭头+斜线纹理 | 两端圆+外公切线皮带+箭头，素面无纹理 | 小 |
| V形槽 | 有厚度梯形壁面，有填充 | V形线条，无厚度感，底部可有斜线阴影 | 中 |
| 斜面 | 三角形+填充+描边 | 三角形+填充+描边+**底部斜线阴影** | 小 |

## 优化方案

### 0. 通用基础设施：Hatching 工具函数

**新增** `src/renderer/hatching.ts`：

```typescript
/**
 * 在指定线段的一侧绘制 45° 斜线阴影（固定端标记）
 * @param ctx       Canvas 上下文
 * @param x1,y1     线段起点（屏幕坐标，已在 body 局部坐标系）
 * @param x2,y2     线段终点
 * @param side      斜线方向（'outside' = 远离物体一侧）
 * @param options    hatchLength(斜线长度), hatchSpacing(间距), color, alpha
 */
function drawHatching(ctx, x1, y1, x2, y2, side, options?)
```

多个物体共用此函数：斜面底部、锚点固定面、滑轮座固定面、半球面端部。

---

### 1. 固定锚点 anchor — 改动最大

**当前**：实心深色圆 + 虚线外圈 + 三角指示器

**优化为教材标准的「固定铰支座」画法**：

```
挂载面（mountSide=top 示例）：

    ╱╱╱╱╱╱╱╱╱╱   ← 45° 斜线阴影（4-6条）
    ━━━━━━━━━━   ← 固定面粗实线
         │         ← 连接杆
         ▽         ← 倒三角支架
         ○         ← 铰链圆圈（空心圆 + 中心实心点）
```

- 固定面：根据 `mountSide` 方向画一条短粗实线（约 2r 宽）
- 斜线阴影：在粗实线外侧画 4-6 条 45° 短斜线
- 连接杆：从固定面到铰链点的短直线
- 铰链点：空心圆圈 + 中心实心小点（替代当前的实心大圆）

---

### 2. 滑轮座 pulley-mount — 中等改动

**当前**：圆环 + 中心点 + 倒三角

**优化**：

```
    ╱╱╱╱╱╱╱╱   ← 固定面斜线阴影
    ━━━━━━━━   ← 固定面粗实线
       │        ← 支架杆
      ╭─╮
     │ ● │     ← 滑轮：外圈（稍粗描边）+ 中心轴实心点
      ╰─╯
```

- 顶部支架：固定面粗实线 + 斜线阴影 + 向下的连接杆（替代当前倒三角）
- 轮体：保持圆环，加轻微凹槽感（外圈描边加粗 或 画双同心圆）
- 中心轴：保持中心实心小点

---

### 3. 半球面 hemisphere — 中等改动

**当前**：弧线 + 半透明面积填充

**优化**：

```
    ─╮               ╭─    ← 端部封口短线
      ╲             ╱
       ╲  ╱╱╱╱╱╱  ╱       ← 弧线外侧斜线阴影
        ╲╱╱╱╱╱╱╱╱╱
         ╰───────╯         ← 主弧线（加粗）
```

- 主弧线：保持，线宽从 2 增加到 3（更醒目）
- 去掉/大幅减弱半透明填充（从 0.15 降到 0.05 或 0）
- 弧线外侧（凸面）：沿弧线法线方向画稀疏 45° 短斜线，表示固定实体
- 端部封口：弧线两端各画一条向外的短线段（约 5px），表示轨道截面边缘

---

### 4. 斜面 slope — 小改动

**当前**：三角形 + 填充 + 描边，无底部标记

**优化**：

```
    ╱│
   ╱ │
  ╱  │
 ╱───┘
 ╲╲╲╲╲   ← 底边下方 45° 斜线阴影
```

- 底边下方增加一排 45° 短斜线（约 5-8px 高）
- 仅在 `isStatic` 时显示斜线（动态斜面不画）
- 颜色：与描边同色，alpha 约 0.4

---

### 5. 传送带 conveyor — 微调

**当前**：矩形皮带 + 两端滚轮 + 轴心 + 方向箭头 + 斜线纹理

**优化**：

- 皮带面减弱或去掉斜线纹理（`hatchAlpha` 从 0.3 降到 0.12），教材中传送带面通常是素面
- 上表面线条略加粗（接触面，lineWidth +0.5）表示承载面
- 其余保持不变（整体已接近教材风格）

---

### 6. V形槽 groove — 小改动

**当前**：有厚度梯形壁面，有填充

**优化**：

- 壁面填充色降低透明度（更淡），突出线条
- V形底部外侧增加斜线阴影标记（表示固定）
- 仅在 `isStatic` 时显示底部斜线
