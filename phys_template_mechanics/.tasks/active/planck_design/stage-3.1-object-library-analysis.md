## 三、物体分层建议

### 第一梯队：核心物体（已实现，覆盖 80% 场景）

| 物体 | 类型 | 说明 |
|------|------|------|
| block | Dynamic | 最常用，几乎所有场景都用 |
| ball | Dynamic | 圆形物体，圆周运动/单摆/天体 |
| ground | Static（内置） | 水平面，不可删除 |
| slope | 用户设定 | 斜面场景，三角形 |
| wall | 用户设定 | 竖直障碍 |
| anchor | 用户设定 | 悬挂/连接的固定端 |
| bar | Dynamic | 杆件/杠杆 |

### 第二梯队：高频扩展物体（覆盖圆周运动 + 滑轮场景）

| 物体 | 类型 | Planck.js 实现 | 难度 | 覆盖场景 |
|------|------|---------------|------|---------|
| **turntable** (转盘) | Kinematic | `type: 'kinematic'` + `setAngularVelocity(ω)` | 中 | 水平圆周(摩擦提供向心力)、圆盘旋转 |
| **pulley-mount** (滑轮座) | Static | anchor + 圆形渲染 + PulleyJoint 支持 | 中 | 定滑轮、阿特伍德机、动滑轮 |

### 第三梯队：专项物体（单一模块使用，可后期补充）

| 物体 | 类型 | Planck.js 实现 | 难度 | 覆盖场景 |
|------|------|---------------|------|---------|
| **conveyor** (传送带) | Static | Box + `pre-solve` 修改接触切线速度 | 高 | 传送带模型(FM-054) |
| **hemisphere** (半球面) | Static | ChainShape(圆弧采样点) | 中 | 半球面模型(FM-053) |
| **groove** (V形槽) | Static | 两条 EdgeShape 或 Polygon | 低 | V形凹槽模型(FM-055) |

### 特殊处理：天体（P-09）

P-09 天体运动场景较为特殊：
- **无地面**，无碰撞，纯引力驱动
- 物体本质是 `ball`，但渲染需要太空风格（大尺寸、发光效果、轨道线）
- 引力通过每帧 `applyForce(F=GMm/r²)` 实现

**建议方案**：
- **方案A**：不新增物体类型，用 `ball` + 场景级"引力模式"开关实现。天体外观通过场景主题/渲染模式切换。
- **方案B**：新增 `celestial` 类型，自带 mass（用天文单位）、radius（渲染半径）、引力计算参数。物体面板放在"天体"分组下。

> 推荐方案A——天体功能更适合作为"场景模式"而非物体类型，因为 P-09 的物理规则（无碰撞、自定义引力）与其他模块完全不同。

## 四、建议的物体库面板结构

```
╔══════════════════╗
║ 物体库           ║
╠══════════════════╣
║ ▼ 基础物体       ║  ← 第一梯队（已实现）
║  [□] 物块        ║
║  [○] 球体        ║
║  [━] 杆件        ║
╠──────────────────╣
║ ▼ 支撑面         ║  ← 第一梯队（已实现）
║  [△] 斜面        ║
║  [│] 墙壁        ║
║  [•] 固定锚点    ║
╠──────────────────╣
║ ▼ 机构           ║  ← 第二梯队（待实现）
║  [◎] 转盘        ║
║  [⊙] 滑轮座      ║
╠──────────────────╣
║ ▼ 特殊表面       ║  ← 第三梯队（后期补充）
║  [≈] 传送带      ║
║  [⌒] 半球面      ║
║  [∨] V形槽       ║
╚══════════════════╝
```

与设计文档 8.1 对比：
- 去掉了 `[▬] 地面`（内置不可添加，不需要在面板中显示）
- 去掉了 `[⚖] 杠杆` / `[⊙] 滑轮`（复合体，由 bar+anchor+Joint 组装，不是独立物体）
- 第三梯队的特殊表面可在对应模块开发时再添加

## 五、实现优先级建议

| 优先级 | 物体 | 推荐阶段 | 理由 |
|--------|------|---------|------|
| P0 | block, ball, ground, slope, wall, anchor, bar | ✅ 阶段3已完成 | 核心物体 |
| P1 | turntable | 阶段4（约束系统）或独立子阶段 | 圆周运动场景需要 |
| P1 | pulley-mount | 阶段4（约束系统） | 与 PulleyJoint 配合实现 |
| P2 | groove | 阶段4之后 | 简单实现，V形两条边 |
| P2 | hemisphere | 阶段4之后 | ChainShape 弧线采样 |
| P3 | conveyor | 专项阶段 | 需要 pre-solve hook，实现复杂度高 |
