## 2. T3 阶段范围结论

### 2.1 覆盖模块
T3 只覆盖 `A 直接实现` 模板，共 5 个模块 40 个模板：
- `P-01 受力分析`：16 个
- `P-02 运动模拟`：7 个
- `P-05 简谐运动`：3 个
- `P-12 动量定理及动量守恒`：7 个
- `P-14 机械能守恒`：7 个

### 2.2 当前编辑器底座可直接复用的物体/关系
- 物体：
  - `block`：方块/滑块/木板
  - `ball`：小球/质点
  - `ground`：地面
  - `slope`：斜面
  - `anchor`：固定锚点
  - `conveyor`：传送带
  - `hemisphere` / `half-sphere` / `groove`：半球、半球壳、凹槽
  - `pulley-mount`：滑轮座
- 约束：
  - `rope`：绳
  - `rod`：杆
  - `spring`：弹簧
  - `pulley`：滑轮
- 外力：
  - 用户外力用 `force`
  - 重力、支持力、摩擦力、张力、弹簧力由系统计算

### 2.3 T3 模板统一构建原则
- 命令顺序统一按：`setGravity -> addBody -> snapTo -> addJoint -> addForce`。
- 支撑体 `slope/conveyor/hemisphere/groove` 必须通过 `snapTo -> ground` 贴地，不手填估值坐标。
- 悬挂链路优先让执行器按锚点自动推导 `rope/rod` 长度，避免手写长度与几何不一致。
- 除模板显式声明外，动态体初速度默认 `0`。
- 除模板显式声明外，`block/ball` 采用默认尺寸基线。
- 悬挂模板保持可读性基线：
  - `anchor.y > 9`
  - 静止悬挂链路 `d > 4`
- 每个模板必须能解释三个问题：
  - 场景里的物体是谁
  - 它们如何接触/连接/约束
  - 这个结构要服务哪个教学结论

## 3. 模块化整理

## 3.1 P-01 受力分析

### 模块目标
- 面向“受力图 + 条件变化 + 正交分解”。
- 构建重点不是轨迹复杂性，而是让系统能稳定给出重力、支持力、摩擦力、张力、弹力等受力结果。

### 模块共性构建原则
- 尽量让单个主体物体处于“受力清晰、箭头不拥挤”的状态。
- 对同一模板中的不同条件组合，优先复用同一几何场景，只改摩擦、外力、初速度、角度等参数。
- 若教学重点是临界条件，模板初始状态应落在“刚好平衡/刚好通过/刚好共速”等辨识度高的位置。

### FM-001 水平面静止
- 场景用途：展示水平面上物体在不同摩擦与外力条件下的受力。
- 建议物体：`ground + block`
- 关系/约束：
  - `block snapTo ground`
  - 无连接件
  - 可选用户外力作用于 `block`
- 关键教学关系：
  - 支持力与重力平衡
  - 摩擦力是否出现、方向如何变化
  - 斜向力会改变支持力大小
- 教学目的：
  - 区分静摩擦与滑动摩擦
  - 讲清 `N` 与 `mg` 不总是相等
  - 讲“恰好滑动”与受力平衡
- 构建提示：
  - 默认用一个 `block` 放在 `ground` 上
  - 通过切换 `friction` 和 `force.direction` 生成多个教学子场景
  - 这是 P-01 的基础母模板，应尽量参数化复用

### FM-002 水平面运动
- 场景用途：展示水平面运动过程中的受力与运动方向关系。
- 建议物体：`ground + block`
- 关系/约束：
  - `block snapTo ground`
  - 无连接件
  - 初速度为主要变量，可选外力
- 关键教学关系：
  - 摩擦力方向总与相对运动趋势相反
  - 受力与速度方向可以不同
- 教学目的：
  - 讲减速到停止、先减速后反向加速等典型情景
  - 让学生区分“有速度”和“有合力”
- 构建提示：
  - 将 `initialVelocity.x` 暴露为主参数
  - 若要做“恒力向左/向右”变体，可保留同一几何结构，仅切换外力

### FM-011 斜面静止/运动
- 场景用途：展示斜面上的受力分析与正交分解。
- 建议物体：`ground + slope + block`
- 关系/约束：
  - `slope snapTo ground`
  - `block snapTo slope`
  - 无连接件，可选沿斜面外力
- 关键教学关系：
  - 重力分解为沿斜面与垂直斜面分量
  - 摩擦力沿斜面、方向依运动趋势变化
  - 支持力垂直斜面
- 教学目的：
  - 训练坐标系选择
  - 讲 `tanθ = μ` 等临界条件
  - 讲斜面角度、摩擦与运动状态关系
- 构建提示：
  - `slope` 和 `block` 的贴合必须交给 `snapTo`
  - 若需要“上推/下滑/匀速”多个子情景，只改参数，不改结构

### FM-021 单绳悬挂
- 场景用途：最基础的悬挂受力模型。
- 建议物体：`anchor + ball`
- 关系/约束：
  - `anchor ->rope-> ball`
  - `ball` 竖直悬挂，初态静止
- 关键教学关系：
  - 张力沿绳方向
  - 平衡时 `T = mg`
- 教学目的：
  - 建立张力概念
  - 让学生理解“绳只能拉不能推”
- 构建提示：
  - 保证 `anchor.y > 9`，链路足够长，避免力箭头拥挤
  - 作为所有悬挂模型的基线模板

### FM-022 双绳悬挂
- 场景用途：展示二维平衡中两个张力分担重力。
- 建议物体：`anchor(left) + anchor(right) + ball`
- 关系/约束：
  - `leftAnchor ->rope-> ball`
  - `rightAnchor ->rope-> ball`
  - 两绳夹角不同
- 关键教学关系：
  - 水平分量相互抵消
  - 竖直分量合成平衡重力
- 教学目的：
  - 讲二维受力平衡
  - 讲力的分解与合成
- 构建提示：
  - 两个锚点要拉开距离，避免双绳几乎重合
  - 这是验证双张力箭头避让的重点模板

### FM-023 绳 + 杆悬挂
- 场景用途：对比绳和杆的力学语义差异。
- 建议物体：`anchor(left) + anchor(right) + ball`
- 关系/约束：
  - 一侧 `rope`
  - 一侧 `rod`
  - 共同连接到同一个悬挂体
- 关键教学关系：
  - 绳只提供拉力
  - 杆既可提供拉力也可提供推力
- 教学目的：
  - 建立“连接件性质不同，受力图不同”的认识
- 构建提示：
  - 几何布局要明显体现两种连接件方向不同
  - 适合做与 `FM-022` 的对照模板

### FM-031 绳连水平面两物体
- 场景用途：整体法与隔离法的经典两物体模型。
- 建议物体：`ground + block(m1) + block(m2)`
- 关系/约束：
  - 两个 `block` 都 `snapTo ground`
  - `block1 ->rope-> block2`
  - 外力作用于其中一个物体
- 关键教学关系：
  - 两物体共享同一加速度
  - 张力是内部作用力
- 教学目的：
  - 讲整体法/隔离法
  - 讲系统内力与外力区分
- 构建提示：
  - 两个物体水平排布，中间留出清晰绳段
  - 适合参数化 `m1/m2/F/μ`

### FM-032 绳连斜面-悬挂
- 场景用途：斜面体与悬挂体联动系统。
- 建议物体：`ground + slope + block(on slope) + anchor/pulley-mount + ball(hanging)`
- 关系/约束：
  - `slope snapTo ground`
  - `block snapTo slope`
  - 斜面体经 `rope` 跨过顶部连接到悬挂体
- 关键教学关系：
  - 两物体同一绳长约束、加速度大小相同
  - 一个物体的重力分量与另一个物体重力竞争
- 教学目的：
  - 讲连接体系统受力分析
  - 讲斜面分解和悬挂张力联立
- 构建提示：
  - 顶部建议用 `anchor` 或 `pulley-mount` 提示转折点
  - 重点保证绳几何自然，不出现穿模

### FM-033 弹簧连接
- 场景用途：展示弹簧形变量与弹力关系。
- 建议物体：`block + block` 或 `anchor + block + spring`
- 关系/约束：
  - 主关系是 `spring`
  - 可做水平双物体，也可做单端固定
- 关键教学关系：
  - 弹力大小与形变量相关
  - 弹力方向总指向恢复方向
- 教学目的：
  - 讲胡克定律
  - 为后续简谐运动模块做铺垫
- 构建提示：
  - 若重点是“受力”而非“振动”，初始形变量要明显但不过大
  - 可以显式允许非零初速度，用于展示瞬时受力变化

### FM-043 竖直圆周（绳连）
- 场景用途：讲绳模型竖直圆周中的张力与临界速度。
- 建议物体：`anchor + ball`
- 关系/约束：
  - `anchor ->rope-> ball`
  - `ball` 以最低点初速度进入圆周运动
- 关键教学关系：
  - 最高点 `T + mg = mv²/r`
  - 临界时最高点张力为零
- 教学目的：
  - 讲“恰好通过最高点”的临界条件
  - 讲绳模型只能拉不能推
- 构建提示：
  - 初速度应按 `sqrt(5gR)` 附近设置并留安全系数
  - 该模板必须显式允许非零初速度

### FM-044 竖直圆周（杆连）
- 场景用途：对比杆模型和绳模型在最高点受力上的差异。
- 建议物体：`anchor + ball`
- 关系/约束：
  - `anchor ->rod-> ball`
  - `ball` 以最低点初速度进入圆周运动
- 关键教学关系：
  - 杆可拉可推
  - 最高点既可能受拉也可能受推
- 教学目的：
  - 讲杆与绳的本质差异
  - 讲圆周运动中向心力方程不是“某个单独力”
- 构建提示：
  - 与 `FM-043` 共用半径与布局最合适，方便做对照
  - 初速度可设在 `sqrt(gr)` 上下两侧，突出受力差异

### FM-051 弹簧模型
- 场景用途：展示弹簧悬挂或水平弹簧中的受力。
- 建议物体：`anchor + block/ball`
- 关系/约束：
  - `anchor ->spring-> body`
  - 方向可为竖直或水平
- 关键教学关系：
  - 平衡时弹力与其他力平衡
  - 竖直时平衡位置不在自然长度
- 教学目的：
  - 讲 `kx` 与平衡位置
  - 为 `P-05/P-14` 弹簧模板建立先验
- 构建提示：
  - 若做竖直版，重心和锚点之间要留出足够伸长空间
  - 若做水平版，可直接用地面参考线辅助观察位移

### FM-052 杆模型
- 场景用途：单独强调轻杆可推可拉。
- 建议物体：`anchor + ball/block`
- 关系/约束：
  - `anchor ->rod-> body`
- 关键教学关系：
  - 杆能施加沿杆方向的拉力或推力
- 教学目的：
  - 和绳模型形成概念对比
  - 服务于后续圆周、连接体问题
- 构建提示：
  - 初始状态可取静止倾斜位置，便于画出杆力方向
  - 尽量让杆方向不要与重力共线

### FM-053 半球模型
- 场景用途：展示球面约束下的重力分解与正压力变化。
- 建议物体：`ground + hemisphere + ball`
- 关系/约束：
  - `hemisphere snapTo ground`
  - `ball snapTo hemisphere`
- 关键教学关系：
  - 正压力沿法线方向
  - 随位置变化，重力分解和法向约束变化
- 教学目的：
  - 讲曲面约束中的受力分析
  - 讲“将要滑落”临界
- 构建提示：
  - 球应放在半球顶部或接近顶部的辨识位置
  - 适合配合非零初速度观察脱离趋势

### FM-054 传送带模型
- 场景用途：讲相对运动决定摩擦方向。
- 建议物体：`ground + conveyor + block`
- 关系/约束：
  - `conveyor snapTo ground`
  - `block snapTo conveyor`
  - 无连接件
- 关键教学关系：
  - `v0 < v_belt`、`v0 > v_belt`、`v0 = v_belt` 三种情形摩擦不同
- 教学目的：
  - 讲摩擦方向由相对滑动趋势决定
  - 讲达到共速后的受力变化
- 构建提示：
  - 传送带速度要作为显式参数
  - 初始速度建议分三档预设，便于教师切换演示

### FM-055 凹槽模型
- 场景用途：展示凹槽内双支持力作用。
- 建议物体：`ground + groove + ball`
- 关系/约束：
  - `groove snapTo ground`
  - `ball snapTo groove`
- 关键教学关系：
  - 两侧支持力同时存在
  - 可借助对称性分析合力
- 教学目的：
  - 讲非单接触面的受力分解
  - 讲对称性在受力分析中的作用
- 构建提示：
  - 球体默认放在凹槽中心位置最利于对称讲解
  - 若偏置放置，可进一步展示两侧支持力不等

## 3.2 P-02 运动模拟

### 模块目标
- 面向“轨迹、速度、加速度、图像联动”。
- 同一几何场景下，重点是初速度、加速度、重力、摩擦等参数驱动的运动变化。

### 模块共性构建原则
- 需要确保运动方向和图表趋势一致，避免“图像对了、场景不直观”。
- 尽量让轨迹主现象明显，如抛物线、直线匀速、斜面往返、竖直圆周。
- 关键量应可直接从当前底座图表读取：`x-t / v-t / energy / momentum`。

### MOT-001 匀速直线运动
- 场景用途：最基础的匀速运动模板。
- 建议物体：`ground + block`
- 关系/约束：
  - `block snapTo ground`
  - 无连接件、无外力
  - 非零初速度，摩擦为零
- 关键教学关系：
  - 位移随时间线性增长
  - 速度恒定、加速度为零
- 教学目的：
  - 建立 `x-t` 与 `v-t` 图像直观对应
- 构建提示：
  - 模板初速度必须显式设置
  - 场景中不要加入多余物体，保持单一观察对象

### MOT-002 匀加速直线运动（水平）
- 场景用途：展示恒定加速度下的直线运动图像。
- 建议物体：`ground + block`
- 关系/约束：
  - `block snapTo ground`
  - 通过恒定外力或等效恒加速度实现
- 关键教学关系：
  - `v-t` 为直线
  - `x-t` 为抛物线
- 教学目的：
  - 讲图像斜率与面积含义
  - 讲三大匀变速公式
- 构建提示：
  - 若底座更易表达外力，则用恒定 `force`
  - 若更偏展示运动结果，则保持几何简单，突出图表联动

### MOT-011 斜面匀加速下滑
- 场景用途：展示斜面上沿切向的恒加速运动。
- 建议物体：`ground + slope + block`
- 关系/约束：
  - `slope snapTo ground`
  - `block snapTo slope`
- 关键教学关系：
  - 加速度由 `g(sinθ - μcosθ)` 决定
  - 临界下滑条件 `tanθ > μ`
- 教学目的：
  - 讲斜面运动受力与运动学联动
- 构建提示：
  - 初速度可默认为 0
  - 斜面长度要足够，让学生看出速度逐渐增大

### MOT-012 斜面减速上滑后下滑
- 场景用途：展示同一斜面上上滑和下滑加速度不同。
- 建议物体：`ground + slope + block`
- 关系/约束：
  - `slope snapTo ground`
  - `block snapTo slope`
  - 非零初速度沿斜面向上
- 关键教学关系：
  - 上滑时摩擦力向下
  - 下滑时摩擦力向上
  - 两阶段加速度大小不同
- 教学目的：
  - 讲摩擦方向随运动趋势改变
- 构建提示：
  - 需要给 `block` 明确上滑初速度
  - 斜面长度和初速度要保证“先上滑后回落”完整发生

### MOT-021 标准平抛
- 场景用途：平抛运动基础模板。
- 建议物体：`wall/高台 + ball + ground`
- 关系/约束：
  - `ball` 从高台边缘水平抛出
  - 与 `ground` 构成落体参照
- 关键教学关系：
  - 水平匀速、竖直自由落体互相独立
  - 轨迹为抛物线
- 教学目的：
  - 讲分运动思想
  - 讲水平射程、落地速度方向
- 构建提示：
  - 用 `ball` 比 `block` 更符合质点感
  - 起点高度应足够，否则抛物线不明显

### MOT-022 斜面上平抛
- 场景用途：展示平抛物体落在斜面上的位置变化。
- 建议物体：`ground + slope + ball + 高台/发射点`
- 关系/约束：
  - `slope snapTo ground`
  - `ball` 从斜面上方或斜面边缘处水平抛出
- 关键教学关系：
  - 初速度改变导致落点沿斜面移动
- 教学目的：
  - 讲几何条件对平抛落点的影响
  - 适合做多组初速度对比
- 构建提示：
  - 斜面倾角与抛出点高度要一起调
  - 需要给出一个默认“恰好落在中段”的基线

### MOT-033 竖直圆周运动
- 场景用途：从运动学角度看竖直圆周中的速度变化与临界条件。
- 建议物体：`anchor + ball`
- 关系/约束：
  - `anchor ->rope/rod-> ball`
  - `ball` 由最低点非零初速度启动
- 关键教学关系：
  - 最低点、最高点、水平位置速度不同
  - 绳模型与轨道/杆模型条件不同
- 教学目的：
  - 讲临界速度
  - 讲圆周运动不同位置的动力学特征
- 构建提示：
  - 可优先用 `rope` 版本做 A 类模板
  - 初速度要按最小闭环条件以上设置

## 3.3 P-05 简谐运动

### 模块目标
- 面向“平衡位置、周期、位移/速度/加速度相位关系”。
- 核心是让振动现象稳定、周期清晰、图像可读。

### 模块共性构建原则
- 重点不是复杂接触，而是单自由度振动。
- 平衡位置、最高点、最低点必须视觉上可辨认。
- 所有模板都应能稳定输出至少一组趋势图。

### SHM-001 竖直弹簧振子
- 场景用途：展示重力参与下的竖直简谐运动。
- 建议物体：`anchor + block/ball`
- 关系/约束：
  - `anchor ->spring-> body`
  - `body` 竖直方向自由振动
- 关键教学关系：
  - 平衡位置相对自然长度下移
  - 位移、速度、加速度相位关系清晰
- 教学目的：
  - 讲“平衡位置不等于自然长度”
  - 讲位移对称与加速度对称
- 构建提示：
  - 竖直空间要留足，避免一开始就碰地
  - 初始位移应小到保持教学近似，又大到能看清振动

### SHM-002 水平弹簧振子
- 场景用途：展示最纯粹的弹簧简谐运动和能量交换。
- 建议物体：`anchor + block`
- 关系/约束：
  - `anchor ->spring-> block`
  - `block` 沿水平方向往复
- 关键教学关系：
  - 平衡位置就是自然长度位置
  - 动能与弹性势能周期性交换
- 教学目的：
  - 讲 `ω = sqrt(k/m)` 与总机械能守恒
- 构建提示：
  - 水平场景最适合叠加 `x-t / v-t / a-t` 图像
  - 保持地面或参考线水平，强化位移感

### SHM-003 简单单摆
- 场景用途：展示小角度单摆的简谐近似。
- 建议物体：`anchor + ball`
- 关系/约束：
  - `anchor ->rope-> ball`
  - `ball` 以初始摆角偏离平衡位置
- 关键教学关系：
  - 周期主要由 `L` 和 `g` 决定
  - 小角度近似下接近简谐运动
- 教学目的：
  - 讲等时性近似
  - 讲单摆测重力加速度
- 构建提示：
  - 默认摆角宜小于 10°
  - 若需要展示偏离近似，可再加大角度变体

## 3.4 P-12 动量定理及动量守恒

### 模块目标
- 面向“碰撞、相互作用、系统动量守恒”。
- 构建重点是保证碰撞真实发生、碰前碰后量可观察、参数可解释。

### 模块共性构建原则
- 初速度、质量、恢复系数是第一优先级参数。
- 物体初始间距要既不会重叠，也不能太远导致课堂节奏拖沓。
- 需要尽量让碰前和碰后速度变化明显。

### MOM-001 完全弹性碰撞
- 场景用途：展示一维完全弹性碰撞。
- 建议物体：`ground + block/ball(m1) + block/ball(m2)`
- 关系/约束：
  - 两物体共线运动
  - 无连接件
  - 恢复系数 `e = 1`
- 关键教学关系：
  - 总动量守恒
  - 总动能也守恒
- 教学目的：
  - 讲碰后速度公式
  - 讲等质量交换速度
- 构建提示：
  - 推荐两球或两个低摩擦滑块
  - 地面摩擦尽量关掉，避免引入额外耗散

### MOM-003 部分弹性碰撞
- 场景用途：展示恢复系数变化对碰撞结果的影响。
- 建议物体：`ground + body1 + body2`
- 关系/约束：
  - 一维对碰或追碰
  - 恢复系数 `0 < e < 1`
- 关键教学关系：
  - 动量守恒但动能部分损失
  - `e` 越小，耗散越明显
- 教学目的：
  - 讲恢复系数概念
  - 讲弹性碰撞与非弹性碰撞连续过渡
- 构建提示：
  - 可基于 `MOM-001` 只改 `restitution`
  - 适合做同质量对比模板

### MOM-011 爬坡模型
- 场景用途：展示“碰撞类过程”中的系统动量和机械能转化。
- 建议物体：`ground + slope(wedge) + block`
- 关系/约束：
  - `slope` 放在地面上，可视为大质量斜面
  - `block` 冲上斜面
- 关键教学关系：
  - 无摩擦地面时系统水平方向动量守恒
  - 动能转化为重力势能
- 教学目的：
  - 讲非典型碰撞中的守恒思想
- 构建提示：
  - 若当前底座难以精确表示“斜面可动”，可先做近似静止斜面教学版
  - 重点是“到达最高点共速”的教学语义

### MOM-012 弹簧压缩释放
- 场景用途：展示内力作用下的反向弹开与动量守恒。
- 建议物体：`ground + block1 + block2 + spring`
- 关系/约束：
  - `block1 <->spring<-> block2`
  - 初始状态弹簧压缩
- 关键教学关系：
  - 系统初始总动量为零
  - 释放后两物体反向分离，动量大小相等方向相反
- 教学目的：
  - 讲内力不能改变系统总动量
  - 讲质量越大速度越小
- 构建提示：
  - 两物体质量差要明显，演示效果更好
  - 初始压缩量不要过大，避免数值不稳

### MOM-013 滑块模型
- 场景用途：展示摩擦参与下的平台碰撞/滑行问题。
- 建议物体：`ground + block1 + block2`
- 关系/约束：
  - 两物体在平台上运动与碰撞
  - 地面或平台有摩擦
- 关键教学关系：
  - 碰撞本身满足动量关系
  - 后续滑行受摩擦耗散影响
- 教学目的：
  - 讲“碰撞阶段”和“碰后滑行阶段”分段分析
- 构建提示：
  - 推荐将碰撞和摩擦滑行放在同一可视区域
  - 需要清楚区分碰撞前后速度和最终停下位置

### MOM-031 人在船上行走
- 场景用途：经典内力系统位移模型。
- 建议物体：`ground(水平参照) + conveyor/长平台充当船 + block/person`
- 关系/约束：
  - 人相对船移动
  - 船相对地面反向位移
- 关键教学关系：
  - 水平无外力时系统动量守恒
  - 质心位置不变
- 教学目的：
  - 讲相对位移和对地位移区别
- 构建提示：
  - 当前底座可将“人”抽象为小方块，将“船”抽象为大平板
  - 更适合做低速演示，强调位移关系而非碰撞瞬间

### MOM-032 凹槽滑块模型
- 场景用途：展示受约束系统中的动量问题。
- 建议物体：`ground + groove + block/ball`
- 关系/约束：
  - `groove snapTo ground`
  - 物体在槽内运动
- 关键教学关系：
  - 无摩擦时可讨论系统动量守恒
  - 有摩擦时需引入摩擦冲量
- 教学目的：
  - 讲受约束运动中的系统分析
- 构建提示：
  - 槽体几何要足够明显，避免误读成普通斜面
  - 更适合作为“特殊系统”案例模板

## 3.5 P-14 机械能守恒

### 模块目标
- 面向“动能、重力势能、弹性势能、摩擦热”的转化。
- 构建重点是让场景现象与能量条变化相互印证。

### 模块共性构建原则
- 应优先选择势能变化明显的位置差、伸缩量差。
- 若涉及摩擦，必须让 `Q` 的增长在演示中有直观价值。
- 同一类模板优先做“无摩擦基线版 + 有摩擦对比版”。

### ENE-001 自由落体
- 场景用途：最基础的重力势能转动能模型。
- 建议物体：`ball + ground`
- 关系/约束：
  - 单球自由下落
  - 与地面构成势能零点参照
- 关键教学关系：
  - `Ep` 递减，`Ek` 递增，总机械能守恒
- 教学目的：
  - 讲最简单的机械能守恒
- 构建提示：
  - 初始高度要足够高，能量条变化才明显
  - 推荐用 `ball` 保持“质点”直觉

### ENE-002 竖直上抛
- 场景用途：展示上抛和回落全过程中的能量变化。
- 建议物体：`ball + ground`
- 关系/约束：
  - 单球竖直向上初速度
- 关键教学关系：
  - 上升时 `Ek -> Ep`
  - 下降时 `Ep -> Ek`
- 教学目的：
  - 讲最高点 `Ek = 0`
  - 讲任意高度机械能守恒
- 构建提示：
  - 必须显式设置竖直初速度
  - 高度边界要足够，不要一开始就出框

### ENE-011 阿特伍德机
- 场景用途：双物体与定滑轮系统中的机械能守恒。
- 建议物体：`pulley-mount + ball/block(m1) + ball/block(m2)`
- 关系/约束：
  - `m1 <-rope via pulley-> m2`
  - 两物体一上一下运动
- 关键教学关系：
  - 较重物体势能减少
  - 系统动能增加
- 教学目的：
  - 讲多物体系统中的能量转化
- 构建提示：
  - 两质量差要适中，既能动起来又不过快
  - 需保证绳长和初始高度差清晰

### ENE-021 光滑斜面
- 场景用途：展示斜面下滑中的势能转动能。
- 建议物体：`ground + slope + block`
- 关系/约束：
  - `slope snapTo ground`
  - `block snapTo slope`
- 关键教学关系：
  - 光滑条件下机械能守恒
- 教学目的：
  - 讲高度决定末速度
- 构建提示：
  - 与 `ENE-022` 共用同一几何结构最利于对比
  - 只需切换摩擦参数即可

### ENE-022 有摩擦斜面
- 场景用途：展示机械能不守恒但总能量守恒。
- 建议物体：`ground + slope + block`
- 关系/约束：
  - `slope snapTo ground`
  - `block snapTo slope`
  - 存在动摩擦
- 关键教学关系：
  - 机械能减少，摩擦热 `Q` 增加
  - `E总末 + Q = E总初`
- 教学目的：
  - 讲非保守力做功
  - 讲机械能守恒的适用条件
- 构建提示：
  - 摩擦系数不要太大，否则物体几乎不动
  - 需要能量条里显式显示 `Q`

### ENE-041 弹簧振子能量转化
- 场景用途：展示弹性势能与动能的周期性交换。
- 建议物体：`anchor + block + spring`
- 关系/约束：
  - `anchor ->spring-> block`
  - 初始压缩或拉伸
- 关键教学关系：
  - 最大压缩/伸长处 `Ek = 0`
  - 平衡位置 `Ek` 最大
  - 有阻尼时总机械能衰减
- 教学目的：
  - 讲弹簧系统能量转化
  - 讲阻尼导致的能量耗散
- 构建提示：
  - 无阻尼版应作为默认基线
  - 阻尼版可作为同模板参数切换

### ENE-051 单摆能量转化
- 场景用途：展示单摆在最高点和最低点之间的能量交换。
- 建议物体：`anchor + ball`
- 关系/约束：
  - `anchor ->rope-> ball`
  - 由初始摆角释放
- 关键教学关系：
  - 两端 `Ep` 最大
  - 最低点 `Ek` 最大
- 教学目的：
  - 讲摆长、摆角与最低点速度关系
- 构建提示：
  - 默认摆角不宜过大，先强调基本能量转化
  - 可与 `SHM-003` 共用结构，不同处在于展示重点转向能量条
