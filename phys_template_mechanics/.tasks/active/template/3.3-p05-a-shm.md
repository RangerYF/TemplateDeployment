3. 在 `src/templates/catalog-data/templates.json` 配置 `ready` 并接入 `sceneJsonPath + sceneSource`。
4. 用 `loadTemplateById` 执行加载与 sanity 校验。
5. 在编辑器中运行仿真验证周期性表现与可编辑性。
