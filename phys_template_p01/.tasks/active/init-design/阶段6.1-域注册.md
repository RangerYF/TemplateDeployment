# 阶段6.1：域注册（实体 + 求解器）

> 父任务：阶段6-参考实现与端到端打通

## 目标

在 `src/domains/mechanics/` 中注册 block（物块）和 surface（水平面）两个实体类型，并实现 block-on-surface 静力平衡求解器。完成后，注册表查询应返回正确结果，求解器匹配应成功。

---

## 任务清单

### T1：注册实体类型 block（物块）

**创建文件**：`src/domains/mechanics/entities/block.ts`

**实现要点**：
- 调用 `entityRegistry.register()` 注册 block 实体
- type: `'block'`，category: `'object'`，label: `'物块'`
- defaultProperties: `{ mass: 1, width: 0.5, height: 0.5 }`
- paramSchemas: 质量滑块（0.1~10，step 0.1，unit kg）
- hitTest: 使用 `pointInRect`（从 `core/physics/geometry` 导入），根据 entity.transform 和 properties.width/height 计算矩形范围
- createEntity: 生成唯一 EntityId（`block-${crypto.randomUUID().slice(0,8)}`），合并默认属性

**注意**：
- hitTest 接收的 point 是物理坐标，需要根据 entity.transform.position 和 width/height 构造 Rect
- block 底边中心在 transform.position，所以矩形范围是 `{ x: pos.x - width/2, y: pos.y, width, height }`（Y轴向上）

### T2：注册实体类型 surface（水平面）

**创建文件**：`src/domains/mechanics/entities/surface.ts`

**实现要点**：
- 调用 `entityRegistry.register()` 注册 surface 实体
- type: `'surface'`，category: `'surface'`，label: `'水平面'`
- defaultProperties: `{ length: 6, friction: 0 }`
- paramSchemas: `[]`（水平面无用户可调参数）
- hitTest: 使用 `pointOnLine`（从 `core/physics/geometry` 导入），判断点是否在水平线附近
- createEntity: 生成唯一 EntityId

**注意**：
- 水平面从 transform.position 开始，向右延伸 length 长度
- hitTest 的线段：`from = position`, `to = { x: position.x + length, y: position.y }`

### T3：实现求解器 block-on-surface

**创建文件**：`src/domains/mechanics/solvers/block-on-surface.ts`

**实现要点**：
- 调用 `solverRegistry.register()` 注册求解器
- id: `'mech-block-on-horizontal-surface'`
- label: `'水平面物块受力'`
- pattern: `{ entityTypes: ['block', 'surface'], relationType: 'contact', qualifier: { surface: 'horizontal' } }`
- solveMode: `'analytical'`
- solve 函数（SolverFunction 签名）：
  1. 从 scene.entities 中找到 block 实体，读取 mass
  2. 计算 G = m × 9.8，方向 `{ x: 0, y: -1 }`
  3. 计算 N = m × 9.8，方向 `{ x: 0, y: 1 }`
  4. 构造 ForceAnalysis: forces = [G, N], resultant = { magnitude: 0, direction: {x:0,y:0} }
  5. 构造 MotionState: velocity = {0,0}, acceleration = {0,0}
  6. 返回 PhysicsResult

**注意**：
- 静力平衡：任何时刻 t 的结果相同，step(dt) 每次返回相同 PhysicsResult
- Force.type 使用 `'gravity'` 和 `'normal'`
- Force.label 使用 `'G'` 和 `'N'`
- ForceAnalysis.resultant 的 magnitude 为 0（合力为零）
