## 修改清单

### 1. §1 `category` 字段定义
- 旧：`${domain}/${model}`（如 `mechanics/slope`）
- 新：PRD 模块 ID（如 `P-01`）

### 2. §1 `id` 字段定义
- 旧：`${domain}-${model}-${variant}`（如 `mech-slope-sliding-down`）
- 新：`${模块}-${模型ID}-${变体}`（如 `P01-FM002-sliding-down`）

### 3. §2 分类体系设计
- 完全重写 §2.1 分类路径规范 → 改用 PRD 模块
- §2.2 搜索标签规范 → 已删除（非 PRD 需求）
- §2.3 教材章节编码 → 已删除（非 PRD 需求）
- §2.4 预设库浏览数据结构 → 重写为按 PRD 模块组织

### 4. §6 三个预设示例的 id 和 category
- 斜面受力 → category: `P-01`, id: `P01-FM002-sliding-down`
- 平抛运动 → category: `P-02`, id: `P02-MOT003-basic`
- 洛伦兹力 → category: `P-08`, id: `P08-EMF007-circular`

### 5. §7 预设 JSON 模板
- 更新 id 和 category 的模板格式

### 6. §8 总结
- 更新 category 格式说明
