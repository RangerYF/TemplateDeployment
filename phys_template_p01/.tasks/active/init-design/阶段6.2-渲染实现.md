# 阶段6.2：渲染实现（实体渲染器 + 受力视角）

> 父任务：阶段6-参考实现与端到端打通
> 风险等级：L1
> 流程路径：MODE 3 → MODE 4 → MODE 5 → MODE 6

---

## 目标

实现 block 和 surface 的 Canvas 渲染器，以及受力视角（force viewport）的力箭头渲染器。完成后，Canvas 应能绘制物块、水平面和力箭头。

---

## 任务清单

### T4：实现实体渲染器

**创建文件**：
- `src/domains/mechanics/renderers/block-renderer.ts`
- `src/domains/mechanics/renderers/surface-renderer.ts`

#### block-renderer

- 注册：`rendererRegistry.registerEntity({ entityType: 'block', renderer, layer: 'object' })`
- 渲染器签名：`(entity: Entity, result: PhysicsResult | null, ctx: RenderContext) => void`
- 绘制内容：
  1. 从 entity.properties 读取 width/height，从 entity.transform.position 读取位置
  2. 使用 `worldToScreen` 将物理坐标转为画布坐标
  3. 使用 `worldLengthToScreen` 将物理尺寸转为像素尺寸
  4. 绘制填充矩形（浅蓝色填充 `#EBF8FF` + 深蓝色边框 `#2B6CB0`）
  5. 矩形应有圆角（rx=4）和轻微阴影
  6. 在矩形中心显示 label（如果有）

**注意**：
- 物理坐标 Y 轴向上，画布 Y 轴向下，`worldToScreen` 已处理翻转
- block 的 transform.position 是底边中心，画布绘制时需要计算左上角

#### surface-renderer

- 注册：`rendererRegistry.registerEntity({ entityType: 'surface', renderer, layer: 'surface' })`
- 绘制内容：
  1. 从 entity.properties 读取 length，从 entity.transform.position 读取起点
  2. 使用 `worldToScreen` 转换起点和终点坐标
  3. 绘制一条实线（深灰色 `#4A5568`，lineWidth=2）
  4. 在线条下方绘制斜线填充（标准物理教材地面画法）
    - 斜线间距约 8px，角度约 -45°
    - 斜线高度约 12px
    - 颜色 `#A0AEC0`

### T5：实现受力视角渲染器

**创建文件**：`src/domains/mechanics/viewports/force-viewport.ts`

- 注册：`rendererRegistry.registerViewport('force', forceViewportRenderer)`
- 渲染器签名：`(data: ViewportData, entities: Map<EntityId, Entity>, ctx: RenderContext) => void`
- 实现：
  1. 确认 `data.type === 'force'`，读取 `data.data.analyses`
  2. 遍历每个 ForceAnalysis：
    - 获取对应实体的位置作为力箭头起点（实体中心）
    - 每个 Force → 计算箭头终点：起点 + direction × forceToLength(magnitude)
    - 调用 `drawArrow(ctx, from, to, style)` 绘制力箭头
    - 在箭头中点偏移位置调用 `drawTextLabel` 绘制数值标注
  3. 合力不为零时，绘制合力箭头（虚线，绿色）

**力箭头长度映射函数 `forceToLength`**：
```
baseLength = 60px（对应 10N）
length = magnitude / 10 * baseLength
clamp(length, 30, 200)  // 确保小力可见、大力不溢出
```

**颜色约定**：
- gravity（重力）：`#E53E3E`（红色）
- normal（支持力）：`#3182CE`（蓝色）
- friction（摩擦力）：`#DD6B20`（橙色）
- resultant（合力）：`#38A169`（绿色，虚线）

**标注格式**（standard 密度）：
- `"G=19.6N"`
- `"N=19.6N"`

**标注位置**：
- 箭头中点，向右偏移 12px，避免压在箭头上
