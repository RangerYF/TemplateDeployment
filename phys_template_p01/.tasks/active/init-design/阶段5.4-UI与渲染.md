# 阶段5.4：UI 骨架 + 渲染 Primitives

> 批次4/6 | 依赖：5.2 + 5.3已完成 | 包含：T8 shell/UI骨架 + T9 renderer/primitives

---

## 上下文

5.2已完成类型系统和设计系统，5.3已完成注册表、数学工具和 Store。本批次搭建用户可见的 UI 骨架（三栏布局）和 Canvas 渲染基础设施。

## 实施任务

### T8: shell/ UI 骨架

**需要创建的文件**：
- `src/shell/App.tsx` — 顶层布局组件（三栏 + 底部时间轴）
- `src/shell/layout/MainLayout.tsx` — 三栏布局容器
- `src/shell/panels/ParamPanel.tsx` — 左侧参数面板框架（schema 驱动渲染的骨架）
- `src/shell/panels/InfoPanel.tsx` — 右侧信息面板框架（视角区域容器）
- `src/shell/canvas/CanvasContainer.tsx` — 中央画布容器（管理 Canvas 元素和 resize）
- `src/shell/timeline/TimelineBar.tsx` — 底部时间轴控制栏（播放/暂停/重置按钮 + 进度条）
- `src/shell/index.ts` — 统一导出

**布局规格**（参照产品方案 3.3 页面布局）：
- 左面板：240-280px 固定宽度，参数面板
- 中画布：flex 自适应，Canvas 元素
- 右面板：280-320px 固定宽度，信息/视角面板
- 底时间轴：全宽，固定高度 48-56px

**设计关注点**：
- [x] 使用 design_guid 迁移的 UI 组件（Button, Slider, Input, Switch, Select, Label）
- [x] ParamPanel 的 schema 驱动渲染：接收 `ParamSchema[]` 和 `ParamValues`，根据 schema 的 discriminated union 类型自动选择渲染控件（Slider/Input/Toggle/Select），支持按 group 分组
- [x] CanvasContainer 需要管理 Canvas ref、处理 resize（ResizeObserver + devicePixelRatio + ctx.scale）、暴露 CanvasRenderingContext2D
- [x] TimelineBar：播放/暂停/重置三个按钮 + 时间显示 + 进度条，按钮状态由 SimulationStatus 驱动
- [x] 所有面板组件阶段5只需骨架（空框架 + 占位内容），阶段6填充真实数据

### T9: renderer/ 基础绘制 primitives

**需要创建的文件**：
- `src/renderer/primitives/arrow.ts` — 箭头绘制（力向量箭头、速度箭头等）
- `src/renderer/primitives/text-label.ts` — 文字标注（力标签、数值标签）
- `src/renderer/primitives/trail.ts` — 轨迹线绘制
- `src/renderer/primitives/shapes.ts` — 基础形状（矩形、圆形、线段）
- `src/renderer/primitives/index.ts` — 统一导出
- `src/renderer/render-loop.ts` — 渲染循环管理（requestAnimationFrame + RenderLayer 排序）
- `src/renderer/coordinate.ts` — 物理坐标 ↔ 画布像素坐标转换
- `src/renderer/index.ts` — 统一导出

**箭头绘制 API**（关键 primitive）：
- drawArrow(ctx, from, to, style) — 绘制带箭头的向量
- style 包含：颜色、线宽、箭头大小、是否虚线

**渲染循环骨架**：
- 按 RenderLayer 枚举顺序（background → surface → field → object → connector → overlay）调用对应渲染器
- 主视角不透明，叠加视角 globalAlpha = 0.3
- 选中高亮由渲染循环统一处理（描边/光晕）

**设计关注点**：
- [x] 所有绘制函数接收 `CanvasRenderingContext2D` 作为第一参数（对齐 RenderContext）
- [x] 坐标转换：worldToScreen / screenToWorld，处理 Y 轴翻转（Canvas Y 向下，物理 Y 向上）+ worldLengthToScreen 标量转换
- [x] 箭头缩放：drawArrow 支持 arrowHeadSize 参数，长度由调用方计算
- [x] 阶段5只实现 primitives 的基础 API，不实现具体物理实体的渲染器（留给阶段6域注册）
