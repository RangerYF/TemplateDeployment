# 阶段3：交互设计与功能点说明

> 状态：**方案已确认** — 2026-03-19 评审通过，进入实施阶段

## 1. 阶段目标

在阶段2已完成的 8 个预设（受力分析 + 视觉渲染）基础上，增加用户交互能力：

**一句话概括**：老师可以直接点击任意力箭头查看详情并进行正交分解，分解过程有动画，分量数值实时跟随参数变化。

## 2. 选中模型

画布上可交互的所有目标统一在一个池子里，**任何时刻只有一个选中目标**：

| 选中目标 | selection.type | 画布效果 | 附加 UI |
|---------|---------------|---------|--------|
| 无 | null | 无高亮 | 无 |
| 物块 | `'entity'` | 蓝色边框 | InfoPanel 显示物块信息 |
| 斜面 | `'entity'` | 三角形高亮 | InfoPanel 显示斜面信息 |
| 力箭头 G | `'force-arrow'` | G 加粗，其他力降透明度 | ForcePopover 弹出 |

选中力箭头就是选中力箭头，选中物块就是选中物块。**不存在两级选中**（不需要"先选中实体再选中力"）。

点击新目标自动替换旧目标，点击空白清除所有选中。

## 3. 用户操作流程（完整路径）

```
老师加载预设（如"斜面静止"）
  │
  ├─ 画面显示物块 + 斜面 + 所有力箭头（已有能力）
  │
  ├─ 🖱️ 鼠标悬停物块 → 物块边框轻微高亮 + cursor 变 pointer
  │
  ├─ 🖱️ 鼠标悬停力箭头 G → G 加粗发光 + cursor 变 pointer
  │
  ├─ 🖱️ 点击物块 → 物块选中（蓝色边框）
  │
  ├─ 🖱️ 点击力箭头 G → G 选中 + 弹出浮动信息面板
  │     │
  │     ├─ 显示：力名称 + 数值 + 方向描述
  │     ├─ 显示：「正交分解」勾选框（仅当力不沿坐标轴时）
  │     │
  │     ├─ ☑️ 勾选「正交分解」→ 0.8s 动画：
  │     │     - 坐标轴参考线淡入
  │     │     - 分量箭头从零长度生长到目标长度
  │     │     - 引导虚线 + 直角标记淡入
  │     │     - 分量标签淡入
  │     │
  │     ├─ 🎚️ 调整参数（如质量、角度）→ 分量数值实时更新（无重播动画）
  │     │
  │     └─ ☐ 取消勾选 → 0.3s 反向消失动画
  │
  ├─ 🖱️ 点击另一个力 N → 关闭 G 的面板，打开 N 的面板
  │
  ├─ 🖱️ 点击物块 → 关闭力面板，选中物块
  │
  └─ 🖱️ 点击空白 → 取消所有选中，关闭面板，清除分解
```

### F5：正交分解 — 勾选触发

| 属性 | 值 |

## 5. 浮动信息面板规格（ForcePopover）

### 5.1 面板布局

```
┌────────────────────────────┐
│  G = 19.6 N                │  ← 力名称 = 数值 + 单位
│  方向：竖直向下              │  ← 力方向的自然语言描述
│────────────────────────────│
│  ☐ 正交分解                 │  ← checkbox（仅力不沿轴时显示）
└────────────────────────────┘
```

### 5.2 分解选项显示逻辑

```
对于选中的力 F 和当前坐标系 (axis1, axis2)：
  - cosα = |F·axis1| / |F| （力与 axis1 的夹角余弦）
  - cosβ = |F·axis2| / |F| （力与 axis2 的夹角余弦）

  if cosα > cos(5°) ≈ 0.996 → 力沿 axis1，不显示分解选项
  if cosβ > cos(5°) ≈ 0.996 → 力沿 axis2，不显示分解选项
  otherwise → 显示分解选项
```

**典型案例**：

| 场景 | 力 | 坐标系 | 是否可分解 |
|------|-----|--------|-----------|
| 斜面 | G（竖直下） | 沿斜面/垂直斜面 | ✅ 可（与两轴都不平行） |
| 斜面 | N（垂直斜面） | 沿斜面/垂直斜面 | ❌ 不可（沿 axis2） |
| 斜面 | f（沿斜面） | 沿斜面/垂直斜面 | ❌ 不可（沿 axis1） |
| 水平面+外力 | F（斜向） | 水平/竖直 | ✅ 可（与两轴都不平行） |
| 水平面+外力 | G（竖直下） | 水平/竖直 | ❌ 不可（沿 axis2） |

### 5.3 方向描述生成

| 力方向 | 描述文本 |
|--------|---------|
| (0, -1) | 竖直向下 |
| (0, 1) | 竖直向上 |
| (1, 0) | 水平向右 |
| (-1, 0) | 水平向左 |
| (cosθ, -sinθ) 斜面法线方向 | 垂直斜面向上 |
| (-sinθ, -cosθ) 沿斜面向下 | 沿斜面向下 |
| 其他 | 方向角 XX° |

### 5.4 面板定位规则

1. 默认位置：力箭头终点右侧偏上 (tipX + 16, tipY - 8)
2. 超出画布右边界 → 移到终点左侧
3. 超出画布下边界 → 移到终点上方
4. 面板尺寸：宽 ≈ 220px，高 ≈ 自适应

## 6. 分解计算规格

### 6.1 通用投影公式

```
给定：
  F = (Fx, Fy)    — 力的分量
  axis1 = (a1x, a1y) — 第一个坐标轴单位向量
  axis2 = (a2x, a2y) — 第二个坐标轴单位向量

投影计算：
  comp1 = Fx * a1x + Fy * a1y    （力在 axis1 方向的分量，带符号）
  comp2 = Fx * a2x + Fy * a2y    （力在 axis2 方向的分量，带符号）

分量力：
  F1 = comp1 * axis1    （分量1向量）
  F2 = comp2 * axis2    （分量2向量）

验证：
  F1 + F2 应该 ≈ F（向量和还原原力）
```

### 6.2 手算验证用例

**用例1：斜面重力分解（θ=30°, m=2kg）**

```
G = (0, -19.6)    — 重力
axis1 = (cos30°, -sin30°) = (0.866, -0.5)    — 沿斜面向下
axis2 = (sin30°, cos30°) = (0.5, 0.866)      — 垂直斜面向上

comp1 = 0*0.866 + (-19.6)*(-0.5) = 9.8 N     ← mgsinθ ✓
comp2 = 0*0.5 + (-19.6)*0.866 = -16.97 N     ← -mgcosθ ✓（负号表示沿 axis2 反方向）
```

**用例2：水平外力分解（F=10N, θ=30°向右上）**

```
F = (10*cos30°, 10*sin30°) = (8.66, 5.0)
axis1 = (1, 0)    — 水平
axis2 = (0, 1)    — 竖直

comp1 = 8.66 N    ← Fcosθ ✓
comp2 = 5.0 N     ← Fsinθ ✓
```

### 清理策略

2. **替代**：分解渲染移入 `force-interaction-handler.renderOverlay()`，由用户交互触发

## 10. 不在阶段3范围内的功能

以下功能**有意不做**，避免 scope 膨胀：

| 排除项 | 原因 |
|--------|------|
| 实体拖拽移动 | Phase 2 搭建器功能 |
| 键盘快捷键（ESC 取消选中等） | 可选增强，不阻塞核心功能 |
| 多实体同时选中 | Phase 2 功能 |
| 力箭头拖拽改变方向/大小 | 与预设驱动理念冲突 |
| 合力的分解 | 合力本身是计算结果，分解无物理意义 |
| 水平面场景的分解 | G/N/f 都沿轴，无需分解（外力场景除外） |
| ParamPanel 选中联动 | 可选增强，不在核心路径上 |
| SelectionManager 模块 | Phase 1 单选场景，一个 `selection` 字段够用 |
| PropertyPanel 模块 | ParamPanel + ForcePopover 已覆盖需求，无需统一 schema |
