# 状态梳理：电路 / 电场与磁场可视化 / 电磁感应

更新时间：2026-04-06

说明：

- 本文以当前仓库代码为准，结合需求文档做差异对照。
- `已完成` 指代码里已能找到对应实体、求解器、预设、页面或交互入口。
- `待办` 指需求文档明确提出、但当前代码里未见完整落地，或只落了简化版。

## 一页结论

### 电路（P-04）

- 当前完成度最高。
- 已有预设电路、自由搭建、模板驱动搭建、专题页、通用求解器和多类专用求解器。
- 主要待办不在“有没有功能”，而在“模板体系是否扩展”“RC/故障等能力是否做完整”。

### 电场与磁场可视化（P-08）

- 主干能力已成型。
- 静电场、磁感线、带电粒子在电场/磁场/复合场中的运动都已有预设与求解器，独立场搭建器 MVP 也已接入。
- 当前剩余差异主要收缩到 builder 的保存/加载和模板扩展，而不是核心物理链路。

### 电磁感应（P-13）

- 当前完成度明显低于前两块。
- 已有一个“矩形线框穿过匀强磁场”的基础版预设和求解器。
- 需求里规划的楞次定律、单棒/双棒、含电源/电容、终态分析、自由组装，大部分还没落地。

- 需求文档里“自由搭建 + 五大实验模板 + 故障排查 + 动态 RC”这套全量 P-04 目标，代码已完成大半，但还不是完全版。

## 三、电磁感应（P-13）

### 已完成

- 已有电磁感应相关实体基础：
  - 匀强磁场
  - 矩形线框
- 已有一个感应求解器：
  - `wire-frame-induction`
- 已有一个感应预设：

- 已支持多视角展示：
  - `circuit`
  - `field`
  - `force`
  - `motion`
- 已能输出的核心量：
  - 磁通量
  - 感应电动势
  - 感应电流
  - 安培力标注
  - 运动轨迹

### 待办

- 当前只完成了非常基础的一支：
  - “矩形线框匀速穿过匀强磁场”
- 求解器仍是简化版：
  - 安培力只做标注，不反作用到速度
  - 线框默认匀速
  - 还不是完整的“力-电-运动耦合”模型
- 需求文档里的下列模型未见代码完整落地：
  - 磁棒-线圈楞次定律演示
  - 单棒基础模型
  - 单棒 + 含电源
  - 单棒 + 含电容
  - 双棒模型
  - 双棒 + 摩擦
  - 双棒 + 含电容
  - 竖直导轨单棒
- 需求里强调的终态分析也未完成：
  - 匀速终态
  - 静止终态
  - 含电容时 `I -> 0`
  - 含电源时 `v终 / I终`
- 需求里规划的“自由组装模式下的综合分析”当前未见对应 builder/专用入口。
- 当前预设分类仍是 `P-02`，未形成独立的 P-13 产品入口。

## 四、建议优先级

### 优先级 A

- 电磁感应补到“单棒基础模型”
- 电磁感应补上安培力反作用到速度的真正耦合
- 电路 builder 模板区补到 `2~3` 个高频模板，而不只是伏安法一个

### 优先级 B

- 电路模块补全 RC 动态链路
- 电路故障模拟做成统一用户入口，而不是部分元件支持
- P-08 补 `磁发散` 独立预设

### 优先级 C

- P-13 再扩到双棒、含电源、含电容
- P-08 再补更偏压轴题的跨域模型

## 五、当前判断

- 电路：可以视为当前电磁域里最成熟的模块。
- 电场与磁场可视化：已经具备教学演示主干，但还有少量模型缺口。
- 电磁感应：目前更像“起了一个基础样例”，离需求文档中的完整 P-13 还差一大截。
