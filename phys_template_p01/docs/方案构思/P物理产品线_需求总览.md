## P-01 受力分析器

### 产品价值
高中物理受力分析是核心考点，黑板画力的示意图无法动态调整，学生搞不清不同条件下受哪些力。受力分析器让老师选好模型和约束条件，系统自动完成受力分析并以清晰示意图展示。

### 目标用户
高中物理老师，无编程基础。

### 使用场景
| 场景 | 描述 |
|------|------|
| 课前 | 预设好"斜面有摩擦，物体静止"等条件，上课直接展示 |
| 课中 | 在投影仪上选择条件，让学生观察受力变化 |
| 讲题 | 动态演示力的分解，让学生直观理解正交分解 |

### 交互说明
1. **选择受力模型**：顶部模型按钮组，点击即选，右侧立即出现默认受力示意图预览
2. **设置约束条件**：左侧条件面板，勾选运动状态/接触面性质/斜面角度等，修改后示意图**实时更新**，无需确认
3. **受力分析展示**：中央区域显示物体+力的箭头，支持正交分解动画、合力显示、各力大小列表

### 功能点
| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | 受力模型选择 | 13种模型：水平面、斜面、悬挂、连接体、圆周运动、水槽、滑轮、杠杆、杆、半球、传送带、凹槽、弹簧 |
| 2 | 物体选择 | 物块/小球/滑环/木板，可选数量，自由组合 |
| 3 | 约束条件设置 | 运动状态（初速度/初始力）、接触面性质（光滑/粗糙+动摩擦因数）、斜面角度 |
| 4 | 自动受力分析 | 根据模型+约束自动识别受力，实时更新 |
| 5 | 正交分解动画 | 分量箭头和数值实时更新 |
| 6 | 合力计算与显示 | 各力大小列表 + 合力大小与方向 |
| 7 | 条件矛盾提示 | 系统检测并提示不合理的约束组合 |

### 验收标准
- 选择"斜面有摩擦，物体静止"，正确给出重力、支持力、摩擦力三个力，方向正确
- 正交分解动画流畅，分量箭头和数值实时更新
- 条件矛盾时正确提示
- 1080p投影仪上文字和箭头清晰可辨

## P-02 运动模拟器（与P-01深度融合）

### 产品价值
匀变速直线运动、抛体运动、圆周运动，学生只看公式没有感性认识。运动模拟器让老师设定初始条件，系统自动演示运动过程，实时显示速度/加速度向量和运动轨迹，并同步生成 v-t/s-t 图。

### 目标用户
高中物理老师，无编程基础。

### 使用场景
| 场景 | 描述 |
|------|------|
| 演示平抛运动 | 边播放边指出"水平速度不变，竖直速度逐渐增大" |
| 对比多个初速度 | 同屏展示不同初速度下的轨迹，直观看出落点变化 |
| 分析圆周运动 | 演示竖直圆周不同位置的速度和向心加速度变化 |

### 交互说明
1. **选择运动场景**：顶部6个场景按钮
2. **设置初始参数**：左侧面板输入质量、初速度（大小+方向角，拖拽箭头或数值输入）、受力、环境参数，修改后预览图实时更新
3. **运动演示**：中央区域播放动画，底部 v-t/s-t 图联动，时间轴控制栏统一操作

### 功能点
| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | 运动轨迹实时绘制 | 轨迹线留存画布，可选保留历史轨迹 |
| 2 | 速度/加速度向量显示 | 速度蓝色、加速度红色，长度按比例变化；平抛可显示水平/竖直分速度 |
| 3 | 时间轴控制 | 播放/暂停/单步前进后退/倒放/调速（0.1x~2x） |
| 4 | v-t/s-t 图联动 | 动画与图像同步，当前时刻用竖线标出 |
| 5 | 多物体对比模式 | 最多3组不同参数，不同颜色区分 |
| 6 | 圆周运动专项分析 | 水平圆周：摩擦力/绳拉力随角速度变化，超出最大静摩擦力高亮；竖直圆周：最高/最低点速度、向心加速度、脱轨临界条件 |
| 7 | 数值读取 | 点击轨迹任意点，显示 t/x/y/vx/vy/v/a |

### 运动场景
水平面运动、斜面运动、平抛运动、斜面上平抛、水平圆周运动、竖直圆周运动

### 验收标准
- 平抛运动水平速度不变、竖直速度线性增大，v-t 图正确
- 竖直圆周最高点临界速度 v=√(gR) 时正确提示脱轨
- 多物体对比模式颜色清晰，不卡顿
- 倒放功能正常

## P-03 光学实验台

### 产品价值
折射、反射、透镜成像靠画图容易出错，干涉、衍射极度抽象。光学实验台提供可交互的光路追迹环境，老师拖拽摆放光学元件，系统实时绘制光路。

### 目标用户
高中物理老师，无编程基础。

### 使用场景
| 场景 | 描述 |
|------|------|
| 演示全反射 | 调整入射角，直观看到临界角，演示光导纤维原理 |
| 展示干涉条纹 | 改变双缝间距，实时看条纹疏密变化 |
| 讲解衍射 | 切换波长，看不同颜色光的衍射图样差异 |

### 交互说明
1. **选择实验模块**：顶部5个按钮（折射与全反射、透镜成像、双缝干涉、单缝衍射、薄膜干涉）
2. **参数调节**：左侧面板调节折射率n/入射角θ/缝间距d/波长λ/屏幕距离L
3. **实验展示**：中央区域显示光路图或干涉衍射图样，右下角实时数值

### 功能点
| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | 折射与全反射 | 平行界面折射、矩形玻璃砖、半球形玻璃、光导纤维；形状（球/半球/矩形/挖空半球）+材料（玻璃/水/光导纤维/空气） |
| 2 | 透镜成像（初中） | 凸/凹透镜成像，透镜公式验证，演示为主 |
| 3 | 双缝干涉 | 展示实验现象，不同缝间距/距离/波长下的干涉条纹 |
| 4 | 单缝/圆孔衍射 | 可调缝宽/孔径和波长，实时显示衍射强度分布图 |
| 5 | 薄膜干涉 | 肥皂泡/楔形薄膜/牛顿环三种模型，可调薄膜厚度或波长 |

### 验收标准
- 全反射临界角与 sin θc = 1/n 一致
- 双缝干涉条纹间距随缝间距增大而减小
- 不同波长下衍射图样宽度差异可见
- 牛顿环为同心圆

## P-04 电路搭建器

### 产品价值
实物电路实验成本高、课堂操作慢，动态电路分析靠口述难以直观。电路搭建器提供拖拽式电路搭建环境，覆盖五大电学实验，支持动态调节参数，实时显示电流电压。

### 目标用户
高中物理老师，无编程基础。

### 使用场景
| 场景 | 描述 |
|------|------|
| 演示动态电路 | 调节滑动变阻器，实时看各支路电流电压变化 |
| 分析实验误差 | 对比内接法/外接法的测量值与真实值差异 |
| 五大实验讲解 | 直接调出对应实验电路模板 |

### 交互说明
1. **选择搭建方式**：自由搭建模式（拖拽元件+连线）或实验模板模式（预设电路）
2. **元器件操作**：左侧元器件库拖拽，点击元件显示参数面板
3. **实时计算**：搭建完成后自动计算各支路电流/电压/功率，滑动变阻器拖动时数值实时更新
4. **视图切换**：电路图 ↔ 实物图一键切换

### 功能点
| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | 自动电路分析 | 实时计算各支路电流、各元件电压、总功率 |
| 2 | 动态电路分析 | 调节任意元件参数，所有相关支路实时重算，箭头显示电流流向 |
| 3 | 量程超限提示 | 电流表/电压表超量程时高亮变红提示 |
| 4 | 故障模拟 | 设置某元件断路/短路，高亮显示异常支路 |
| 5 | 内/外接法对比 | 一键切换内接法/外接法，显示测量值与真实值误差及原因 |
| 6 | 电路图↔实物图切换 | 标准电路图与实物连线图对应（初中考察） |

### 元器件库
电源（ε、r）、电阻/滑动变阻器（R）、电容（C）、开关、电流表、电压表、灯泡、电阻箱、电动机

### 五大实验模板
选择电学元件、伏安法测电阻（内/外接）、半偏法测电流计内阻、测电源电动势和内阻、欧姆表原理与使用、电桥法测电阻

### 验收标准
- 调节滑动变阻器，数值实时变化且正确
- 内接法测量值偏小/外接法偏大，误差方向正确
- 量程超限时正确高亮提示
- 五大实验模板均可正常加载

## P-05 简谐运动与弹簧振子

### 产品价值
弹簧振子、单摆的运动图像与运动本身难以对应，学生难理解平衡位置、振幅、对称性等核心概念。本工具用 2D 动画同步展示振子运动与图像，直观展示对称性和变化规律。

### 目标用户
高中物理老师，无编程基础。

### 使用场景
| 场景 | 描述 |
|------|------|
| 展示平衡位置变化 | 不同弹簧/不同重物，对比平衡位置移动 |
| 讲解对称性 | 指出最高/最低点关于平衡位置位移对称、加速度对称 |
| 引入机械波 | 振子排列成一列，演示波的形成过程 |

### 交互说明
1. **选择振动模型**：顶部4个按钮（竖直弹簧振子、水平弹簧振子、单摆、多摆耦合）
2. **设置参数**：弹簧劲度系数k、物体质量m、初始位移、绳长L、摆角θ₀
3. **运动演示**：上方振子动画 + 下方 x-t/v-t/a-t 图联动 + 时间轴控制

### 功能点
| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | 平衡位置动态显示 | 虚线标出平衡位置，不同k或m下平衡位置变化以动画展示 |
| 2 | 最高/最低点对称性标注 | 实时标注位移对称性和加速度大小相等方向相反 |
| 3 | 运动图像联动 | x-t/v-t/a-t 图与动画实时同步（弹簧振子用，单摆不需要） |
| 4 | 小角度vs精确运动对比 | 单摆模式同屏显示近似简谐与精确数值解 |
| 5 | 多摆耦合能量传递 | 2~3个不同绳长单摆演示能量传递和共振 |
| 6 | 波的形成预览 | 多个振子排成一列，相邻固定相位差，演示行波形成（联动P-06） |

### 验收标准
- 更换质量后平衡位置正确下移，x=mg/k一致
- x-t图为标准正弦/余弦曲线，v-t与a-t相位关系正确
- 最高/最低点处速度为零、加速度最大的标注正确

## P-06 波动与振动演示台

### 产品价值
波的叠加、驻波、共振等概念极度抽象，粉笔图画不清楚。本工具用动画展示机械波传播、两列波叠加与干涉，重点解决"波形图"和"单质点振动图"的混淆。

### 目标用户
高中物理老师，无编程基础。

### 使用场景
| 场景 | 描述 |
|------|------|
| 展示波传递 | 选中一个质点，单独显示其振动图像，对应到整体波形图 |
| 演示干涉 | 切换相同/不同频率，直观看到稳定/不稳定干涉图样 |
| 展示驻波 | 慢动作演示驻波形成过程，标注波节和波腹 |

### 交互说明
1. **选择演示模块**：顶部6个按钮
2. **参数设置**：频率f、振幅A、波速v（两列波可分别设置）
3. **波形交互**：上方波形图可点击任意质点，下方自动显示该质点振动图

### 功能点
| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | 波形图与振动图对应 | 点击波形图上任意质点，下方自动切换该质点的 x-t 振动图 |
| 2 | 两列波叠加对比 | 同频叠加→稳定干涉（加强/减弱区域固定）；异频叠加→不稳定图样 |
| 3 | 驻波形成动画 | 入射波+反射波+合成驻波，波节波腹颜色标注，可控制慢放 |
| 4 | 多普勒效应可视化 | 声源运动波前实时绘制，前方压缩后方拉伸，可调声源速度 |
| 5 | 横波/纵波质点运动 | 横波箭头动画，纵波疏密变化动画 |

### 演示模块
单列波传播、两列波叠加、驻波演示、多普勒效应（初中）、波的平移问题、横波与纵波

### 验收标准
- 点击质点后振动图正确切换，相位关系正确
- 同频叠加图样稳定，异频叠加图样持续变化
- 驻波波节位置不动，波腹振幅为分量波两倍

## P-07 热力学与气体分子模拟器

### 产品价值
压强、温度、体积的宏观变化与微观分子运动关系难以建立直觉。液柱封闭气体、气缸活塞等模型下的变化过程抽象。本工具从微观粒子碰撞出发，可视化宏观热力学量变化。

### 目标用户
高中物理老师，无编程基础。

### 使用场景
| 场景 | 描述 |
|------|------|
| 演示三种变化过程 | 切换等温/等压/等容，显示对应图像 |
| 液柱模型 | 升温后气柱长度变化，与公式对照 |
| 气缸模型 | 双活塞升温后两侧活塞移动方向 |

### 交互说明
1. **选择演示模块**：顶部5个按钮
2. **参数输入**：温度T（拖动）、体积（拖活塞）、液柱长度、管方向
3. **演示区**：中央显示分子动画/液柱示意图/气缸图，下方 p-V/p-T/V-T 图

### 功能点
| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | 气体分子微观动画 | 升温→分子加快；减小体积→碰撞频率增大；实时显示p/V/T，验证pV/T=常数 |
| 2 | 三种变化过程对比 | 三容器并排演示等温/等压/等容，对应p-V/p-T/V-T图实时绘制 |
| 3 | 液柱密封气柱模型 | 竖直/水平/倾斜管，温度变化后气柱长度和压强动画+数值显示。**需要压强平衡计算** |
| 4 | 气缸/双活塞模型 | 单活塞温度调节+双活塞中间加热。**需要受力分析计算** |
| 5 | 布朗运动 | 大颗粒被分子随机撞击，无规则运动轨迹留存，颗粒越大运动越规则 |

> 功能 1/2/5 做到一起；功能 3/4 为常考计算题，需要试管/U型管/气缸/活塞美术资源。

### 验收标准
- 等温压缩时pV不变，误差≤1%
- 液柱模型升温后气柱变化量与公式一致
- 双活塞两侧移动方向正确，最终压强相等

## P-08 电场与磁场可视化器（仅次于力学，第二重要）

### 产品价值
电场线、磁感线是抽象概念，课本静态图形无法展示场的叠加。带电粒子在复合场中的运动轨迹、洛伦兹力边界模型难以手工画出。本工具提供可交互的场线绘制和粒子运动模拟。

### 目标用户
高中物理老师，无编程基础。

### 使用场景
| 场景 | 描述 |

## P-14 机械能守恒

### 产品价值
机械能守恒是高中力学核心定理，从自由落体到多物体复杂模型，能量转化过程不直观，弹性势能无法直接"看见"。本工具用能量条形图实时展示各种能量转化过程。

### 目标用户
高中物理老师，无编程基础。

### 使用场景
| 场景 | 描述 |
|------|------|
| 自由落体入门 | 高度减小速度增大，势能转化为动能，能量条实时变化 |
| 轻杆双球 | 演示从最低点到最高点，动能和势能如何交换 |
| 带弹簧模型 | 压缩释放，弹性势能转化为动能 |

### 交互说明
1. **选择模型**：顶部6个按钮
2. **参数设置**：质量m、初始高度h、弹簧系数k、摩擦系数，光滑/有摩擦切换
3. **三栏布局**：左侧参数面板 + 中央运动动画 + **右侧能量条形图**（特色交互）
4. **时间轴**：可暂停在任意位置查看能量数值

### 功能点
| 编号 | 功能 | 说明 |
|------|------|------|
| 1 | 实时能量条形图 | 重力势能（蓝）、动能（红）、弹性势能（绿）、总机械能（灰），运动中实时变化 |
| 2 | 自由落体基础演示 | 从设定高度下落，任意位置暂停验证 Ek+Ep=E总 |
| 3 | 轻杆圆周运动（双球） | 轻杆两端不同质量小球竖直面圆周运动，标注最高/最低点速度、向心力、能量关系 |
| 4 | 定滑轮双物体 | 设质量比，动画显示位移速度，验证总机械能守恒；可加摩擦显示能量减少 |

### 验收标准
- 任意时刻 Ek+Ep=E总，误差<0.1%
- 轻杆双球最高点速度与能量守恒计算一致
- 有摩擦时总能量减少量=摩擦力做功
- 弹簧振子压缩量最大时弹性势能最大，平衡位置时为零
