## 四种情况的判断口径

本阶段固定以下课堂约定，确保方向判断一致：

- 磁棒位于线圈右侧。
- 参数中的磁极，指的是“靠近线圈的一端”。
- 电流方向统一按“从右看线圈”判断。

由此得到：

- `N 极插入`
  - 原磁通量方向：向左
  - 磁通量变化：增大
  - 感应电流：逆时针（从右看）
  - 感应磁场：向右
  - 对磁棒安培力：排斥，向右，阻碍插入
- `N 极拔出`
  - 原磁通量方向：向左
  - 磁通量变化：减小
  - 感应电流：顺时针（从右看）
  - 感应磁场：向左
  - 对磁棒安培力：吸引，向左，阻碍拔出
- `S 极插入`
  - 原磁通量方向：向右
  - 磁通量变化：增大
  - 感应电流：顺时针（从右看）
  - 感应磁场：向左
  - 对磁棒安培力：排斥，向右，阻碍插入
- `S 极拔出`
  - 原磁通量方向：向右
  - 磁通量变化：减小
  - 感应电流：逆时针（从右看）
  - 感应磁场：向右
  - 对磁棒安培力：吸引，向左，阻碍拔出

## 分步分析入口

- 入口页：`#p13`
- 模型卡片：`EMI-001 磁棒-线圈楞次定律`
- 模型页左侧控制区提供：
  - `开始分析`

1. 原磁通量方向
2. 磁通量变化方向

5. 安培力方向
