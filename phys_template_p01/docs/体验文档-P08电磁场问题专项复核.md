# 体验文档：P-08 电磁场问题专项复核

更新时间：2026-04-14

## 1. 体验目标

本轮体验不再做泛化浏览，而是专门复核以下 6 个历史反馈点是否已经解决：

1. 平行板电容器场强、极板电荷量和相对介电常数的关系是否正确
2. 电容器偏转模型默认参数是否可穿板，是否支持挡板落点和自定义场景
3. 通电线圈空间磁场分布是否正确，圆形边界磁场中的粒子运动是否保留轨迹
4. 回旋加速器是否采用圆形磁场区域
5. 电场与重力同向时，速度不足的粒子是否会脱离轨道并转入斜抛
6. 磁场模块是否已补齐粒子源、多粒子轨迹、平移圆、旋转圆、放缩圆、磁聚焦、磁发散

本轮结论口径：

- 以上 6 项功能问题，当前版本均可按预期体验
- 仍有 1 项需要单独留意：部分历史文档曾统一写成 `E = U / d`，但当前运行逻辑已经区分恒压模型和定电荷模型

## 5. 分项体验说明

### 5.1 平行板电容器场强、极板电荷量与介电常数

问题背景：

- 早期担心“没有体现电荷量变化对电场的影响”
- 早期担心“相对介电常数对场强不该有影响，但公式可能写错”

体验入口：

- `P-08` → `静电场` → `平行板电容器`
- `P-08` → `带电粒子在电场中` → `电容器偏转模型`

操作步骤：

1. 先保持 `电容器模型 = 恒压`
2. 调节 `介电常数 εr`
3. 观察结果区中的 `E、C、Q`
4. 再切换到 `电容器模型 = 定电荷`
5. 调节 `极板电荷 Q`
6. 再次调节 `介电常数 εr`

当前正确现象：

- 在 `恒压模型` 下：
  - `E = U / d`
  - 改变 `εr` 时，`E` 不应直接变化
  - `C` 和 `Q` 会变化
- 在 `定电荷模型` 下：
  - `E = Q / (ε0 εr S)`
  - 改变 `Q` 时，`E` 会变化
  - 改变 `εr` 时，`E` 会反向变化

当前复核结论：

- 该问题已修复
- 当前系统已经明确区分 `恒压` 与 `定电荷` 两条口径
- 若后续有人引用旧文档里统一的 `E = U / d` 说法，应判定为历史文案残留，不是当前运行逻辑错误

### 5.2 电容器偏转模型默认参数、挡板落点与自定义场景

问题背景：

- 早期默认参数不合适，粒子可能不能穿过电容器
- 希望在平行板后方增加挡板继续运行
- 希望场景模型可自行设置

体验入口：

- `P-08` → `带电粒子在电场中` → `电容器偏转模型`
- `P-08` → `进入场搭建器`

操作步骤：

1. 进入 `电容器偏转模型`
2. 不改参数直接点击运行
3. 观察粒子是否穿出极板并命中接收屏
4. 增大 `板间电压 U` 或减小 `水平初速度 v0`
5. 观察结果区是否出现 `预计撞板`
6. 进入 `P-08 场搭建器`
7. 添加 `匀强电场`、`带电粒子`、`接收屏`
8. 手动拖拽位置，组成自定义偏转场景

当前正确现象：

- 默认参数下，粒子应能穿出极板并命中后方接收屏
- 参数过大导致无法穿板时，不应无提示停住，而应显示 `预计撞板`
- 在 Builder 中可手动添加 `接收屏`
- 在 Builder 中可自由调整电场区域、粒子与接收屏位置，形成自定义场景

当前复核结论：

- 该问题已修复
- 默认参数已能稳定穿板
- 挡板落点链路已由 `接收屏` 承接
- 自定义场景能力已由 `P-08 场搭建器` 提供

### 5.3 通电线圈空间磁场分布、圆形边界磁场轨迹

问题背景：

- 早期认为通电线圈的空间磁场分布不正确
- 早期圆形边界磁场中的粒子运动没有轨迹留存

体验入口：

- `P-08` → `静磁场` → `圆形电流磁场`
- `P-08` → `带电粒子在磁场中` → `圆形边界磁场`

操作步骤：

1. 进入 `圆形电流磁场`
2. 切换 `逆时针 / 顺时针`
3. 调节 `线圈半径`
4. 观察中心轴向磁场和外侧闭合回弯磁场
5. 再进入 `圆形边界磁场`
6. 点击运行
7. 确认右上角 `轨迹留存` 处于显示状态
8. 观察粒子在圆形磁场区域内走圆弧，出场后继续直线

当前正确现象：

- 圆形电流场景应表现为典型偶极型磁场
- 切换电流方向后，中心磁场方向和整体回绕方向同步翻转
- 圆形边界磁场中的粒子轨迹应被持续绘制
- 出场后应保留完整历史轨迹，不是只显示当前位置

当前复核结论：

- 该问题已修复
- 线圈空间场分布已采用专门教学视图重做
- 圆形边界磁场的粒子运动已支持轨迹留存

### 5.4 回旋加速器磁场形状

问题背景：

- 早期反馈认为回旋加速器磁场应该是圆形，而不是普通矩形场区

体验入口：

- `P-08` → `复合场` → `回旋加速器`

操作步骤：

1. 进入 `回旋加速器`
2. 观察磁场区域外边界
3. 点击运行
4. 观察粒子在缝隙处加速、在 D 形区内做圆弧运动
5. 调节 `磁感应强度 B`
6. 观察轨迹半径与周期变化

当前正确现象：

- 磁场区应呈圆形区域
- 粒子轨迹应在圆区内逐步放大
- 穿越电场缝隙后速度增加，回旋半径增大

当前复核结论：

- 该问题已修复
- 当前回旋加速器预设已明确使用圆形磁场区域

### 5.5 重力场与电场复合：电场与重力同向时的低速运动

问题背景：

- 早期反馈认为“电场与重力同向、速度小上不去”时，不应继续假装走整圆，而应脱离轨道并转为斜抛

体验入口：

- `P-08` → `复合场` → `静电场 / 重力场圆周运动临界`

操作步骤：

1. 进入该场景
2. 把 `电场方向` 切到 `向下，与重力同向`
3. 逐步减小 `底端初速度 v0`
4. 点击运行
5. 观察粒子是否在某处失去约束
6. 观察结果区是否给出 `已脱离圆轨道，转入斜抛`

当前正确现象：

- 当速度不足时，粒子不会继续沿整圆运动
- 张力或支持力降到零后，应从圆轨道脱离
- 脱离后应沿切线方向进入受 `电场 + 重力` 共同作用的斜抛运动

当前复核结论：

- 该问题已修复
- 当前模型已显式区分“仍受约束的圆轨道运动”和“脱离后的斜抛运动”

### 5.6 粒子源模型、多粒子轨迹与磁场几何模型

问题背景：

- 早期希望补充粒子源模型
- 希望能同时发射多个粒子并显示轨迹
- 希望补齐 `平移圆`、`旋转圆`、`放缩圆`、`磁聚焦`、`磁发散`

体验入口：

- `P-08` → `带电粒子在磁场中`
- `P-08` → `进入场搭建器`

建议体验预设：

- `平移圆模型`
- `旋转圆模型`
- `放缩圆模型`
- `磁聚焦模型`
- `磁发散模型`

操作步骤：

1. 依次进入以上 5 个预设
2. 点击运行
3. 观察是否为多粒子同时发射
4. 观察每条粒子轨迹是否持续留存
5. 再进入 Builder
6. 添加 `粒子源` 与 `匀强磁场`
7. 切换 `发射模式`
8. 调节 `粒子数、基准速度、速度跨度、入射角张角、会聚距离`

当前正确现象：

- `平移圆模型`：多粒子等半径，仅整体平移
- `旋转圆模型`：多粒子同点出发、等半径、不同入射角
- `放缩圆模型`：同方向下不同速度，对应不同半径
- `磁聚焦模型`：轨迹在焦点附近重新汇聚
- `磁发散模型`：速度越大半径越大，轨迹逐渐分散
- Builder 中应存在 `粒子源` 实体，并支持以上发射模式

当前复核结论：

- 该问题已修复
- 当前版本已经补齐粒子源和多粒子轨迹模型
- 上述 5 类几何模型均可直接体验或在 Builder 中复用

## 6. 体验时重点记录什么

若出现下列情况，建议截图并记录参数：

- 恒压模型下，调节 `εr` 时 `E` 发生明显变化
- 定电荷模型下，调节 `Q` 或 `εr` 时 `E` 没有变化
- 电容器偏转默认参数仍然无法穿板
- 圆形边界磁场中轨迹不显示，或 `轨迹留存` 开关无效
- 回旋加速器磁场区肉眼看起来仍像矩形场
- 重力与电场同向、低速时，粒子没有脱轨而是继续假整圆
