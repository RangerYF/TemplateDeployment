**2. 静电场**

**EMF-001 单点电荷场**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  -------------- -------------- -------------- -------------- --------------

场强：E = kQ/r²（k = 9×10⁹ N·m²/C²）

电势：V = kQ/r

**EMF-003 平行板电容器**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  板间电压 U     100            10\~1000       10             V

  板间距 d       5              1\~20          0.5            cm

  板长 L         10             2\~30          1              cm
  -------------- -------------- -------------- -------------- --------------

匀强电场（当前产品口径，恒压模型）：E = U/d

介电常数 εr 通过 C = ε0εrS/d 与 Q = CU 影响储能和极板电荷，不直接改变板间 E

场线：平行等距直线（边缘弯曲）

等势面：主体匀强区中近似平行于极板，边缘区随 fringe field 做合理弯曲近似（2D 画面中表现为中段平直、边缘渐弯的等势线）

**3. 带电粒子在电场中运动**

注：当前产品 preset id / route 口径采用 `EMF-011=加速电场`、`EMF-012=电容器偏转模型`；
原始草案中两者曾互换，以下按当前实现同步编号，但保留本节原有叙述顺序。

**EMF-012 电容器偏转模型**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  板间电压 U     200            10\~2000       10             V

  板间距 d       5              1\~20          0.5            cm

  板长 L         10             2\~30          1              cm

  粒子初速度 v0  10⁶            10⁴\~10⁸       ---            m/s

  荷质比 q/m     1.76×10¹¹      可调           ---            C/kg
  -------------- -------------- -------------- -------------- --------------

偏转量：y = qUL²/(4mdv0²)

出口偏转角：tan θ = qUL/(2mdv0²)

运动分析：

水平：匀速 x = v0 t

竖直：匀加速 y = (1/2)(qE/m)t² = (1/2)(qU/md)t²

轨迹：抛物线

**EMF-011 加速电场**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  加速电压 U     1000           100\~10000     100            V

  粒子电荷 q     1.6×10⁻¹⁹      ---            ---            C

  粒子质量 m     1.67×10⁻²⁷     ---            ---            kg
  -------------- -------------- -------------- -------------- --------------

动能定理：qU = (1/2)mv² - (1/2)mv0²

从静止加速：v = sqrt(2qU/m)

**6. 复合场模型**

**EMF-041 速度选择器**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  电场强度 E     10⁴            10²\~10⁶       ---            V/m

  磁感应强度 B   0.1            0.01\~1        0.01           T
  -------------- -------------- -------------- -------------- --------------

选择条件：qE = qvB → v = E/B

只有速度 v = E/B 的粒子能直线通过

**EMF-042 回旋加速器**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  磁感应强度 B   1.0            0.1\~5         0.1            T

  加速电压 U     1000           100\~10000     100            V

  D形盒半径 R    50             10\~100        1              cm
  -------------- -------------- -------------- -------------- --------------

交变电场频率 = 回旋频率：f = qB/(2πm)

最大动能：Ek_max = q²B²R²/(2m)

螺旋加速轨迹：每加速一次半径增大

**EMF-043 电磁流量计**

导电流体在管道中流动

教学表达采用“三阶段近似”：

1. 正负电荷先在 `qvB` 作用下向两侧分离

2. 管道两侧积累异号电荷，建立感应电场 `E`

3. 达到平衡时 `qE = qvB`，故 `E = vB`，端电压 `U = EL = BvL`

外加磁场 B，管道两侧产生电压

U = BvL（v 为流速，L 为管道直径）
