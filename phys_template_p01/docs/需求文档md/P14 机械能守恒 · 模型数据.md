  -------------- -------------- -------------- -------------- --------------

**3. 定滑轮双物体**

**ENE-011 阿特伍德机**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  m1（重物）     3.0            0.1\~20        0.1            kg

  m2（轻物）     2.0            0.1\~20        0.1            kg

  初始高度差 h   2.0            0.5\~10        0.1            m
  -------------- -------------- -------------- -------------- --------------

加速度：a = (m1-m2)g/(m1+m2)

下落 h 后速度：v = sqrt(2ah)

能量守恒：

m1 下降 h：Ep1 减少 m1gh

m2 上升 h：Ep2 增加 m2gh

两物体动能增加：(1/2)(m1+m2)v²

守恒：(m1-m2)gh = (1/2)(m1+m2)v²

可选添加摩擦力（绳与滑轮摩擦）

**4. 斜面滑块**

**ENE-021 光滑斜面**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  质量 m         2.0            0.1\~10        0.1            kg

  斜面角度 θ (°) 30             5\~85          1              ---

  斜面长度 L     5.0            1\~20          0.1            m

  初速度 v0      0              0\~20          0.1            m/s
  -------------- -------------- -------------- -------------- --------------

高度 h = L sin θ

能量守恒：(1/2)mv² = (1/2)mv0² + mgh

**ENE-022 有摩擦斜面**

  ----------------------------------- -----------------------------------
  参数                                额外参数

  动摩擦因数 μ                        默认 0.2，范围 0\~0.8
  ----------------------------------- -----------------------------------

摩擦做功：Wf = μmg cos θ × L

能量关系：(1/2)mv² = (1/2)mv0² + mgh - Wf

总机械能减少量 = 摩擦热 Q = Wf

**9. 机械能公式汇总**

  ----------------------------------- -----------------------------------
  公式                                说明

  Ek = (1/2)mv²                       动能

  Ep = mgh                            重力势能

  Ep弹 = (1/2)kx²                     弹性势能

  E = Ek + Ep + Ep弹                  总机械能

  ΔE = -Wf                            机械能变化量 = 非保守力做功

  Wf = μmg cos θ × d                  摩擦力做功
  ----------------------------------- -----------------------------------
