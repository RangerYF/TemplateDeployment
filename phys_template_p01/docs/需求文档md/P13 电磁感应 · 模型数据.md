**2. 单棒模型**

**EMI-011 单棒基础（纯电阻电路）**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  磁感应强度 B   0.5            0.1\~5         0.1            T

  导轨间距 L     0.5            0.1\~2         0.1            m

  导体棒质量 m   0.1            0.01\~1        0.01           kg

  导体棒电阻     1.0            0.1\~10        0.1            Ω
  R_棒                                                        

  外接电阻 R     2.0            0.1\~20        0.1            Ω

  初速度 v0      5.0            0\~20          0.1            m/s

  摩擦系数 μ     0              0\~0.5         0.01           ---
  -------------- -------------- -------------- -------------- --------------

感应电动势：ε = BLv

电路电流：i = ε/(R + R_棒) = BLv/(R + R_棒)

安培力：F_安 = BiL = B²L²v/(R + R_棒)（阻碍运动）

**无摩擦终态**：v 趋于 0（指数衰减）

v(t) = v0 × exp(-B²L²t / (m(R+R_棒)))

i(t) = BLv0/(R+R_棒) × exp(\...)

**有摩擦终态**：

若 μmg \< 初始安培力 → 先减速

最终停止（v = 0）

**EMI-012 单棒 + 含电源**

  ----------------------------------- -----------------------------------
  参数                                额外参数

  外接电源电动势 ε0 (V)               默认 3.0，范围 0\~20
  ----------------------------------- -----------------------------------

终态：安培力 = 0（或安培力平衡其他力）

终态速度：v终 = ε0/(BL)（电源驱动匀速时）

终态电流取决于具体电路

**EMI-013 单棒 + 含电容**

  ----------------------------------- -----------------------------------
  参数                                额外参数

  电容 C (μF)                         默认 100，范围 1\~1000
  ----------------------------------- -----------------------------------

终态特征：

电流为零（电容充满后不再有电流）

导体棒匀速或静止

电容两端电压 = BLv终

**3. 双棒模型**

**EMI-021 双棒基础（无摩擦）**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  B              0.5            0.1\~5         0.1            T

  L              0.5            0.1\~2         0.1            m

  棒1质量 m1     0.1            0.01\~1        0.01           kg

  棒2质量 m2     0.2            0.01\~1        0.01           kg

  棒1初速度 v1   5.0            0\~20          0.1            m/s

  棒2初速度 v2   0              0\~20          0.1            m/s

  各棒电阻 R1,   1.0, 1.0       0.1\~10        0.1            Ω
  R2                                                          
  -------------- -------------- -------------- -------------- --------------

无摩擦终态：两棒共速

动量守恒：m1v1 + m2v2 = (m1+m2)v终

v终 = (m1v1 + m2v2)/(m1+m2)

棒1减速、棒2加速，电流先大后小

**EMI-022 双棒（有摩擦）**

终态：

棒1可能停止

棒2可能停止或匀速

取决于安培力与摩擦力的关系

**EMI-023 双棒 + 含电容**

含电容时终态电流为零

两棒各自匀速（速度可能不同）

电容存储能量

**4. 竖直导轨单棒**

**EMI-031 竖直导轨**

  -------------- -------------- -------------- -------------- --------------
  参数           默认值         范围           步进           单位

  B              0.5            0.1\~5         0.1            T

  L              0.5            0.1\~2         0.1            m

  棒质量 m       0.1            0.01\~1        0.01           kg

  棒电阻 R_棒    1.0            0.1\~10        0.1            Ω

  外接电阻 R     2.0            0.1\~20        0.1            Ω
  -------------- -------------- -------------- -------------- --------------

导体棒从静止开始下落

安培力 = B²L²v/(R+R_棒)（向上，阻碍下落）

终态：重力 = 安培力

mg = B²L²v终/(R+R_棒)

v终 = mg(R+R_棒)/(B²L²)

终态电流：i终 = BLv终/(R+R_棒) = mg/(BL)

**5. 安培力分析流程**

**6. 电磁感应公式汇总**

  ----------------------------------- -----------------------------------
  公式                                说明

  ε = BLv                             动生电动势

  ε = -dΦ/dt                          法拉第电磁感应定律

  ε = nΔΦ/Δt                          线圈感应电动势

  F = BIL                             安培力

  Φ = BS cos θ                        磁通量

  i = ε/(R+r)                         感应电流
  ----------------------------------- -----------------------------------
