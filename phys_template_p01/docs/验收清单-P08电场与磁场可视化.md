# 验收清单：P-08 电场与磁场可视化器

更新时间：2026-04-06

## 1. 适用范围

- 本清单对应当前仓库的 `P-08 电场与磁场可视化器`。
- 本轮验收目标是确认 P-08 已达到“可验收、可说明、可对外发布教学体验”的状态。
- 本清单覆盖 `/#p08` 预设入口与 `/#p08-builder` 场搭建器 MVP。
- 本文按当前产品 preset id 口径记录：`EMF-011=加速电场`，`EMF-012=电容器偏转模型`。

## 2. 通用进入方式

1. 启动应用后进入首页。
2. 点击 `P-08 电场与磁场可视化器`。
3. 在 `/#p08` 页面顶部选择对应模块。
4. 点击场景卡片进入具体预设，或点击 `进入场搭建器` 进入独立 builder 页面。
5. 进入后，使用左侧参数面板、中间画布、右下角实时结果区和右上角显示与测量区完成验收。

## 3. 手动验收项

### 3.1 电场线与等势线

- 入口：`/#p08` -> `静电场` -> `单点电荷场`、`双点电荷场`、`平行板电容器`
- 操作步骤：
  1. 进入 `单点电荷场`，调节电荷量正负和大小。
  2. 观察电场线方向和疏密变化。
  3. 在右上角确认 `电场线`、`等势线` 开关可独立控制，并可切换 `场线密度 = 疏 / 标准 / 密`。
  4. 在画布上依次点击两点 A、B，确认两点电势差可显示。
  5. 进入 `双点电荷场`，切换 `等量异号`、`等量同号（正）`、`不等量异号` 等构型。
  6. 进入 `平行板电容器`，调节极板电压、间距和介质参数，观察板间场线、等势线和边缘效应。
- 预期现象：
  - 正电荷场线向外、负电荷场线向内，等势线呈同心分布。
  - 双点电荷不同构型下，合场线与中点场强表现不同。
  - `电场线`、`等势线` 可以单独显示或隐藏。
  - `场线密度` 切到 `疏 / 标准 / 密` 时，点电荷和平行板场线条数会有明确变化，不是空开关。
  - 放置 A/B 点后，右下角和信息面板都能读取 `ΔV = VA - VB`。
  - 平行板场景中，板间主体区域近似匀强，边缘位置出现弯曲；等势线在主体区与极板平行，在边缘区会随 fringe field 做近似弯曲，不再是整块死板平行线。
- 关键数值或公式：
  - `E = kQ / r²`
  - `V = kQ / r`
  - `E = U / d`；当前平行板预设按恒压模型处理，`εr` 影响 `C` 和 `Q`

### 3.2 电场偏转（当前产品 preset id：EMF-012）

- 入口：`/#p08` -> `带电粒子在电场中` -> `电容器偏转模型`
- 操作步骤：
  1. 进入场景后点击运行。
  2. 调节 `板间电压 U`、`板间距 d`、`极板长度 L`、`水平初速度 v0`、粒子 `q`、`m`。
  3. 观察右下角的 `当前速度`、`当前加速度`、`当前偏转量`、`当前/出口偏角`、`预计结果` 与 `预计/实际屏上落点`。
  4. 在右上角切换 `轨迹留存`，确认轨迹显示可控。
- 预期现象：
  - 默认参数下，粒子在极板间受电场力偏转，离开极板后保持离场瞬间速度继续做匀速直线运动，并命中接收屏。
  - 电场越强、荷质比越大，偏转越明显。
  - 水平初速度越大，偏转量减小。
  - 参数过大导致无法穿板时，结果区会明确提示“预计撞板”，而不是无提示地提前停住。
  - 右下角结果区持续刷新偏转量、偏角、速度与落点信息，信息面板公式口径为 `E = U / d`。
- 关键数值或公式：
  - `E = U / d`
  - `y = 1/2 (qU / md) (L / v0)²`
  - `tanθ = qUL / (mdv0²)`

### 3.3 电场加速（当前产品 preset id：EMF-011）

- 入口：`/#p08` -> `带电粒子在电场中` -> `加速电场`
- 操作步骤：
  1. 进入场景后点击运行。
  2. 观察粒子从静止出发的运动过程。
  3. 调节 `加速电压 U`、粒子 `q`、`m`。
  4. 查看右下角 `当前速度`、`当前加速度`、`理论出口速度` 的变化。
- 预期现象：
  - 粒子从静止开始沿电场方向加速，离开加速区后保持离场速度继续做匀速直线运动。
  - 加速电压增大或荷质比增大时，加速度增大。
  - 右下角结果区能够持续给出速度、加速度和理论出口速度，信息面板口径为 `qU = ΔEk`。
- 关键数值或公式：
  - `F = qE = ma`
  - `qU = ΔEk`
  - `v = √(2qU / m)`

### 3.4 磁感线可视化

- 入口：`/#p08` -> `静磁场` -> `长直导线磁场`、`圆形电流磁场`、`螺线管磁场`
- 操作步骤：
  1. 分别进入三个场景。
  2. 调节电流大小和 `电流方向`，观察磁感线箭头、截面符号、结果区和课堂提示是否同步变化。
  3. 在圆形电流场景中调节线圈半径。
  4. 在螺线管场景中调节电流、方向和匝数。
- 预期现象：
  - 长直导线周围出现同心圆磁感线，切换电流方向后环绕箭头方向同步反转。
  - 圆形电流场景中，线圈中心磁场强度随半径和电流变化，切换电流方向后左右两侧截面符号和轴线箭头同步翻转。
  - 螺线管内部磁场更接近匀强，切换电流方向后内部 `B` 箭头与说明同步翻转。
  - 右下角结果区和左下角课堂提示都能显示方向描述。
- 关键数值或公式：
  - `B = μ0I / (2πr)`
  - `B0 = μ0I / (2R)`
  - `B ≈ μ0nI`

### 3.5 洛伦兹力圆周和边界模型

- 入口：`/#p08` -> `带电粒子在磁场中` -> `洛伦兹力圆周运动`、`直线边界磁场`、`圆形边界磁场`、`平移圆模型`、`旋转圆模型`、`半圆边界磁场`、`双边界磁场`
- 操作步骤：
  1. 先进入 `洛伦兹力圆周运动`，调节磁场方向、粒子速度和荷质比。
  2. 确认右下角实时显示 `圆周半径 r`、`周期 T`。
  3. 进入 `直线边界磁场`，改变入射速度或电荷量，观察入射和出射轨迹。
  4. 进入 `圆形边界磁场` 与 `半圆边界磁场`，观察边界几何变化对出射点和出射角的影响。
  5. 进入 `平移圆模型`，确认多个粒子以相同速度从不同入射点进入后，形成等半径的平移圆轨迹族。
  6. 进入 `旋转圆模型`，确认多个粒子从同一点以不同入射角进入后，形成等半径的旋转圆轨迹族。
  7. 进入 `双边界磁场`，确认粒子会形成分段 S 形偏转。
- 预期现象：
  - 基础圆周运动场景中，粒子在匀强磁场里按电荷正负与磁场方向决定偏转方向。
  - 边界场景中，粒子离开磁场区域后转为匀速直线运动。
  - 直线边界、圆形边界、平移圆、旋转圆、半圆边界和双边界的几何表现彼此可区分。
  - `平移圆模型` 中不同粒子的半径相同，但整条轨迹整体平移。
  - `旋转圆模型` 中不同粒子的半径相同，但圆心位置会随入射角旋转分布。
  - 结果区数值会随 `B`、`v`、`q/m` 变化而变化。
- 关键数值或公式：
  - `r = mv / (|q|B)`
  - `T = 2πm / (|q|B)`

### 3.6 电磁流量计

- 入口：`/#p08` -> `复合场` -> `电磁流量计`
- 操作步骤：
  1. 进入场景后点击运行。
  2. 调节 `流速 v`、`磁感应强度 B`、`管径 L`。
  3. 观察右下角结果区是否同步显示 `v`、`B`、`L`、`U`。
  4. 观察正负两粒子的速度是否保持同一参数口径。
  5. 观察画面、结果区和信息面板是否都按“正负电荷分离 -> 建立感应电场 -> 达到平衡”的阶段口径说明过程，并在信息面板确认文案包含“平衡时 `E = vB`，端电压 `U = BvL`”。
- 预期现象：
  - 正负两粒子共用同一流速参数，结果区与信息面板读数一致。
  - 画面中的感应电场会经历“由弱到强”的建立过程，并与磁场、流速共同构成平衡关系；粒子主运动方向保持沿管道前进。
  - 改变 `v`、`B` 或 `L` 后，端电压 `U` 会按 `BvL` 同步变化。
- 关键数值或公式：
  - `平衡时 E = vB`
  - `U = EL = BvL`

### 3.12 静电场 / 重力场圆周运动临界

- 入口：`/#p08` -> `复合场` -> `静电场 / 重力场圆周运动临界`
- 操作步骤：
  1. 进入场景后点击运行，观察带电小球沿竖直圆轨道运动。
  2. 调节 `圆轨道半径 R`、`底端初速度 v0`、`荷质比 q/m`、`电场强度 E`。
  3. 在 `电场方向` 中切换“向上，抵消部分重力”和“向下，与重力同向”。
  4. 观察右下角的 `等效重力 g_eff`、`顶端临界速度`、`底端临界速度`、`当前张力 / 支持力`、`判定`。
- 预期现象：
  - 当电场力与重力反向时，`g_eff` 变小，整圆所需的临界速度降低。
  - 当电场力与重力同向时，`g_eff` 变大，整圆所需的临界速度升高。
  - 若 `v0` 低于 `√(5g_effR)`，粒子会在失去约束后沿切线方向脱离圆轨道，并继续在重力场与电场共同作用下做类斜抛运动。
  - 结果区和信息面板都会同步显示临界公式与判定口径。
- 关键数值或公式：
  - `g_eff = g ± qE/m`
  - `v顶临界 = √(g_eff R)`
  - `v底临界 = √(5g_eff R)`

### 3.13 场搭建器 / 场编辑器 MVP

- 入口：首页 `P-08 场搭建器` 或 `/#p08` -> `进入场搭建器`，也可直接访问 `/#p08-builder`
- 操作步骤：
  1. 进入后在左侧依次添加 `点电荷`、`带电粒子`、`匀强电场`、`匀强磁场`、`长直导线`、`圆形电流`、`螺线管`。
  2. 在左侧切换 `场 / 运动 / 受力` 视角。
  3. 选中任一实体，调节其核心参数：电荷量、电场强度/方向、磁场强度/方向、电流大小/方向、匝数、粒子质量、初速度大小/方向。
  4. 在画布中拖拽实体改变位置，使用底部时间轴进行 `播放 / 暂停 / 重置`。
  5. 观察右上角显示与测量区、左下角课堂提示、右下角实时结果区、右侧信息面板是否继续工作。
- 预期现象：
  - builder 为独立页面，不依赖预设卡片占位。
  - 添加/删除/拖拽都可直接操作，参数修改后画面和摘要同步更新。
  - 纯静电场场景可继续测两点电势差；带粒子运动场景可继续显示轨迹和结果区。
  - 当前 builder 只做课堂搭建 MVP，不要求保存/加载。
