# 体验文档：P-08 电场与磁场可视化器

更新时间：2026-04-06

## 1. 这是什么

- `P-08 电场与磁场可视化器` 面向高中物理老师，用来演示电场线、等势线、磁感线以及带电粒子在电场、磁场和复合场中的运动。
- 当前版本已经不是“零散预设集合”，而是包含 `/#p08` 预设首页和 `/#p08-builder` 场搭建器的双入口。
- 本轮在前 5 个阶段基础上，补齐了独立场搭建器、电场偏转/加速参数口径，以及静磁场三类场景的电流方向联动。

## 2. 怎么进入

1. 从首页点击 `P-08 电场与磁场可视化器`。
2. 或先进入模板库，再点击顶部的 `P-08` 独立模块卡片。
3. 也可以直接访问 `/#p08`。
4. 需要空白搭建时，可从首页进入 `P-08 场搭建器`，或在 `/#p08` 顶部点击 `进入场搭建器`，也可直接访问 `/#p08-builder`。

## 3. 页面怎么看

- 顶部：5 个教学模块按钮。
- 左侧：参数调节区。
- 中间：场线、磁感线和粒子轨迹演示区。
- 右上角：`显示与测量`，可切换电场线、等势线、轨迹留存，并在静电场场景测两点电势差。
- 右下角：`实时结果`，读取速度、半径、周期、偏转量、场强等关键量。
- 右侧信息面板：课堂速览、关键公式、结果解读。
- builder 页面沿用同一套画布、结果区、显示与测量区、信息面板和时间轴，只把左侧改成“加实体 + 选中对象参数”。

## 4. 五个模块分别适合讲什么

### 4.1 静电场

- 适合讲：点电荷场、双点电荷场的叠加，电场线与等势线，两点电势差，平行板匀强电场。
- 推荐先点：`单点电荷场`
- 再看：`双点电荷场`、`平行板电容器`
- 当前已完成核心能力：
  - 电场线与等势线显示
  - 场线密度 `疏 / 标准 / 密` 调节
  - 两点电势差测量
  - 平行板电场的近似匀强区、边缘效应与“主体平直 + 边缘渐弯”的等势线演示

### 4.2 带电粒子在电场中

- 适合讲：电容器偏转模型、加速电场、电场力做功、类平抛轨迹。
- 推荐先点：`电容器偏转模型`
- 再看：`加速电场`
- 当前已完成核心能力：
  - 电场偏转轨迹
  - 离场后轨迹延续、出口偏角与接收屏落点
  - `U、d、L、v0、q、m` 一套参数口径
  - `qU = ΔEk` 的加速电场演示

### 4.3 静磁场

- 适合讲：长直导线、圆形电流、螺线管的磁感线，安培定则和左手定则。
- 推荐先点：`长直导线磁场`
- 再看：`螺线管磁场`、`安培力演示：导线在外磁场中受力`
- 当前已完成核心能力：
  - 三类磁感线可视化
  - 电流方向可调，画面方向提示同步
  - 右下角磁场代表值读取
  - 导线在外磁场中受力方向动画

### 4.4 带电粒子在磁场中

- 适合讲：洛伦兹力圆周运动、直线边界、圆形边界、平移圆、旋转圆、半圆边界、双边界、磁聚焦、磁发散。
- 推荐先点：`洛伦兹力圆周运动`
- 再看：`直线边界磁场`、`圆形边界磁场`、`平移圆模型`、`旋转圆模型`、`半圆边界磁场`、`双边界磁场`、`磁聚焦模型`、`磁发散模型`
- 当前已完成核心能力：
  - 圆周半径与周期实时显示
  - 多类边界模型
  - 平移圆 / 旋转圆几何模型
  - 磁聚焦 / 磁发散成对对比

### 4.5 复合场

- 适合讲：速度选择器、回旋加速器、电磁流量计、静电场 / 重力场圆周运动临界条件。
- 推荐先点：`速度选择器`
- 再看：`回旋加速器`、`电磁流量计`、`静电场 / 重力场圆周运动临界`
- 当前已完成核心能力：
  - `v = E / B` 的速度选择演示
  - 回旋加速器中的周期与半径变化
  - 电磁流量计中的“正负电荷分离 -> 建立感应电场 -> 达到平衡时 `E = vB`，端电压 `U = BvL`”
  - `g_eff = g ± qE/m` 的临界条件演示
  - 底端临界速度、顶端临界速度和整圆判定输出

## 5. 当前对外可怎么说

- 可以对外说明“P-08 已形成 `/#p08` 预设入口 + `/#p08-builder` 场搭建器 MVP 的双入口课堂体验”。
- 可以对外说明“电场线、等势线、场线密度调节、两点电势差、电场偏转 / 加速、磁感线、洛伦兹力边界模型、平移圆 / 旋转圆、速度选择器、回旋加速器、电磁流量计、静电场 / 重力场圆周运动临界、磁聚焦 / 磁发散、安培力方向演示，以及独立场搭建器均已可体验并可手动验收”。
- 当前不应对外宣称 builder 已支持保存/加载或模板扩展能力。

## 6. 推荐的课堂试用顺序

1. `静电场` -> `单点电荷场`
2. `带电粒子在电场中` -> `电容器偏转模型`
3. `静磁场` -> `长直导线磁场`
4. `带电粒子在磁场中` -> `洛伦兹力圆周运动`
5. `复合场` -> `速度选择器`

## 7. 本轮完成与保留

### 7.1 已完成核心能力

- P-08 产品命名、模块命名、场景命名、摘要文案已统一到同一口径。
- 当前产品编号口径已同步为：`EMF-011=加速电场`，`EMF-012=电容器偏转模型`。
- P-08 相关 preset 已统一归到 `P-08` 分类，模板库通过独立入口承接，不再重复展示 P-08 场景列表。
- `平移圆模型`、`旋转圆模型`、`静电场 / 重力场圆周运动临界` 已补齐到可演示、可验收状态。
- 独立 `场搭建器 / 场编辑器` 入口已补到 MVP：支持添加/删除/拖拽、调核心参数、运行/暂停/重置。
- 电场偏转 / 加速页已改成 `U、d、L、v0、q、m` 口径；偏转页默认参数可穿出极板，并新增接收屏落点与撞板预判。
- 单点电荷 / 平行板场景已补齐 `场线密度` 控制，平行板等势线可独立显示，边缘区已改成与 fringe field 相匹配的弯曲近似。
- 电磁流量计已补齐 `v、B、L、U` 结果链，画面、信息面板与结果区统一使用“正负电荷分离 -> 建立感应电场 -> 达到平衡时 `E = vB`，端电压 `U = BvL`”口径。
- 长直导线 / 圆形电流 / 螺线管已补齐电流方向控制，渲染与说明同步。
- 验收清单、教师体验文档和 Phase 5 自查文档已补齐，并新增本轮代码级/渲染级近似验收记录。

### 7.2 未完成 / 保留项

- 场搭建器当前只做 MVP：不含保存/加载、模板库扩展和持久化
- 电磁流量计当前仍是教学化近似，不是完整流体瞬态数值求解
- 真实浏览器截图级人工验收在当前环境中受限于 Playwright 浏览器下载失败（`EAI_AGAIN`），因此本轮主要完成了代码级与渲染级近似验收
