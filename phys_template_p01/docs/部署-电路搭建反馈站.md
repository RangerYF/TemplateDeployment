# 电路搭建反馈站部署说明

## 目标

本次部署只开放 `电路搭建器` 能力，范围包含：

- 模板驱动搭建
- 空白自由搭建

页面打开后进入搭建器首页，不展示实验模板库和专题页。

对应构建模式：

```bash
VITE_APP_SCOPE=builder-feedback
```

在这个模式下：

- 默认进入搭建器首页
- 首页提供两个入口：
  - `/#builder`：模板驱动搭建
  - `/#builder-free`：空白自由搭建
- 不会进入实验模板库和专题页
- 模板模式当前默认落到 `伏安法测电阻`

---

## 本地启动与构建

开发联调：

```bash
pnpm dev:builder-feedback
```

生产构建：

```bash
pnpm build:builder-feedback
```

---

## Vercel 配置

推荐单独新建一个 Vercel 项目用于搭建器反馈，避免和已有电学实验反馈站混在一起。

建议配置：

- Framework Preset: `Vite`
- Install Command: `pnpm install`
- Build Command: `pnpm build:vercel`
- Output Directory: `dist`
- Root Directory: `.`

并在 Vercel 项目环境变量里设置：

```bash
VITE_APP_SCOPE=builder-feedback
```

说明：

- 当前仓库的 `build:vercel` 已支持根据 `VITE_APP_SCOPE` 自动切到搭建器反馈构建
- 使用 hash 路由，静态托管不需要额外 rewrite

---

## CLI 部署

如果已经完成 Vercel 登录并绑定项目，可以直接发起一次预览部署：

```bash
pnpm dlx vercel deploy --yes --build-env VITE_APP_SCOPE=builder-feedback
```

如果确认该链接要直接对外，再执行正式部署：

```bash
pnpm dlx vercel deploy --prod --yes --build-env VITE_APP_SCOPE=builder-feedback
```

---

## 上线前检查

- 打开链接后进入搭建器首页
- `模板驱动搭建` 可进入 `/#builder`
- `空白自由搭建` 可进入 `/#builder-free`
- 模板入口默认可加载 `伏安法测电阻`
- 左侧元件库可正常拖拽元件
- 可正常连线、删线、运行、停止、清空
- 右侧属性面板可修改元件参数
- 滚轮缩放、拖动画布正常
- 移动端至少确认无明显遮挡

---

## 推荐外发方式

如果本轮是体验反馈，建议先发 `preview` 链接给内部，再决定是否升为 `prod`。

适合一起发出的说明：

- 这是电路搭建器专用入口
- 可以先从模板起步，也可以直接空白搭建
- 重点体验模板起步、自由补元件、连线、改参数、运行判读这条链路
