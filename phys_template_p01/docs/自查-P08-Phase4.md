## 3. `P-02 / P-08` 分类口径是否统一；如果未完全统一，原因是什么

结论：产品层和 preset 分类层已统一，保留旧 preset id 前缀。

- 当前 P-08 相关 preset 的 `category` 已统一为 `P-08`。
- `PresetGallery` 中保留独立 `P-08` 模块入口，不再在通用分类列表里重复展开 P-08 场景，避免“独立入口 + 分类重复展示”。
- 旧 preset id 仍保留原有 `P02-...` 前缀，例如 `P02-EMF036-magnetic-divergence`、`P02-EM003-cyclotron-motion`。
- 保留旧 id 的原因：
  - 避免破坏现有 hash 路由、场景引用和已存在的代码依赖。
  - 本轮目标是统一产品命名和分类口径，而不是重做 id 体系。
