**一、这个东西是干什么的？**

高中数学课讲立体几何，老师最头疼的就是：正方体的截面、棱锥的侧面展开、圆柱的外接球......这些东西，在黑板上怎么画都是\"感觉差点意思\"，学生脑子里转不过来。

**立体几何展示台**就是帮老师解决这个问题的，目标是覆盖小学、初中、高中80%的立体集合教学需求，第一版暂时不介入自然语言-LLM入口，后续可以做一些Agent。

老师选一个几何体，设置几个参数，就能得到一个**可以用鼠标拖着转的 3D 模型**。顶点名字可以改，线段可以标字母，还能加坐标系、画外接球、选几个点做外接圆，侧面积体积自动算好显示出来。

**二、用户是谁？他们怎么用？**

**目标用户：** 高中数学老师，不要求有任何编程或软件操作基础。

**典型使用场景：**

课前：提前做好一个\"正四棱锥，底面边长3，高4\"的模型，上课时打开浏览器直接展示

课中：在投影仪上拖动几何体旋转，让学生从各个角度观察

讲题时：边讲边指出\"这条边叫 AB，这个面叫侧面 ABС\"，顶点名字都标在模型上

**老师对工具的期望（直白版）：**

我不会编程，不要让我写代码

操作越简单越好，最好点几下就搞定

必须好看，要能在教室大屏上清晰显示

数据要准确，什么外接球半径、体积，算错了我就被学生问懵了

**三、功能清单（老师视角描述）**

**3.1 第一步：选几何体类型**

打开工具，老师首先看到的是**几何体类型选择区**，提供以下选项，点击即选：

选好之后，右侧立刻出现一个**默认参数的 3D 模型预览**。

  -------------------- --------------------------------------------------
  几何体               说明（不选择均自设默认值）

  棱锥                 可设置底面边数（三棱锥、四棱锥等）和高

  圆锥                 可选设置底面半径和高

  圆柱                 可选设置底面半径和高

  长方体               可选设置长、宽、高三个参数

  正方体               可选只需设置边长一个参数

  球                   可选只需设置半径一个参数
  -------------------- --------------------------------------------------

**3.2 第二步：设置参数**

选完几何体后，出现该几何体的**参数输入面板**，老师用数字输入框修改：

棱锥：底面边数、底面边长、高

圆锥：底面半径、高

圆柱：底面半径、高

长方体：长、宽、高

正方体：边长

球：半径

参数修改后，3D 模型**实时更新**，无需点击确认按钮。

**3.3 第三步：标注与编辑**

这一组功能让老师可以把 3D 模型\"教学化\"------加上数学符号、名字标注，让它看起来像教材里的图。

**✅ 功能 1：给顶点改名字**

每个顶点旁边有一个小标签，默认显示 A、B、C、D......

老师双击标签，可以输入任意文字，比如改成 P、Q，或者 A₁、B₁

修改后，3D 模型上立即显示新名字

**✅ 功能 2：给线段加名字**

老师点击两个顶点，即可选中这两点之间的线段

选中后出现一个输入框，可以给这条线段命名，比如写\"斜高 l\"

名字显示在线段中点附近

**✅ 功能 3：点击取新的点**

老师选中一条线段，点击\"取中点\"按钮

老师点击边任意位置，可以取可在边上任意点，且可以滑动

系统自动在线段中点处添加一个新的顶点，可以给这个中点命名（比如 M）

中点和两端点之间会自动连线

**✅ 功能 4：拖动顶点位置**

老师可以按住某个顶点，拖动来调整它的位置

拖动顶点时，所有相关联的线段和面自动跟着更新

用于做一些非标准形状，比如讲斜切棱柱的时候

**3.4 第四步：辅助功能**

这一组是高中数学中经常配合几何体讲解的工具。

**✅ 功能 5：建立坐标系**

点击\"建立坐标系\"按钮，出现一个**选择原点的提示**

老师点击模型上的某个顶点作为原点

系统自动以这个顶点为原点，生成 x、y、z 三轴，轴上有刻度

各顶点自动显示对应坐标值

**✅ 功能 6：外接球**

点击\"显示外接球\"按钮

系统自动计算当前几何体的外接球，并用半透明的球面显示出来

同时显示外接球半径的计算结果，精确到小数点后两位

仅对有外接球的几何体开放此功能（球本身不显示）

**✅ 功能 7：对选定点做外接圆**

老师在模型上选定**3个或以上**的顶点

点击\"做外接圆\"按钮

系统在这几个点所在平面内画出外接圆，并显示圆的半径

**✅ 功能 8：显示表面积和体积**

页面某个角落（建议右下角）**始终显示**当前几何体的：

表面积（S = \_\_\_）

体积（V = \_\_\_）

参数变化后实时更新

支持精确分数或根号形式展示（如正四面体的体积显示为 √2/12 × a³），也提供数值近似

点击数字详情可以看到计算步骤

**✅ 功能 9：两点间可以创建线段**

创建线段颜色可以选择，虚实可以选择（集合体原有线段默认黑色）

点击可以选择计算线段长度

**✅ 功能 10：切面展示**

工具区可以选择建立切面

点击切面可在展示区展示切面

可计算切面面积

**✅ 功能 11：展开图和三视图**

工具区可以选择建立展开图

工具区可选择查看三视图

可以下载三视图和展开图图片

**3.5 第五步：旋转和查看 3D 模型**

这是整个工具最核心的视觉体验。

**鼠标左键拖动**：旋转模型，从各个角度查看

**鼠标滚轮**：放大缩小

**鼠标右键拖动**：平移模型位置

触屏设备（平板）：支持单指旋转、双指缩放

提供一个\"重置视角\"按钮，一键回到默认角度

提供\"正视图 / 俯视图 / 侧视图\"三个快捷视角按钮

界面设计要点：

整体风格简洁，不要花里胡哨，白底、深色文字

UI主色调参考#00C06B

按钮要大，老师不用戴眼镜也能看清楚点击目标

操作反馈要明显：选中的顶点/线段要有高亮

字体不要太小，至少 14px 以上

**五、验收标准（判断做没做对）**

一位从没用过这个工具的高中数学老师，打开后，**5分钟内**能上手独立做出一个\"正四棱锥，底面边长4，高3，顶点标 P、A、B、C、D\"的 3D 模型

模型可以流畅旋转，不卡顿

外接球半径的计算结果和手算结果一致

体积和表面积显示数值正确

在 1080p 分辨率的教室投影仪上，文字和模型清晰可辨
