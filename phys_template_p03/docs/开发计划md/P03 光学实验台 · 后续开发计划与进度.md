## 一、当前目标重设

在原有 PRD 基础上，P03 后续开发不再只满足“能演示”，而是升级为：

1. **教师可操作度显著提升**
   - 不能只靠几个固定参数滑杆。
   - 老师需要能够自由放置、移动、组合关键光学元件。
   - 典型例子：透镜、屏幕、光源、介质块、双缝、单缝、圆孔等应支持画布内直接操作。

2. **光源体系扩展**
   - 除单束入射光外，必须支持更多课堂常见光源。
   - 至少纳入：
   - 平行光
   - 点光源
   - 单色激光
   - 白光
   - 发散 / 会聚演示光束（用于透镜与成像教学）

3. **覆盖全部高中光学教学场景**
   - 不只覆盖文档中的标准实验编号。
   - 还要支持老师在课堂中临时搭建“变式题”“对比题”“综合题”场景。
   - 目标是从“实验演示器”升级为“光学教学搭建台”。

4. **产品必须有创造性**
   - 不能只是把公式做成滑杆动画。
   - 要让老师能快速搭建、改造、比较不同光学系统。
   - 要让课堂上“为什么会这样”比“结果是什么”更容易讲清楚。

## 二、当前进度快照

### 2.1 已完成

- 已完成独立项目运行壳，可用 Vite 启动和构建。
- 已完成 5 个一级模块入口：
- 折射与全反射
- 透镜与成像
- 双缝干涉
- 单缝 / 圆孔衍射
- 薄膜干涉
- 已完成一轮 PRD 基础实现，支持主要实验参数调节、实时图样绘制、右侧实时数值、公式区。
- 已建立实验数据文件化基础：`src/data/p03-experiments.ts`
- 已把实验标题、默认值、教学点、关键公式、主要参数范围接入部分界面。

### 2.2 当前不足

- 仍以“固定实验 + 滑杆调参”为主，**教师自由搭建能力不足**。
- 缺少统一的“元件系统”和“光源系统”。
- 光学元件还不能在画布中拖拽摆放。
- 多数实验仍是“单场景演示”，还不支持老师临时拼装复杂光路。
- 实验数据虽然已经结构化，但尚未完全驱动所有控件与渲染逻辑。

## 三、后续开发总路线

后续开发分为 **四个阶段**，先保证正确性，再提升可操作度，再做全场景覆盖，最后做课堂体验增强。

## 四、阶段计划

### 阶段 1：现有实验收口与正确性验收

**目标：** 把当前已有的 12 个实验做到“数值和表现可靠”，为后续搭建模式打基础。

**重点任务：**

1. 折射与全反射实验逐项验收
   - `OPT-001` 平行界面折射
   - `OPT-002` 矩形玻璃砖
   - `OPT-003` 半球形玻璃砖
   - `OPT-004` 光导纤维
2. 透镜模块补齐典型物距条件校验
   - `u > 2f`
   - `u = 2f`
   - `f < u < 2f`
   - `u = f`
   - `u < f`
3. 双缝 / 衍射 / 薄膜干涉逐实验验收
4. 修复白光效果、边界条件、读数一致性问题

**阶段交付：**

- 当前 12 个实验全部达到“课堂可用”标准
- 每个实验的公式、读数、趋势、图样均与 PRD 和实验数据一致

**当前状态：** `进行中`

### 阶段 2：高可操作度元件系统

**目标：** 从“参数演示器”升级到“可摆放的光学元件画布”。

**重点任务：**

1. 建立统一的光学元件模型
   - 透镜
   - 光源
   - 屏幕
   - 平行界面
   - 玻璃砖
   - 半球形玻璃
   - 双缝
   - 单缝
   - 圆孔
2. 建立画布内交互
   - 选中
   - 拖拽移动
   - 旋转（必要时）
   - 删除 / 复制
3. 建立基础光线追迹框架
   - 不再只服务单个固定实验
   - 能服务多个元件组合场景

**阶段交付：**

- 老师可在画布中自由摆放透镜、光源、屏幕等核心元件
- 至少支持透镜成像和折射场景的“半自由搭建”

**当前状态：** `已启动`

### 阶段 3：光源系统与全场景覆盖

**目标：** 补足光源类型，并支持更多高中光学课堂变式。

**重点任务：**

1. 建立统一光源类型
   - 单束入射光
   - 平行光
   - 点光源
   - 单色激光
   - 白光
   - 发散光束 / 会聚光束
2. 让各光源接入元件系统
   - 平行光 + 凸透镜
   - 点光源 + 透镜成像
   - 白光 + 双缝 / 薄膜
   - 单色光 + 衍射
3. 建立“课堂变式场景库”
   - 透镜前后移动对比
   - 两块介质组合折射
   - 多透镜组合示意
   - 多光源对比

**阶段交付：**

- 光源不再只有单一默认入射光
- 常见高中光学教学场景都能通过“选元件 + 放位置 + 调参数”完成

**当前状态：** `未开始`

### 阶段 4：创造性教学模式与体验增强

**目标：** 做出明显区别于普通“公式演示器”的课堂价值。

**重点任务：**

1. 加入“搭建模式 / 演示模式”双模式
   - 搭建模式：老师自由摆放元件
   - 演示模式：锁定布局并专注讲解
2. 加入“对比教学”
   - 左右分屏对比不同参数 / 不同光源 / 不同元件布局
3. 加入“预设 + 自定义场景”
   - 教师可从标准实验出发再改造
4. 加入课堂增强能力
   - 高亮关键光路
   - 临界角提示
   - 成像性质标签
   - 重要结论一键标注

**阶段交付：**

- P03 不仅能做标准实验，还能支撑课堂发散讲解
- 产品具备“创造性教学工具”特征

**当前状态：** `未开始`

## 九、结论

P03 后续开发的重点已经明确：

- **短期**：把现有实验做准、做稳
- **中期**：把光学元件做成可摆放、可组合的画布系统
- **长期**：把它做成覆盖全部高中光学教学场景的创造性教学搭建台

后续推进时，所有需求优先级判断都应服从这三条：

1. 是否提升教师可操作度
2. 是否扩大课堂场景覆盖面
3. 是否增强产品的创造性与教学表达力
