# P03 光学实验台教师使用手册

## 一、快速开始

本工具用于高中物理光学课堂演示。教师可以通过调节参数，动态展示折射、透镜成像、干涉、衍射和薄膜干涉等现象。

课堂使用时，建议使用电脑连接投影或电子白板，优先横屏展示。

打开页面后，顶部可以切换 5 个模块：

| 模块 | 用途 |
| --- | --- |
| 折射/全反射 | 讲折射定律、临界角、全反射、玻璃砖、光导纤维 |
| 透镜成像 | 讲凸透镜、凹透镜、物距、像距、像的性质 |
| 双缝干涉 | 讲条纹间距与波长、缝间距、屏距的关系 |
| 单缝/圆孔衍射 | 讲中央亮纹、艾里斑和衍射图样变化 |
| 薄膜干涉 | 讲肥皂泡、楔形薄膜和牛顿环 |

## 二、界面怎么用

页面分为四个主要区域：

| 区域 | 作用 |
| --- | --- |
| 顶部模块栏 | 切换不同光学实验 |
| 左侧控制区 | 调节实验对象、参数和显示开关 |
| 中央演示区 | 查看光路、像、条纹或干涉图样 |
| 右侧读数区 | 查看实时数值、公式和教学要点 |

常用操作：

- 拖动滑杆：改变入射角、折射率、焦距、物距、波长、屏距等参数。
- 点击选项按钮：切换实验类型，例如凸透镜/凹透镜、单缝/圆孔。
- 打开显示开关：显示法线、角度、公式、强度曲线等辅助信息。
- 观察右侧读数：把图像变化和公式结果对应起来。

建议上课时一次只改变一个参数，方便学生看清因果关系。

## 三、推荐课堂使用流程

1. 先选择要讲的模块。
2. 使用默认画面，让学生先观察当前现象。
3. 指出关键对象，例如法线、焦点、屏幕、条纹、中央亮纹。
4. 调节一个关键参数，让学生观察变化。
5. 对照右侧实时读数和公式，总结规律。
6. 切换典型情境或模型，做对比讲解。

## 四、折射与全反射

适合讲：

- 折射定律
- 入射角、反射角、折射角
- 临界角和全反射
- 玻璃砖侧向位移
- 半球玻璃砖
- 光导纤维

课堂操作建议：

1. 先选择“平行界面折射”，观察入射光、反射光、折射光和法线。
2. 调节光线角度，观察折射角如何变化。
3. 调节两侧介质折射率，说明光线向哪一侧偏折。
4. 当光从折射率较大的介质射向折射率较小的介质时，继续增大入射角，观察全反射。
5. 切换到“矩形玻璃砖”，观察出射光线与入射光线平行。
6. 切换到“半球形玻璃砖”，强调曲面处法线与球心有关。
7. 切换到“光导纤维”，说明光在纤芯和包层之间连续全反射。

讲解提醒：

- 角度是相对法线测量，不是相对界面测量。
- 全反射必须同时满足两个条件：从光密介质到光疏介质，且入射角大于临界角。
- 右侧会显示临界角、折射角、是否发生全反射等信息。

## 五、透镜与成像

适合讲：

- 凸透镜成像规律
- 凹透镜成像特点
- 物距、像距和焦距关系
- 实像、虚像、倒立、正立、放大、缩小
- 屏幕能否接到像

课堂操作建议：

1. 先选择凸透镜，观察主光轴、焦点、物体、像和特殊光线。
2. 调节物距，依次观察以下情况：
   - `u > 2f`
   - `u = 2f`
   - `f < u < 2f`
   - `u = f`
   - `u < f`
3. 每改变一次物距，都让学生判断像是放大还是缩小、正立还是倒立、实像还是虚像。
4. 打开屏幕显示，说明只有实像可以成在屏幕上。
5. 切换到凹透镜，观察凹透镜成正立、缩小的虚像。

讲解提醒：

- `u = f` 时出射光近似平行，屏幕上不能得到清晰像。
- 凸透镜 `u < f` 时成虚像，不能用屏幕承接。
- 右侧会显示物距、像距、放大率和像的性质。

## 六、双缝干涉

适合讲：

- 杨氏双缝干涉
- 条纹间距公式 `Δy = λL / d`
- 波长、缝间距、屏距对条纹疏密的影响
- 白光干涉

课堂操作建议：

1. 先观察默认单色光干涉条纹。
2. 调节缝间距 `d`，观察条纹变密或变疏。
3. 调节波长 `λ`，观察条纹间距变化。
4. 调节屏距 `L`，观察屏幕越远条纹间距越大。
5. 打开强度曲线，对应亮纹、暗纹和曲线峰谷。
6. 打开白光干涉，观察中央亮纹和彩色条纹。

讲解提醒：

- `d` 增大，条纹变密。
- `λ` 增大，条纹变宽。
- `L` 增大，条纹变宽。
- 右侧会显示条纹间距和公式验证。

## 七、单缝与圆孔衍射

适合讲：

- 单缝衍射
- 圆孔衍射
- 中央亮纹
- 艾里斑
- 孔径或缝宽对衍射图样的影响

课堂操作建议：

1. 先选择单缝，观察中央亮纹最宽、最亮。
2. 减小缝宽，观察中央亮纹变宽。
3. 增大波长，观察图样变宽。
4. 打开强度曲线，说明中央主极大和两侧次级亮纹。
5. 切换到圆孔，观察中心亮斑和外侧环纹。
6. 调节孔径，观察孔径越小，艾里斑越大。

讲解提醒：

- 单缝衍射和双缝干涉都会出现明暗条纹，但原因不同。
- 单缝的中央亮纹最宽最亮，是观察重点。
- 右侧会显示第一极小位置或艾里斑半径等信息。

## 八、薄膜干涉

适合讲：

- 肥皂泡干涉
- 楔形薄膜干涉
- 牛顿环
- 膜厚、楔角、曲率半径和波长对图样的影响

课堂操作建议：

1. 先分别切换三种模型，让学生观察图样差异。
2. 选择肥皂泡，观察彩色条纹。
3. 选择楔形薄膜，调节楔角，观察条纹间距变化。
4. 选择牛顿环，调节曲率半径或波长，观察环纹变化。
5. 打开公式验证，对照右侧读数讲规律。

讲解提醒：

- 肥皂泡重点看彩色条纹随膜厚变化。
- 楔形薄膜重点看近似平行条纹。
- 牛顿环重点看同心圆环。
- 讲薄膜干涉时，重点放在图样特征和变化趋势上即可。

## 九、课堂使用小建议

- 调参前先让学生预测变化，再拖动滑杆验证。
- 一次只改一个参数，不要同时改变多个量。
- 讲几何光学时，建议打开法线、角度和公式验证。
- 讲干涉、衍射时，建议打开强度曲线和公式验证。
- 右侧“教学要点”可以作为课堂提词使用。
- 如果画面状态不适合当前讲解，可以切换模块后重新进入，或手动把参数调回常用值。
