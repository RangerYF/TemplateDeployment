## 一、模块定位

模块 4 面向高中课堂，目标不是进行复杂傅里叶光学推导，而是帮助老师稳定讲清：

1. 单缝中央主极大为什么最宽。
2. 缝宽、波长、屏距如何影响图样宽度。
3. 圆孔衍射中的艾里斑是什么。
4. 孔径、波长、屏距如何影响艾里斑半径。

本模块应呈现为“图样 + 强度曲线 + 关键读数”三位一体的教学模块。

## 二、场景范围

### 2.1 第一批正式实现

1. `OPT-031 单缝衍射`
2. `OPT-032 圆孔衍射`
3. 波长切换
4. 强度曲线开关
5. 公式验证开关
6. 有边界的布局拖拽

### 2.2 第二批深化内容

1. 多波长对比教学模式
2. 课堂预设场景库
3. 更完整的光源相干性教学表达

## 三、教学重点

### 3.1 单缝衍射

- 主结论卡必须是 `中央主极大宽度`
- 辅助读数必须包含 `第一极小 y₁`
- 默认讲解重点：
  - `a ↓ => 中央主极大变宽`
  - `λ ↑ => 中央主极大变宽`

### 3.2 圆孔衍射

- 主结论卡必须是 `艾里斑半径`
- 默认讲解重点：
  - `D ↓ => 艾里斑变大`
  - `λ ↑ => 艾里斑变大`

## 四、必备控件

1. 单缝 / 圆孔切换
2. 缝宽或孔径调节
3. 波长调节
4. 屏距调节
5. 强度曲线开关
6. 公式验证开关
7. 光源 / 孔径 / 屏幕布局拖拽

## 五、必备读数

### 5.1 单缝

1. 缝宽 `a`
2. 波长 `λ`
3. 屏距 `L`
4. 第一极小 `y₁`
5. 中央主极大宽度 `2y₁`

### 5.2 圆孔

1. 孔径 `D`
2. 波长 `λ`
3. 屏距 `L`
4. 艾里斑半径 `r₁`

## 六、界面要求

1. 布局区必须清晰展示 `光源 - 孔径 - 屏幕`
2. 图样区必须优先展示课堂现象
3. 强度曲线区必须标出关键位置
4. 右侧信息区必须以主结论卡优先，公式区次之

## 七、默认课堂脚本

1. 默认先打开单缝衍射。
2. 展示中央主极大与左右第一极小。
3. 调小缝宽，说明中央主极大变宽。
4. 调大波长，说明中央主极大继续变宽。
5. 切到圆孔衍射。
6. 读取艾里斑半径，说明孔径与波长的影响。

## 八、验收标准

1. 单缝场景中，`a` 变小，中央主极大宽度增大。
2. 单缝场景中，`λ` 变大，中央主极大宽度增大。
3. 圆孔场景中，`D` 变小，艾里斑半径增大。
4. 圆孔场景中，`λ` 变大，艾里斑半径增大。
5. 单缝与圆孔切换后，图样、曲线、读数、公式必须同步切换。
6. 默认打开即可讲，不依赖额外配置。
