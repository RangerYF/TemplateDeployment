## 一、模块定位

模块 5 不是单纯展示“图样长什么样”，而是要帮助老师讲清：

1. 哪个结构导致了薄膜厚度变化。
2. 哪两束反射光在发生干涉。
3. 为什么不同位置会出现不同条纹 / 环纹 / 颜色。

模块 5 必须呈现为：

`成因示意区 + 图样区 + 关系曲线区 + 教学结论卡`

## 三、教学重点

### 3.1 肥皂泡

- 主结论：不同膜厚位置满足不同波长增强条件
- 必须讲清：膜厚沿高度变化、白光分色

### 3.2 楔形薄膜

- 主结论：楔角越小，条纹越稀疏
- 必须讲清：厚度沿 x 方向变化、相邻条纹对应固定厚度差

### 3.3 牛顿环

- 主结论：中心暗点、环半径随 `R` 与 `λ` 变化
- 必须讲清：空气膜厚从中心向外增大

## 四、必备控件

1. 模型切换
2. 厚度 / 楔角 / 曲率半径调节
3. 波长调节（肥皂泡除外可扩展）
4. 薄膜折射率调节
5. 关系曲线开关
6. 公式验证开关
7. 课堂预设按钮

## 五、必备读数

### 5.1 肥皂泡

1. 薄膜厚度 `t`
2. 折射率 `n`
3. 有效光程 `2nt`

### 5.2 楔形薄膜

1. 楔角 `α`
2. 波长 `λ`
3. 条纹间距 `Δx`
4. 折射率 `n`

### 5.3 牛顿环

1. 曲率半径 `R`
2. 波长 `λ`
3. 第一暗环半径 `r₁`
4. 第五暗环半径 `r₅`

## 六、界面要求

1. 中央上方必须是成因示意区，而不是缺少结构来源的纯图样展示
2. 中央视图区必须优先展示实验图样
3. 中央下方必须提供关系曲线区
4. 右侧信息区必须先显示教学结论，再显示参数和公式

## 七、默认课堂脚本

### 7.1 肥皂泡

1. 看白光入射与上下表面反射
2. 看厚度沿高度变化
3. 再看彩色条纹
4. 说明不同波长在不同位置增强

### 7.2 楔形薄膜

1. 看楔形结构
2. 看厚度沿 x 增大
3. 再看平行条纹
4. 说明条纹间距与 `α`、`λ` 的关系

### 7.3 牛顿环

1. 看平凸透镜与平板玻璃结构
2. 指出中心接触点与空气膜厚变化
3. 再看同心圆环
4. 说明中心暗点与环半径变化

## 八、验收标准

1. 肥皂泡必须展示彩色条纹并能解释为厚度变化导致
2. 楔形薄膜必须展示近似平行条纹，且 `α ↓ => Δx ↑`
3. 牛顿环必须展示中心暗点和同心环
4. 模型切换后，成因示意区、图样区、关系曲线区、读数区、公式区必须同步变化
5. 默认打开即可讲，不依赖教师额外解释界面含义
