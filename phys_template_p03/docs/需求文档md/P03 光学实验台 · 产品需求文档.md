## 一、这个东西是干什么的？

折射、反射、透镜成像如果只靠静态画图，老师很容易在法线、角度和像的位置上画错；干涉、衍射、薄膜干涉又高度抽象，学生往往只能背结论，难以理解光路和图样为什么会这样出现。

**光学实验台**提供一个可交互的光学演示环境，覆盖高中光学重点内容。老师可以选择实验模块、拖拽或切换元件、调节关键参数，系统实时绘制光路、角度标记和干涉衍射图样，让“看不见的光学现象”变成可见、可比、可讲解的课堂演示。

## 二、用户是谁？他们怎么用？

**目标用户：** 高中物理老师，不要求具备任何编程基础。

**典型使用场景：**

- 演示全反射：调节入射角，直观看到临界角出现前后的折射与反射变化，并借此讲解光导纤维原理。
- 展示双缝干涉：改变双缝间距、波长或屏距，实时观察条纹疏密与亮暗分布变化。
- 讲解单缝或圆孔衍射：切换不同波长，比较不同颜色光的衍射图样宽度差异。
- 讲解薄膜干涉：切换肥皂泡、楔形薄膜、牛顿环三种模型，说明图样形状与膜厚变化关系。
- 演示透镜成像：让学生观察物距变化时像的正倒、大小、虚实变化，用于初中/高中入门演示。

## 三、功能清单

### 3.1 第一步：选择实验模块

教师进入工具后，首先看到实验模块选择区，支持以下 5 个入口：

| 模块 | 主要内容 | 交互重点 |
| --- | --- | --- |
| 折射与全反射（几何光学） | 平行界面折射、矩形玻璃砖、半球形玻璃、光导纤维 | 调节入射角、介质折射率、形状，实时看光路 |
| 透镜与成像（初中） | 凸透镜、凹透镜、成像规律演示 | 调节焦距和物距，观察像的位置与性质 |
| 双缝干涉 | 不同缝间距、距离、波长下的干涉条纹 | 以现象演示为主，参数变化即时反馈 |
| 单缝 / 圆孔衍射 | 不同缝宽 / 孔径、不同波长下的衍射图样 | 展示衍射强度分布和中央明纹宽度变化 |
| 薄膜干涉 | 肥皂泡、楔形薄膜、牛顿环 | 切换模型并观察典型图样形状 |

### 3.2 折射与全反射模块

该模块是几何光学核心，要求以“光线追迹 + 实时数值”方式展示。

**支持形状：**

- 球状玻璃砖
- 半球状玻璃砖
- 矩形玻璃砖
- 挖空半球

**支持材料：**

- 玻璃
- 水
- 光导纤维材料
- 空气（默认；高中阶段空气与真空按相同折射率近似）

**核心交互：**

- 调节入射角，实时显示入射光、折射光、反射光。
- 显示法线、入射角、折射角、反射角。
- 当满足全反射条件时，停止绘制折射光并高亮反射路径。
- 在光导纤维模型中连续展示多次全反射路径。

**几何光学量化指标：**

- 斯涅尔定律：`n1 sin(theta1) = n2 sin(theta2)`
- 全反射临界角：`sin(theta_c) = n2 / n1`，其中 `n1 > n2`
- 空气折射率默认取 `1.000`
- 常用玻璃折射率默认取 `1.50`

**建议预设实验：**

- 平行界面折射
- 矩形玻璃砖侧移
- 半球形玻璃砖临界角演示
- 光导纤维导光原理

### 3.3 透镜与成像模块

该模块以课堂演示为主，不强调复杂计算推导，重点是帮助学生建立成像规律直觉。

**支持内容：**

- 凸透镜成像
- 凹透镜成像
- 物距变化与像的正倒、大小、虚实关系

**建议交互：**

- 可调参数：焦距 `f`、物距 `u`
- 实时显示主光轴、焦点、物体、像
- 可选显示透镜公式验证：`1 / u + 1 / v = 1 / f`
- 在右下角显示当前像距和像的性质说明

**实现要求：**

- 以演示效果正确为主
- 初中内容不要求做复杂数值题型求解器

### 3.4 干涉与衍射模块

该部分以“现象演示 + 趋势正确 + 图样可辨识”为核心目标。

**双缝干涉：**

- 只展示实验现象，不要求展开完整推导过程。
- 可调参数：双缝间距 `d`、屏距 `L`、波长 `lambda`
- 需实时显示条纹图样，并可在数值区显示条纹间距 `Delta y = lambda L / d`
- 当 `d` 变大时，条纹应变密；当 `lambda` 或 `L` 变大时，条纹应变疏

**单缝 / 圆孔衍射：**

- 可调缝宽 / 孔径和波长
- 实时显示衍射强度分布图
- 明确体现“缝宽变小时，条纹变宽”的效果
- 不同波长时，图样宽度差异应可见

**薄膜干涉：**

- 支持肥皂泡、楔形薄膜、牛顿环三种模型切换
- 需显示典型图样：
- 肥皂泡：彩色变化带
- 楔形薄膜：近似平行条纹
- 牛顿环：同心圆环
- 可调节参数：薄膜厚度、楔角或波长
- 以实验图样展示为主，不要求做超出高中范围的公式推导交互

### 3.5 通用显示与教学辅助

**左侧参数区：**

- 根据当前模块动态切换参数控件
- 参数类型以滑杆、数字输入、下拉框为主

**显示选项：**

- 角度标注开关
- 公式验证开关
- 颜色显示开关
- 法线显示开关
- 强度分布曲线开关

**实时数值区：**

- 折射率 `n`
- 临界角 `theta_c`
- 条纹间距 `Delta y`
- 焦距 / 物距 / 像距
- 当前波长与对应颜色

**布局建议补充：**

- 顶部按钮切换模块时保留统一交互框架，仅替换左侧参数与中央渲染内容。
- 中央视图区优先保证演示清晰度，避免被过多说明文字挤压。
- 右下角数值区固定位置，便于课堂讲解时快速指读。
- 干涉与衍射模块建议支持“图样视图”和“强度曲线视图”并排或切换显示。

## 五、验收标准

- 全反射临界角数值与 `sin(theta_c) = 1 / n` 或更一般形式 `sin(theta_c) = n2 / n1` 的计算结果一致。
- 当入射角小于临界角时存在折射光；达到并超过临界角后，表现为全反射，变化方向正确。
- 双缝干涉条纹间距随缝间距增大而减小，随波长或屏距增大而增大，变化方向正确。
- 单缝 / 圆孔衍射在不同波长下图样宽度差异明显，且缝宽或孔径减小时中央主极大变宽。
- 薄膜干涉三种图样形态正确，其中牛顿环必须表现为同心圆环。
- 透镜成像在典型物距下的像的正倒、大小、虚实关系正确。
- 各模块参数调整后，中央演示区应在可接受的课堂交互延迟内实时刷新，不出现明显卡顿或错误残影。

### 6.3 实现优先级建议

1. 先完成折射与全反射，因为它既需要光线追迹，也最能验证几何光学框架是否稳定。
2. 再完成双缝干涉和单缝衍射，用统一的波长与颜色映射、图样渲染体系复用能力。
3. 最后补透镜成像和薄膜干涉，优先保证课堂演示正确性，再考虑更细的视觉优化。
