综合演示建议（串联场景）
场景A：基础建模+标注（3分钟）
1. 选择长方体 → 调参数 → 切换为棱锥 → 调底面边数
2. 双击改顶点名 → 画一条辅助线段（红色虚线）
3. 建截面 → 查看截面面积
4. Ctrl+Z 撤销几步 → Ctrl+Y 重做
场景B：坐标系+度量（3分钟）
1. 建坐标系（三步流程）→ 输入坐标创建点
2. 度量角度（二面角 / 线面角）
3. 度量距离（点到面 / 异面线距离+公垂线）
场景C：新几何体展示（2分钟）
1. 遍历几种新几何体（正四面体 → 墙角四面体 → 圆台 → 棱台）
2. 展示分类选择器
3. 查看体积/表面积精确值

需要产品经理确认的发散功能汇总
以下功能不在 v1.5 + v2 需求文档中，是开发基于教学场景自行发散的：
4. 交互细节
- Ctrl穿透选中、工具模式提示条、画线脉冲动画
- 发散理由：提升操作引导，降低老师学习成本
5. 曲面体交互
- 圆柱/圆锥/球的面和曲线可选中、取点
- 发散理由：原需求只提了多面体交互，曲面体补全后体验更完整
6. 坐标系增强
- 任意点建系（非顶点）、坐标输入创建点
- 发散理由：可以解更多坐标题型，如在棱中点建系
7. 距离度量（5种）
- 点点/点线/点面/线线/线面距离 + 公垂线可视化
- 发散理由：高考常考"求距离"题型，与角度度量配套
8. 实体列表面板
- 左侧按类型分组的实体列表 + 搜索
- 发散理由：复杂场景下方便管理大量辅助线/点/标注
9. 属性面板（Inspector）
- 右侧根据选中实体动态显示的属性编辑器
- 发散理由：统一的属性查看/编辑入口，不用右键菜单找功能
